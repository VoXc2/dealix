الأسئلة الشائعة

أسئلة نسمعها كثيراً

هل بياناتي تُخزَّن داخل المملكة العربية السعودية؟ +

نعم. البيانات الحسّاسة تُخزَّن في مناطق سحابية داخل المملكة، مع نسخ احتياطية مُشفَّرة. نتوافق مع متطلبات PDPL الخاصة بتوطين البيانات للفئات الحساسة.

كيف تضمنون خصوصية محادثات WhatsApp؟ +

نستخدم WhatsApp Business API الرسمي من Meta، مع تشفير نقل عبر TLS 1.3، وتشفير تخزين بـ AES-256. الرسائل لا تُشارَك مع أي طرف ثالث، وكل وصول مُسجَّل في Audit trail.

هل يمكنني إلغاء الاشتراك في أي وقت؟ +

نعم، الاشتراك شهري بدون التزام طويل. يمكنك الإلغاء من لوحة التحكم في أي وقت، ونوفّر لك تصدير كامل لبياناتك خلال 14 يوماً من تاريخ الإلغاء.

ما نماذج الذكاء الاصطناعي التي يستخدمها النظام؟ +

نستخدم LLM Router ذكي: Groq كخيار أساسي للسرعة العالية والتكلفة المنخفضة، مع fallback إلى OpenAI GPT-4 للمهام المعقدة التي تتطلب استدلال عميق. الاختيار يتم تلقائياً حسب نوع المهمة.

كيف يتم التكامل مع CRM موجود لدينا؟ +

ندعم تكاملاً أصلياً مع HubSpot, Salesforce, Zoho, و Pipedrive عبر API رسمي. لـ CRM مخصص، نوفّر Webhook وواجهة REST قياسية موثّقة بالعربي والإنجليزي.

هل تُوجد فترة تجريبية مجانية؟ +

نحن في مرحلة Pilot مع عدد محدود من الشركاء. نقدّم إعداداً مُصمَّماً لكل فريق مع فترة تجريب مخصصة حسب حجمك — تواصل معنا لمناقشة التفاصيل.
