PDPL & data residency

Dealix is built Saudi-first. Tenant data lives in PDPL-compliant Postgres with row-level security; cross-border transfers are governed by the addendum at docs/CROSS_BORDER_TRANSFER_ADDENDUM.md. LinkedIn integrations are disabled by default for ToS compliance.

Security disclosure

Report vulnerabilities privately to security@ai-company.sa or via GitHub Security Advisories. We acknowledge within 48 hours and provide a resolution timeline within 7 days. Full policy: SECURITY.md.
