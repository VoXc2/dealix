1. Application-Layer Controls

Sentry PII Scrubber

LIVE

كل event يذهب لـ Sentry يمرّ بـ _scrub_event: يُصفّر authorization/cookie/api-key headers، يستبدل query strings على /admin/* و /auth/*، يُلغي bodies على /checkout و /webhooks/*.

dealix/observability/sentry.py + tests/test_sentry_pii_scrubber.py (12 tests)

Moyasar Webhook Signature Verification

LIVE

كل webhook callback يتم التحقّق منه بـ HMAC قبل أي معالجة. Production env يرفض أي signature غير صحيحة.

dealix/payments.py:verify_webhook + tests/test_moyasar_webhook_persistence.py

WhatsApp Webhook x-hub-signature-256

LIVE

Production مع WHATSAPP_APP_SECRET ينفّذ strict signature check. اختبارات W7.4 تحمي من regression.

api/routers/webhooks.py + tests/test_whatsapp_webhook_integration.py (7 tests)

Tenant Isolation Middleware

LIVE

كل request يحمل tenant_id؛ تحقّق على الـ ORM layer + (في الإنتاج) row-level security.

api/middleware/tenant_isolation.py

Admin API Key Gate

LIVE

كل /api/v1/admin/* route تتطلّب X-Admin-API-Key header. مفاتيح مدوّرة كل 90 يوماً.

api/security/api_key.py:require_admin_key

Rate Limiting (slowapi)

LIVE

per-IP + per-route limits configurable via env (P8 in PRODUCTION_ENV_TEMPLATE). Defaults: global 1000/min, generic 60/min, leads 30/min, sales 60/min, WhatsApp webhook 300/min.

RL_* environment vars · slowapi integration
