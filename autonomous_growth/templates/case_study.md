# Case Study Template

## Structure

### 1. Title
```
[Company Name]: [Result] through [Solution]
شركة [اسم العميل]: [النتيجة] عبر [الحل]
```

### 2. Executive Summary (2-3 lines)
```
Arabic: ملخص تنفيذي للمشروع والنتائج الرئيسية.
English: High-level summary of the project and key results.
```

### 3. Challenge (3-5 lines)
```
Arabic:
قبل التعاون مع Dealix، كان [العميل] يواجه [التحديات الرئيسية]:
• [التحدي 1]
• [التحدي 2]
• [التحدي 3]

English:
Before working with Dealix, [Customer] was facing [key challenges]:
• [Challenge 1]
• [Challenge 2]
• [Challenge 3]
```

### 4. Solution (5-8 lines)
```
Arabic:
طبقنا [حل Dealix] المخصص لقطاع [القطاع]:
• [المكون 1]
• [المكون 2]
• [المكون 3]

English:
We deployed [Dealix solution] tailored to [sector]:
• [Component 1]
• [Component 2]
• [Component 3]
```

### 5. Results (3-5 lines with metrics)
```
Arabic:
النتائج خلال [المدة]:
• [المؤشر 1]: [القيمة] → [القيمة] ([نسبة التحسن])
• [المؤشر 2]: [القيمة] → [القيمة] ([نسبة التحسن])

English:
Results within [timeframe]:
• [Metric 1]: [Before] → [After] ([Improvement])
• [Metric 2]: [Before] → [After] ([Improvement])
```

### 6. Testimonial (optional)
```
Arabic:
"[اقتباس من العميل]" — [الاسم], [المسمى الوظيفي]

English:
"[Customer quote]" — [Name], [Title]
```

### 7. CTA
```
Arabic: هل تريد نتائج مشابهة؟ احجز استشارة مجانية.
English: Want similar results? Book a free consultation.
```

## Evidence Level Requirements
- **L3**: Data from Dealix platform dashboards
- **L4**: Customer-provided metrics with consent
- **L5**: Audited/third-party verified results
