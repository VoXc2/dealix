# Social Media Carousel Template

## Format
- **Slide 1**: Cover / Hook
- **Slides 2-4**: Content (1 key point per slide)
- **Slide 5**: CTA / Summary

## Carousel Structure

### Slide 1: Cover
```
[Eye-catching title in Arabic/English]
[Subtitle with key benefit]

#Dealix #[SECTOR]
```

### Slide 2: The Problem
```
العنوان: [المشكلة]
النص: [وصف المشكلة التي يواجهها الجمهور]
البيان: [إحصائية أو رقم]

Title: [The Problem]
Text: [Description of the audience's pain point]
Data: [Statistic or number]
```

### Slide 3: The Insight
```
العنوان: [الرؤية]
النص: [تحليل أو اكتشاف مهم]
مثال: [مثال واقعي]

Title: [The Insight]
Text: [Important analysis or discovery]
Example: [Real-world example]
```

### Slide 4: The Solution
```
العنوان: [الحل]
النص: [كيف يحل Dealix المشكلة]
الخطوات: [3 خطوات بسيطة]

Title: [The Solution]
Text: [How Dealix solves the problem]
Steps: [3 simple steps]
```

### Slide 5: CTA
```
[العنوان المحفز]
احجز استشارتك المجانية الآن
[LINK]

[Motivating Title]
Book your free consultation now
[LINK]

#Dealix #AI #SaudiVision2030 #[SECTOR]
```

## Design Guidelines
- **Colors**: Dealix brand (#1a1a2e, #e94560, #0f3460)
- **Font**: Clean, readable Arabic/English
- **Images**: High-quality, relevant, culturally appropriate
- **Text**: Max 40 words per slide
- **Logo**: Dealix logo on first and last slide

## Best Practices
- Use data visualization for statistics
- Keep arabic text right-aligned
- English text left-aligned
- Test readability on mobile
- Include alt text for accessibility
