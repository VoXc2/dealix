# Demo Script — Security

## Opening
نركز على إغلاق فجوات الأمن التي تعطل صفقات Enterprise.

## Pain
مراجعات الأمان طويلة، والضوابط غير موثقة بشكل كافٍ.

## Before State
- صلاحيات غير مكتملة
- تتبع أمني ضعيف
- صعوبة اجتياز security questionnaires

## Live Workflow
1. عرض قائمة المخاطر الحرجة
2. تفعيل authorization checks
3. عرض evidence pack للضوابط

## Business Outcome
- تسريع إغلاق مراجعات الأمن
- تقليل الحوادث
- رفع جاهزية البيع المؤسسي

## ROI
وقت أقل في cycle الأمن + فرص بيع أكبر.

## Close
الخطوة التالية: Security Silver sprint لمدة 30 يوم.
