# تقرير جاهزية الإطلاق (Focus 5 Launch Readiness)

**التاريخ:** 2026-06-03
**النطاق:** جاهزية الأنظمة الخمسة للنزول للسوق (frontend + outreach + delivery + governance)

---

## 1. الحالة العامة

🟢 **جاهز للإطلاق التشغيلي (draft-only)** — الصفحات والمحتوى والتسعير وحزم التسليم
ومصنع المسودات جاهزة. الإرسال يبقى محجوبًا حتى تكتمل بوابات البريد.

---

## 2. قائمة الجاهزية

### Frontend
- [x] صفحة `/systems` (مقارنة + شبكة بطاقات + CTA)
- [x] صفحات `/systems/:slug` للأنظمة الخمسة
- [x] صفحة `/pricing`
- [x] كل صفحة: Hero + Pain + Benefits + Who + 7-day + Delivery Pack + Price + CTA + FAQ
- [x] روابط التنقل من الصفحة الرئيسية إلى الأنظمة والأسعار

### المحتوى والتسعير
- [x] سعر افتتاحي معلن لكل نظام (3,000–5,500 SAR)
- [x] ملاحظة «النطاق النهائي بعد التشخيص» تحت كل سعر
- [x] لا وعود نتائج مضمونة في أي صفحة

### Outreach
- [x] مصنع 400 مسودة/يوم (مواصفات + إعداد canonical)
- [x] توزيع المسودات يساوي 400 بالضبط
- [x] بطاقة احتياج العميل (17 حقلًا)
- [x] مكتبة إيميلات لكل نظام
- [x] قائمة Top 100 للاعتماد

### الحوكمة والأمان
- [x] الإرسال غير مفعّل افتراضيًا (`SEND_DEFAULT_ENABLED = false`)
- [x] الممنوعات: cold WhatsApp / LinkedIn automation / purchased lists / fake Re:/Fwd:
- [x] لا أسرار/PII في المسودات أو السجلات
- [ ] SPF / DKIM / DMARC مجهّزة (قبل أول إرسال)
- [ ] unsubscribe + suppression list مفعّلة (قبل أول إرسال)
- [ ] domain health مستقر (قبل رفع الإرسال)

### الجودة (اختبارات آلية)
- [x] `tests/systems.test.ts` — الأنظمة والأسعار وحزم التسليم
- [x] `tests/draftFactory.test.ts` — 400 مسودة، draft-only، البوابات
- [x] `tests/content-guard.test.ts` — لا وعود ولا أسماء داخلية

---

## 3. ما الذي يحجب الإرسال الآن؟

| البند | لماذا يحجب؟ | المالك |
| --- | --- | --- |
| SPF/DKIM/DMARC | متطلب أساسي لتسليم Gmail | المؤسس/التقني |
| unsubscribe + suppression | متطلب امتثال للرسائل التسويقية | المؤسس |
| domain health | حماية السمعة قبل رفع الحجم | المؤسس/التقني |

> الإنتاج (draft-only) لا يتأثر؛ فقط الإرسال محجوب حتى تُغلق هذه البنود.

---

## 4. القرار

- ✅ نشر صفحات الأنظمة والأسعار الآن.
- ✅ تشغيل مصنع المسودات (400/يوم) الآن.
- ⏳ تفعيل الإرسال لاحقًا بعد إغلاق بوابات البريد واعتماد المؤسس.
