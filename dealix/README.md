# Dealix - نظام التشغيل الذكي للأعمال

**الموقع:** https://dealix.me  
**المنصة:** Python FastAPI + Next.js + PostgreSQL + Supabase  
**السوق:** المملكة العربية السعودية (KSA-first)  
**النموذج:** B2B SaaS + خدمات مدارة + وكالة شريكة  

---

## ما هو Dealix؟

Dealix هو نظام تشغيل ذكي يخدم الشركات السعودية عبر:
- **فريق ذكاء اصطناعي** يعمل 24/7
- **إدارة علاقات العملاء (CRM)** متكاملة
- **أتمتة المبيعات والتسويق**
- **الامتثال السعودي** (PDPL + ZATCA + مركزية البيانات)
- **تحليلات وذكاء** للقرارات

## سلم العروض

> المصدر الوحيد للحقيقة: [`../docs/COMPANY_SERVICE_LADDER.md`](../docs/COMPANY_SERVICE_LADDER.md).
> Dealix قبل مرحلة الإيراد ولا عملاء لها — الجدول يصف عروضاً لا زخماً.

| الدرجة | العرض | السعر (ريال) | شرط فتح الدرجة التالية |
|--------|-------|--------------|------------------------|
| 0 | تشخيص العمليات المجاني | 0 | تسليم 3 تشخيصات |
| 1 | سبرنت ذكاء الإيرادات في 7 أيام | 499 | تسليم تجربة مدفوعة واحدة كاملة |
| 2 | حزمة من البيانات إلى الإيراد | 1,500 | 3 تجارب في القطاع نفسه |
| 3 | إدارة عمليات الإيرادات | 2,999–4,999 / شهر | 3 أشهر متتالية من الاشتراك المدفوع |
| 4 | إعداد خدمة ذكاء اصطناعي مخصصة | 5,000–25,000 | تسليم 3 تجارب + إذن نشر موقَّع |

"Executive Command Center" و"Agency Partner OS" ليست عروضاً فعّالة — مغلقة حتى
استيفاء شرط الفتح.
