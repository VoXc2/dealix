# قناة الشركاء للأنظمة الخمسة — Focus 5 Partner Channel

بما أنك تقدم 5 أنظمة، الشراكات ترفع التوزيع بسرعة. المصدر:
`company_os/commercial/partners.json`. الأنبوب المولّد:
`reports/partners/FOCUS_5_PARTNER_PIPELINE.md`.

---

## النموذج

```
أنت تجيب العلاقة.
Dealix يشغّل النظام.
الشريك يحصل على referral fee أو margin.
```

لا تُشارَك بيانات العميل مع الشريك دون موافقة (متوافق مع PDPL وسياسة البيانات).

---

## أي نظام يناسب كل شريك؟

| الشريك | الأنظمة المقترحة |
|--------|-------------------|
| وكالة تسويق | تشغيل الإيرادات / استرجاع المتابعات |
| CRM Implementer | القيادة التنفيذية / تشغيل الإيرادات |
| شركة تدريب | استرجاع المتابعات / عملاء واتساب |
| مستشار أعمال | القيادة التنفيذية / العروض والإثبات |
| Web Agency | العروض والإثبات / تشغيل الإيرادات |
| HR Firm | استرجاع المتابعات / القيادة التنفيذية |
