## Data Collection Checklist

### Required (Minimal)
- [ ] Lead/CRM export file (last 90 days)
- [ ] 20-50 sales conversations (anonymized if possible)
- [ ] 2-3 proposal/quote samples
- [ ] List of communication channels used
- [ ] Current follow-up process description
- [ ] Top 3 services/products sold
- [ ] Price ranges for top services
- [ ] Top 5 customer objections encountered

### Optional (Helpful)
- [ ] Website URL
- [ ] Social media accounts
- [ ] Current CRM name
- [ ] Team size (sales)
- [ ] Average deal size
- [ ] Target monthly revenue
