# Agents Readiness

## Objective

Confirm each agent has a full control pack, passing evals, and rollback path.

## Gate Checklist

| Gate ID | Requirement | Evidence ID | Test ID |
|---|---|---|---|
| G-RDY02-001 | documented control ownership exists | E-RDY02-001 | T-RDY02-001 |
| G-RDY02-002 | release-period evidence package complete | E-RDY02-002 | T-RDY02-002 |
| G-RDY02-003 | last gate-linked tests passed | E-RDY02-003 | T-RDY02-003 |
| G-RDY02-004 | rollback drill evidence available | E-RDY02-004 | T-RDY02-004 |

## Decision Rule

Readiness is pass only when all listed gates are pass.
