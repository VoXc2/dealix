# Client Pilot Ready

## Definition
Dealix is client-pilot ready when one tenant can execute the full governed workflow with:
- tenant isolation
- approval-first high-risk actions
- runtime kill switch propagation
- value evidence with source discipline
- full run trace and rollback gating
