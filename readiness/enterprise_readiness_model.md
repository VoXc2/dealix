# Enterprise Readiness Model

## Stages
1. **Code Health Ready**
   - compile/import/lint + targeted tests.
2. **Control Plane Ready**
   - run lifecycle + approval-gated rollback/policy-edit.
3. **Agent Governance Ready**
   - tenant-scoped routing, isolation, autonomy ceilings, contracts.
4. **Operational Safety Ready**
   - kill switch + circuit breaker + explicit failure behavior.
5. **Value Proof Ready**
   - value discipline + source-backed measured metrics + ROI reporting.
6. **Client Pilot Ready**
   - one tenant, one governed workflow, one rollback drill, one approval flow.
7. **Enterprise Ready**
   - multi-tenant persistence, CI release gates, admin control surfaces, runbooks.

## Exit criteria
- Stage exit requires passing verification commands + evidence artifacts.
