# Tests Required Before Client Sale

## Non-Negotiable Test Set

1. Tenant isolation suite (`T-TNT-*`).
2. RBAC authorization suite (`T-RBAC-*`).
3. Workflow execution suite (`WF-LEAD-*` cases).
4. Governance high-risk suite (`GOV-HR-*` cases).
5. Retrieval + citation suite (`RET-*`, `T-CIT-*`).
6. Cross-layer validation suite (`T-CLV-*`).
7. Observability continuity suite (`T-OBS-*`, `T-ALT-*`).
8. Rollback drill suite (`T-RBK-*`).

## Mandatory Evidence Bundle

- test reports
- trace snapshot of end-to-end run
- approval log for high-risk case
- eval report for quality + policy compliance
- executive ROI report
- rollback drill report

## Sale Readiness Decision

`READY_TO_SELL` only when:
- all non-negotiable tests pass,
- no critical risk remains open without mitigation,
- client pilot checklist is fully complete.
