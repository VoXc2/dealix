# Monitoring Stack Contract

المبدأ: observability ليست مرحلة لاحقة، بل شرط تشغيل أساسي.

## Minimum Stack

- OpenTelemetry tracing
- Prometheus metrics
- Grafana dashboards
- Sentry error tracking
- PostHog product analytics

## Required Outputs

- service health dashboards
- workflow failure heatmap
- alert routing for incidents
