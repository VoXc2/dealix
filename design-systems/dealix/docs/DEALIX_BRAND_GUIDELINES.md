## Visual Identity

### Logo System

#### Primary Logo
- **Mark**: Diamond-shaped geometric form with embedded "D"
- **Proportions**: 1:1 square spacing
- **Color**: Dealix Navy (#001F3F) with Dealix Gold accent
- **Minimum Size**: 40px for digital, 20mm for print
- **Clear Space**: Logo width on all sides

#### Logo Variations
1. **Full Logo** (Mark + Wordmark): Default for primary use
2. **Mark Only**: Square version for favicon, app icons, social avatars
3. **Horizontal**: Extended layout for headers
4. **Vertical**: Stacked layout for vertical spaces
5. **Monochrome**: Single-color versions (navy/white)
6. **Reversed**: White logo on dark backgrounds

#### Logo Do's & Don'ts
✅ DO:
- Maintain aspect ratio
- Use on solid backgrounds initially
- Apply adequate clear space
- Use approved color versions
- Scale proportionally

❌ DON'T:
- Rotate or skew
- Stretch or compress
- Change colors arbitrarily
- Add effects/shadows
- Use outdated versions

|-------|-----|-----|-------|
| **Dealix Navy** | #001F3F | 0, 31, 63 | Primary brand color, headers, buttons |
| **Dealix Gold** | #D4AF37 | 212, 175, 55 | Accents, highlights, premium feel |
| **Dealix Black** | #0A0A0A | 10, 10, 10 | Text, dark backgrounds |
| **Dealix White** | #FFFFFF | 255, 255, 255 | Backgrounds, text on dark |

### Secondary Palette

| Color | Hex | RGB | Usage |
|-------|-----|-----|-------|
| **Slate Blue** | #364558 | 54, 69, 88 | Subtle backgrounds, secondary text |
| **Ocean Blue** | #0066FF | 0, 102, 255 | Links, CTAs, interactive elements |
| **Emerald Green** | #10B981 | 16, 185, 129 | Success states, positive actions |
| **Coral Red** | #EF4444 | 239, 68, 68 | Errors, alerts, destructive actions |
| **Amber Gold** | #F59E0B | 245, 158, 11 | Warnings, attention-needed |
| **Light Gray** | #F3F4F6 | 243, 244, 246 | Card backgrounds, subtle dividers |

### Color Usage Rules

**Navy (#001F3F)**: 
- Primary headers (60-70% of design)
- Navigation bars
- Call-to-action buttons
- Dashboard headers

**Gold (#D4AF37)**:
- Accent elements (5-10%)
- Icons highlighting
- Border accents
- Premium features indicator
- Hover states on navy

**Black (#0A0A0A)**:
- Body text
- Primary content
- Navigation text
- Strong emphasis

**White (#FFFFFF)**:
- Backgrounds
- Text on navy/black
- Card backgrounds
- Negative space

### Accessibility

- Minimum contrast ratio: 4.5:1 for WCAG AA
- Navy + White: Contrast ratio 12.63:1 ✅
- Gold + Navy: Contrast ratio 7.42:1 ✅
- Avoid relying solely on color for information
- Use icons + color combinations for status

## Typography

### Font Stack

#### Display Font (Headlines)
**Font Family**: `'Poppins Bold'`, `'Cairo'` (for Arabic)
- Weight: 700-900
- Usage: H1, H2, H3 main headlines
- Character: Modern, geometric, authoritative

#### Heading Font
**Font Family**: `'Poppins SemiBold'`, `'Cairo'` (for Arabic)
- Weight: 600
- Usage: Section headers, subheadings
- Character: Clear, professional

#### Body Font
**Font Family**: `'Inter'`, `'Tajawal'` (for Arabic)
- Weight: 400-500
- Size: 14-16px
- Line height: 1.6-1.8
- Usage: Body text, labels, descriptions
- Character: Readable, neutral, clean

#### Code Font
**Font Family**: `'IBM Plex Mono'`, monospace
- Weight: 400
- Usage: Code blocks, technical content
- Character: Monospaced, distinctive

### Scale & Hierarchy

```
H1: 48px / 56px line-height / 700 weight
H2: 36px / 44px line-height / 700 weight
H3: 28px / 36px line-height / 600 weight
H4: 24px / 32px line-height / 600 weight
Body Large: 18px / 28px line-height / 400 weight
Body: 16px / 24px line-height / 400 weight
Body Small: 14px / 22px line-height / 400 weight
Caption: 12px / 18px line-height / 500 weight
```

### Arabic Typography

- Use `'Tajawal'` (400, 500, 600, 700) for body and UI
- Use `'Cairo'` (600, 700, 900) for headlines
- Line-height: +0.2 compared to English (1.8 for body)
- Letter-spacing: Negative -0.5px for display fonts

## Imagery & Photography

### Photography Style

**Style Guide**: Enterprise luxury with human elements

**Photography Rules**:
1. **People**: Diverse Saudi professionals, confident, working
2. **Lighting**: Natural, warm, professional
3. **Tone**: Optimistic, forward-looking, trustworthy
4. **Settings**: Modern offices, tech environments, collaborative spaces
5. **Color Treatment**: Warm neutrals, slight color grade matching brand palette

### Iconography

**Style**: Minimal geometric line icons
- **Weight**: 2px stroke
- **Size**: 24px base, scale proportionally
- **Color**: Navy primary, Gold for accent
- **Style**: Geometric, modern, Saudi-influenced geometric patterns in details

### Patterns

**Primary Pattern**: Geometric dots grid (navy + gold)
- Subtle use in backgrounds
- 16px × 16px dot spacing
- 2px dot diameter
- Opacity: 5-10%

**Secondary Pattern**: Arabic calligraphy accent lines
- Subtle decorative element
- Gold color
- Used in headers, page dividers
- Optional, premium feature indicator

## Voice & Tone

### Brand Voice Attributes

- **Authoritative**: We know our domain deeply
- **Accessible**: Complex concepts made simple
- **Confident**: In our AI and platform capabilities
- **Saudi-Centric**: Understanding local business culture
- **Human-First**: Technology serves people, not vice versa

### Tone Contexts

**In Features**:
- Clear, benefit-driven
- Avoid jargon unless necessary
- Action-oriented

**In Errors**:
- Helpful, not blame-oriented
- Suggest next steps
- Apologetic but brief

**In Onboarding**:
- Welcoming, encouraging
- Bite-sized guidance
- Celebratory milestones

**In Marketing**:
- Aspirational
- Results-focused
- Industry-leading positioning

### Copywriting Examples

❌ Bad: "Your data will be processed"
✅ Good: "We'll analyze your revenue patterns and identify growth opportunities"

❌ Bad: "Error 403 - Unauthorized"
✅ Good: "You don't have access to this yet. Contact your admin to request permissions"

## Applications

### Website

- Hero: Full-screen navy background, white headline, gold accent
- Features: Card grid with icons, navy headers
- CTA sections: Navy background, white text, gold buttons
- Footer: Navy background, multi-column white text

### Dashboard

- Sidebar: Navy (#001F3F)
- Header: White background, navy text
- Cards: Light gray backgrounds, navy text
- Charts: Navy primary, gold accent colors
- Status indicators: Green (success), Red (error), Amber (warning)

### Email Templates

- Header: Navy background with gold accent line
- Body: White background, body font 16px
- CTA buttons: Navy or gold (avoid both in same email)
- Footer: Navy background, light gray text

### Mobile App

- Status bar: Navy
- Navigation: Navy bottom bar or drawer
- Headers: Navy background, white text
- Body: White background
- Accent actions: Gold

### Presentation Decks

- Background: White or navy
- Title slides: Navy + gold
- Content slides: White background, navy text
- Charts: Navy + gold + secondary colors
- Accent: Gold lines, geometric patterns
