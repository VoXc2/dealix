# User Guide

## Overview
This guide explains how to use the Dealix system for daily operations.

## Daily Workflow
1. Open Command Room Dashboard
2. Run `npm run revenue-day`
3. Review outreach drafts in approval queue
4. Approve/Reject each draft
5. Check CEO Daily Report

## Weekly Routine
1. Review Pipeline health
2. Run `python scripts/outreach_engine.py`
3. Update Brain OS signals/decisions
4. Proof Pack review

## Safety Reminders
- `OUTBOUND_MODE=draft_only` always
- Never auto-send without approval
- Every AI message tagged [AI]
