# Enterprise Discovery Playbook

## Standard Sequence

1. Map business goals, constraints, and owned outcomes.
2. Inventory systems, data sources, and approval owners.
3. Define initial workflow candidates and risk profile.
4. Agree readiness gates and evidence expectations.
5. Publish scoped implementation charter.

## Control Gates

| Gate ID | Requirement | Evidence ID | Test ID |
|---|---|---|---|
| G-PLY01-001 | sequence executed and documented | E-PLY01-001 | T-PLY01-001 |
| G-PLY01-002 | role ownership confirmed | E-PLY01-002 | T-PLY01-002 |
| G-PLY01-003 | unresolved high-risk items captured | E-PLY01-003 | T-PLY01-003 |

## Exit Criteria

Playbook run is complete only after gate review sign-off.
