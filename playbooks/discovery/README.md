# Discovery Playbook (Revenue OS)

## Outcome

Define one operationally critical workflow with measurable ROI target.

## Inputs

- Current lead handling process
- CRM stack and constraints
- Stakeholders and approval owners
- Baseline metrics (response time, conversion, lead volume)

## Checklist

1. Select one workflow only (`lead_qualification`).
2. Map current-state steps and failure points.
3. Identify required approvals and risk boundaries.
4. Confirm tenant/data isolation requirements.
5. Lock success metrics and target deltas.

## Exit Criteria

- Signed workflow scope
- Named approver role(s)
- Baseline KPI sheet approved
