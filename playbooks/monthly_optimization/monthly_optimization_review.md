# Monthly Optimization Review Playbook

## Standard Sequence

1. Review KPI trends and workflow bottlenecks.
2. Inspect policy denies, approvals, and incident patterns.
3. Prioritize low-risk improvements for next release cycle.
4. Re-run target eval slices before promotion.
5. Update readiness scorecard and optimization backlog.

## Control Gates

| Gate ID | Requirement | Evidence ID | Test ID |
|---|---|---|---|
| G-PLY04-001 | sequence executed and documented | E-PLY04-001 | T-PLY04-001 |
| G-PLY04-002 | role ownership confirmed | E-PLY04-002 | T-PLY04-002 |
| G-PLY04-003 | unresolved high-risk items captured | E-PLY04-003 | T-PLY04-003 |

## Exit Criteria

Playbook run is complete only after gate review sign-off.
