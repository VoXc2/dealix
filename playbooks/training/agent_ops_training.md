# Agent Operations Training Playbook

## Standard Sequence

1. Train operators on agent boundaries and approval flow.
2. Run escalation drills for high-risk actions.
3. Practice interpreting trace, log, and eval artifacts.
4. Verify role-based access and audit etiquette.
5. Certify team readiness with gate-linked test run.

## Control Gates

| Gate ID | Requirement | Evidence ID | Test ID |
|---|---|---|---|
| G-PLY02-001 | sequence executed and documented | E-PLY02-001 | T-PLY02-001 |
| G-PLY02-002 | role ownership confirmed | E-PLY02-002 | T-PLY02-002 |
| G-PLY02-003 | unresolved high-risk items captured | E-PLY02-003 | T-PLY02-003 |

## Exit Criteria

Playbook run is complete only after gate review sign-off.
