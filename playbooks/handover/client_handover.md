# Client Handover Playbook

## Standard Sequence

1. Confirm all agreed workflows are live and evidenced.
2. Hand over runbooks, dashboards, and ownership matrix.
3. Review rollback and incident response procedures.
4. Validate access controls for client-side operators.
5. Complete sign-off on acceptance criteria.

## Control Gates

| Gate ID | Requirement | Evidence ID | Test ID |
|---|---|---|---|
| G-PLY03-001 | sequence executed and documented | E-PLY03-001 | T-PLY03-001 |
| G-PLY03-002 | role ownership confirmed | E-PLY03-002 | T-PLY03-002 |
| G-PLY03-003 | unresolved high-risk items captured | E-PLY03-003 | T-PLY03-003 |

## Exit Criteria

Playbook run is complete only after gate review sign-off.
