# Contributing (Dealix)

## ما نقبله
- تحسينات وثائق
- إصلاحات bugs
- ميزات لها V1 plan + test
- ترجمات

## ما لا نقبله
- scraping code
- auto-send code
- fake ROI / testimonial content
- أي شي يخالف PDPL

## قبل PR
- شغّل `python3 scripts/pre_push_guard.py`
- تأكد CI passes
- وثق التغيير في `CHANGELOG.md`

## الفروع
- `main` — protected
- `feature/*` — for new work
- `hotfix/*` — for production fixes
