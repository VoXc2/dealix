# dealix-invoice-brief

Bilingual invoice brief / cover note that accompanies the actual
invoice PDF generated by `finance_os`. Manual send only — no live
charge action.

## Why this skill exists

Invoices are the most legally-sensitive customer artifact. This
skill enforces: every number resolves to a finance_os invoice record;
the artifact never triggers a charge.

## Safety

- `no_live_charge_action` — the skill output never includes a
  `charge_payment_live` marker.
- Amounts MUST match the invoice record exactly.
- VAT computed from the recorded rate, never invented.
