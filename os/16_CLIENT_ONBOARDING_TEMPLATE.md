# Client Onboarding Template
# قالب استقبال العميل الجديد

**الغرض:** تجميع جميع المعلومات اللازمة لبدء المشروع بشكل صحيح وآمن
**الوقت:** يُكمَّل خلال الأسبوع الأول من بداية العمل
**يُعِدّه:** Onboarding Agent + المؤسس
**يتطلب موافقة:** نعم — Gate G05 قبل طلب أي credentials

---

## رسالة الاستقبال (ترسل للعميل)

```
السلام عليكم [Name],

شكراً لثقتكم في Dealix — يسعدنا بدء العمل معكم.

لنبني المشروع بشكل صحيح وآمن، نحتاج منكم المعلومات التالية.
سيستغرق ذلك 15–30 دقيقة من وقتكم.

نبدأ دائماً ببيئة sandbox أو بيانات sample، ولا نستخدم أي بيانات
حقيقية أو أنظمة production إلا بعد موافقة صريحة منكم.

تحياتي،
[اسم المؤسس]
Dealix
```
