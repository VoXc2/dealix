# Client Onboarding Checklist

## Day 0
- [ ] Signed contract + PO
- [ ] Stakeholder map (decision maker, blockers, influencers)
- [ ] Channel of truth created (Slack/WhatsApp)
- [ ] Confidential data scope document signed
- [ ] First 7-day goals confirmed

## Day 2
- [ ] Workflow map delivered (as-is)
- [ ] Pain points tagged with impact (revenue/cost/risk)
- [ ] Selected OS modules confirmed
- [ ] First automation candidates listed (top 3)

## Day 5
- [ ] Command Center URL live
- [ ] Owners assigned per metric
- [ ] Daily standup template sent
- [ ] Escalation path documented

## Day 9
- [ ] Top 3 automations live
- [ ] Fallback rules for human review
- [ ] Audit log on every external action
- [ ] First end-to-end test report

## Day 15
- [ ] First weekly proof report sent
- [ ] Client sign-off on cadence
- [ ] First proof item logged
- [ ] Next 30-day plan confirmed
