# Terms Review Checklist (Dealix)

Before enabling any connector marked `terms_review_required=true`:

- [ ] Read provider's Terms of Service
- [ ] Identify allowed data fields
- [ ] Identify rate limits
- [ ] Identify pricing tier
- [ ] Confirm no auto-send
- [ ] Confirm human review gate
- [ ] Set token rotation policy (90 days)
- [ ] Document the integration in `business/sources/SOURCE_REGISTRY.md`
- [ ] Update `business/governance/SOURCE_AND_DATA_USAGE_REGISTER.md`
