# Quote Rules (Dealix)

## قبل إصدار عرض سعر
- [ ] تم تشخيص 20 دقيقة
- [ ] تم تأكيد الإشارة العامة
- [ ] تم اختيار tier العرض
- [ ] تم احتساب setup + monthly
- [ ] تم تسجيل العميل في CRM

## صيغة عرض السعر
```
[اسم العرض] لـ [اسم العميل]
- Setup: SAR [X]
- Monthly: SAR [Y]
- Timeline: [Z] days
- Includes: [قائمة مخرجات]
- Excludes: [قائمة استثناءات]
- Next step: [مكالمة Day 0 — 60 دقيقة]
```

## ما لا يحتويه عرض السعر
- أرقام ROI مضمونة
- خصومات غير موثّقة في Change Request
- وعود زمنية غير واقعية

## الصلاحية
- عرض السعر صالح 14 يوم
- بعد 14 يوم: مراجعة setup + monthly قبل الإصدار
- لا تمديد تلقائي
