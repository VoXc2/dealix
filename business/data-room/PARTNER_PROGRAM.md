# Partner Program (Dealix)

## أنواع الشراكة
1. **Agency Partner** — وكالات تسويق/PR
2. **Consulting Partner** — مستشارين
3. **Technology Partner** — مزودي CRMs / data
4. **Training Partner** — معسكرات تدريب

## الفوائد المشتركة
- White-labeled delivery layer
- Co-branded case studies
- Joint proposals
- API integration
- Quarterly reviews

## الشروط
- لا spam
- لا auto-send بدون review
- شفافية كاملة مع العميل
- عقد موقّع قبل أي co-sell
