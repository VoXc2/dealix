# Proposal Sections Library

| Section | Why it matters |
|---------|---------------|

| Governance | Builds trust: no auto-send, no spam |
| Pricing | Transparent, no hidden fees |
| Client responsibilities | Defines the working agreement |
| Next step | Always one specific action |
