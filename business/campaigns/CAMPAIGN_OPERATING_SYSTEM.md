# Campaign Operating System (Dealix)

## One campaign = one file
- `business/campaigns/CAMPAIGN_NAME.md`
- Pack generated by `scripts/generate_campaign_pack.py`

## Required fields
- target segment
- pain angle
- post copy
- WhatsApp / Email / LinkedIn drafts
- landing page CTA
- follow-up cadence
- review checklist

## Review checklist (always)
- [ ] No fake claims
- [ ] No guaranteed ROI
- [ ] No pressure
- [ ] Source note per channel
- [ ] Review status = approved before send
- [ ] Opt-out honored within 24h

## Cadence
- 1 campaign per quarter (4 per year)
- 1 per offer tier
