# AI Human Review Policy (Dealix)

## المبدأ
Dealix **AI-assisted · Human-reviewed · Proof-driven · Built for Saudi operations**.

## أين يدخل الإنسان
| النشاط | بشري؟ |
|--------|-------|
| إنشاء مسوّدة تواصل | مولّد (AI) |
| إرسال مسوّدة | بشري فقط |
| إنشاء عرض رسمي | مولّد (AI) |
| توقيع عرض | بشري (المؤسس) |
| قرار مالي | بشري (المؤسس) |
| قرار تشغيلي غير روتيني | بشري (المالك) |
| ملخص أسبوعي للعميل | مولّد (AI) + مراجعة |
| Proof report | مولّد (AI) + توقيع العميل |
| Expansion offer | مولّد (AI) + موافقة المؤسس |

## ما لا يفعله الـ AI
- وعد ROI
- ادعاء شهادة عميل
- تجاوز لائحة PDPL
- إرسال رسالة بدون review_status = approved
- حذف سجل audit log
- تعديل رقم مالي في تقرير قبل الموافقة
