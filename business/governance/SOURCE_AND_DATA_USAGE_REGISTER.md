# Source & Data Usage Register (Dealix)

## المصادر المعتمدة
| Source | Type | Legal basis | Notes |
|--------|------|-------------|-------|
| Saudi Open Data | Public aggregate | Public domain | No personal data |
| Founder-supplied CSV | Founder-owned | Contract | Source required |
| Manual research | Public web | Public domain | URL/quote required |
| Website signal (local) | Manual save | Founder owns | No crawling |
| Google Places API | Public API | Terms of service | Rate-limit aware |
| HubSpot CRM | OAuth scope | Contract | Read-only by default |
| WhatsApp Business | Approved template | Terms of service | Templates only |
| Referral | Written consent | Consent | Founder records |

## سجل الاستخدام
- كل lead: source + URL/quote + date
- كل outreach: source + channel + reviewer
- كل report: source list in header
