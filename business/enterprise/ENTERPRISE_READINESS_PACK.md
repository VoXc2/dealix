# Enterprise Readiness Pack (Dealix)

## What this is
The pack an enterprise buyer would request:
- Security questionnaire
- Data boundary statement
- AI governance statement
- Human review statement
- Service level boundaries
- Implementation assurance plan
- Buyer FAQ (AR + EN)

- Doc is a living document, updated after every incident
- All docs are marked version + last reviewed date
