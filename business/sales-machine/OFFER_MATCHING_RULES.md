# Offer Matching Rules (Dealix)

## Rule 1
If primary weakness is **inbound response** → **Revenue OS**

## Rule 2
If primary weakness is **reputation response** → **Review & Reputation OS**

## Rule 3
If primary weakness is **founder decision velocity** → **Command Center OS**

## Rule 4
If primary weakness is **delivery visibility** → **Delivery OS**

## Rule 5
If company is **50+ employees** or **multi-entity** → **Custom Enterprise System**

## Rule 6
If client wants **ongoing partner** (no internal ops lead) → **Managed OS Retainer**

## Rule 7
If first touch, unknown, or warm lead → **Diagnostic Sprint** (free)

## Default
If multiple weaknesses, recommend the **highest setup** + **lowest monthly** offer first, then
expand with retainer.
