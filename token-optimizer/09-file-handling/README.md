## الاستراتيجيات الأساسية

### 1. قراءة نطاق محدد
```
اقرأ `api/routes/clients.py` السطور 45-80 فقط.
```
بدلاً من:
```
اقرأ `api/routes/clients.py`
```

### 2. استخدام sed/pipe لتصفية المحتوى
```bash
# أرسل فقط الدوال المتعلقة بـ auth
grep -n "def.*auth\|async def.*auth" api/routes/clients.py

# أرسل فقط الـ 100 سطر الأخيرة من log
tail -100 app.log

# أرسل فقط الأسطر التي تحتوي على الخطأ
grep -n "ERROR\|Exception" app.log | tail -50
```

### 3. خريطة المشروع بدلاً من قراءة الملفات
```bash
# توليد هيكل المشروع بدون المحتوى
tree -L 3 --gitignore -I "__pycache__|node_modules|.git"
```

### 4. قراءة الـ Headers فقط
```bash
# رأس كل ملف Python (imports + class/function names)
grep -n "^import\|^from\|^class\|^def\|^async def" api/routes/clients.py
```
