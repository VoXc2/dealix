# 02 — CLAUDE.md: تحسين السياق الدائم

## لماذا هذا حاسم؟

CLAUDE.md يُحمَّل قبل كل رسالة — في كل مرة، بدون استثناء.
- ملف CLAUDE.md بـ 5,000 توكن = 5,000 توكن **تُكلَّف على كل رسالة**
- في جلسة 50 رسالة → 250,000 توكن مُهدرة على CLAUDE.md وحده

**الهدف: أقل من 500 توكن لـ CLAUDE.md**
