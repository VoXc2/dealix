# 7-Day Growth Proof Sprint — 499 SAR Pilot

> Offered AFTER the prospect has read the Mini Diagnostic and asked
> "what's next?". Manual payment fallback only — no live charge.

## Offer summary

| Field | Value |
| --- | --- |
| Name | 7-Day Growth Proof Sprint |
| Price | 499 SAR |
| Payment | Manual bank transfer / STC Pay (default) — Moyasar invoice optional if configured |
| Live charge | NOT enabled (gate stays false) |
| Window | 7 calendar days from payment confirmation |
| Output | Proof Pack on day 7 |
| Risk to founder | low — pilot scope is strictly bounded |
| Risk to customer | low — written scope, no auto-renew, no long contract |

## Pilot scope (strict — do not expand silently)

1. 10 specific opportunities matching the customer's ICP
2. 6 message drafts in Arabic / English (founder approves before any send)
3. 3 follow-up angles
4. Risk notes — what NOT to do, with reasons (compliance, policy, tone)
5. Safe channel recommendation per opportunity
6. Proof Pack on day 7

## What the pilot does NOT include

- Live external sends of any kind
- Cold WhatsApp / purchased lists
- LinkedIn automation
- Long-term retainer
- Multiple ICPs (one ICP per pilot)
- Custom integrations
- Calls (founder calls at their own discretion only)

## Hard rules

| Rule | Reason |
| --- | --- |
| 499 SAR fixed price | no haggling — protects positioning |
| Manual fallback first, Moyasar invoice optional | live charge gate stays off |
| 1 ICP per pilot | scope creep is the #1 way pilots fail |
| Day 7 hard deadline on Proof Pack | trust = on-time delivery |
| No auto-renew | written 7-day scope ends, customer chooses next step |
| No upsell during the pilot | upsell happens AFTER Proof Pack delivery |
| No guarantees — ever | "نضمن" / "guaranteed" remains forbidden |
