# Saudi Evidence Layer

للسوق السعودي، الأدلة يجب أن تكون **عربية وتنفيذية** — لا ترجمة لاحقة فقط.

## Arabic Evidence Summary (مثال)

تم استخدام مصدر البيانات SRC-001 لغرض التحليل الداخلي فقط.  
تم اكتشاف بيانات تواصل شخصية، لذلك تم تصنيف المخرجات كمسودات فقط.  
لم يتم تنفيذ أي إرسال خارجي من Dealix.  
تم إنشاء 10 توصيات قابلة للمراجعة.  
تم حظر طلبين لتواصل خارجي غير معتمد.

## ثقة سعودية

مصادر واضحة، موافقات بشرية، لا إرسال خارجي بلا اعتماد، لا وعود مبيعات مضمونة، Proof يوضح القيمة والحدود.

## الكود

نفس `evidence_object.summary` — يُفضّل حقل عربي للعميل عند التسليم.
