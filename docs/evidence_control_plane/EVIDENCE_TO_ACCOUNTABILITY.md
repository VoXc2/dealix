# Evidence-to-Accountability

المساءلة تعني **معرفة من يملك القرار** — لا لومًا عشوائيًا.

## حقول (مرجعي)

- AI generated by  
- Human reviewed by  
- Approved by  
- Governed by  
- Owned by  
- Client sponsor  
- Dealix owner  

## مثال JSON

```json
{
  "output_id": "OUT-001",
  "generated_by": "AGT-REV-001",
  "reviewed_by": "Dealix Delivery Operator",
  "approved_by": "Client Sales Owner",
  "governance_owner": "Dealix Governance Owner",
  "final_accountability": "Client approval owner for external use"
}
```

## القواعد

- **No owner = no execution.**  
- **No approval owner = no external action.**
