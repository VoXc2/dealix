# Enterprise Adoption Gates

Seven gates in order. Typed: `enterprise_rollout_os.adoption_gates.evaluate_adoption_gates()`.

1. Sponsor — exec + business owner + problem.
2. Data — Source Passport + PII + allowed use.
3. Workflow — owner + approval owner + success metric.
4. Governance — external actions + boundaries + blocked actions.
5. Proof — Proof Pack complete + score ≥ 70 + limitations.
6. Adoption — outputs used + approvals + workflow used twice.
7. Retainer — proof ≥ 80 + adoption ≥ 70 + monthly value + committed owner.
