# Enterprise Risk Register (R1–R10)

No executive sponsor, data owner unclear, governance rejects use case, users do not adopt, slow approvals, weak proof, unsafe automation requests, IT/security blocks access, expectation of guaranteed outcomes, agent permissions too broad.

Each risk has a control: Sponsor Gate, Source Passport, Governance Boundary, Adoption Review, Approval Matrix, Proof Score, No Unsafe Automation Policy, Security Brief, Expectation Setting, Agent Card.
