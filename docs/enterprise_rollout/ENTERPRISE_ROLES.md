# Enterprise Rollout Roles

- **Executive Sponsor** — قرار وميزانية · value · مخاطر · ROI visibility.  
- **Business Owner** — العملية · وضوح workflow · تقليل يدوي.  
- **Data Owner** — مصدر البيانات · PII · retention.  
- **Governance Reviewer** — audit · approvals · قرارات سياسة.  
- **Operators** — drafts · قوائم · مسار مراجعة.  
- **Dealix Success Owner** — proof · adoption · cadence · توسع · صحة العميل.
