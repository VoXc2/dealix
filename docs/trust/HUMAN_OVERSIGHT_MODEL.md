# Human Oversight Model — إشراف بشري صريح

## المبدأ

- **AI يجهّز** المسودات والتحليلات والخطوات المقترحة.  
- **إنسان يوافق** على الإجراءات الخارجية والمخرجات الحساسة حسب المسار.  
- **النظام يسجّل** التشغيل والقرارات (audit / AI run ledger).  
- **الـ Proof يثبت** الأثر للعميل ولـ Dealix.
