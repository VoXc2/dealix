# الوكلاء (Agentic Operations)

## الهوية والصلاحيات

كل agent: agent_id، owner، purpose، business_unit، risk tier، بيئة، حالة. صلاحيات: مدخلات مسموحة، أدوات مسموحة/ممنوعة، متطلبات موافقة، مستوى استقلالية.
