## سلم الخدمات — نظرة عامة

```
[0] Free AI Ops Diagnostic            0 SAR        ← باب الدخول
[1] 7-Day Revenue Intelligence Sprint 499 SAR      ← بوابة التجربة المدفوعة
[2] Data-to-Revenue Pack              1,500 SAR    ← ترقية بعد السبرنت
[3] Managed Revenue Ops               2,999–4,999 SAR/شهر ← بعد تجربة
[4] Custom AI Service Setup           5,000–25,000 SAR    ← بعد 3 تجارب + إذن نشر موقَّع
```

> Dealix قبل مرحلة الإيراد ولا عملاء لها. الجدول يصف عروضاً لا زخماً.

## عناصر مغلقة / مستقبلية

العناصر التالية ظهرت في مسودات سابقة لكنها **ليست عروضاً فعّالة** ولا تُسعَّر ولا تُباع
كمنتجات مستقلة. قد تُدمَج مستقبلاً ضمن الدرجة 3 أو 4 بعد استيفاء شرط الفتح.

- **Executive Command Center** — سطح محتمل ضمن الدرجة 3/4، لا عرض مستقل.
- **Support OS Add-on** — مكوّن محتمل للدرجة 3، لا عرض مستقل.
- **Agency Partner OS** — حركة شركاء/قنوات، ليست درجة موجَّهة للعميل. مغلقة حتى
  تسليم 3 تجارب + إذن موقَّع.
