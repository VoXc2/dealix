## 🎯 Decision Trees (متى تستخدم أي ملف)

### "عندي VC meeting بكرا"
1. اقرأ `FOUNDER_PITCH_DECK_OUTLINE_AR.md` (60 دقيقة)
2. راجع `FINANCIAL_MODEL_2026_2028_AR.md` (30 دقيقة)
3. احفظ الأرقام
4. اعمل rehearsal مع mentor
5. أرسل data room (200+ وثيقة)

### "جاني customer جديد"
1. اقرأ `SALES_SCRIPTS_ALL_MOTIONS_AR.md` (Motion المناسب)
2. حدّد ICP
3. اسأل 3 أسئلة discovery
4. قدّم عرض مخصص
5. أرسل proposal (5K ر.س+)

### "عندي hire جديد"
1. اقرأ `CULTURE_HIRING_HANDBOOK_AR.md` (3 ساعات)
2. راجع JDs
3. حدد interview panel
4. هيّئ onboarding
5. أرسل welcome kit

### "صار outage"
1. افتح `CRISIS_PLAYBOOK_DETAILED_AR.md` → Scenario 2
2. اتبع protocol
3. Internal all-hands (30 min)
4. Status page update
5. Post-mortem (within 7 days)

### "أحتاج أرفع مستثمر جديد"
1. اقرأ `FOUNDER_PITCH_DECK_OUTLINE_AR.md`
2. راجع `FINANCIAL_MODEL_2026_2028_AR.md`
3. جهّز data room
4. Schedule meeting
5. Follow-up (5 أيام)

### "أحتاج أعمل new feature"
1. اقرأ `CUSTOMER_INTERVIEW_WINLOSS_AR.md` (insights)
2. حدد الـ 5 requests الأكثر
3. Prioritize
4. Build
5. Ship + measure

### "ما عندي motivation"
1. اقرأ `FOUNDER_PERSONAL_OPERATING_SYSTEM_AR.md` → Energy
2. خذ يوم off
3. اتصل بـ mentor
4. ارجع للـ Vision
5. ركّز على Health
