# Compliance Metrics

قِس الامتثال كما تقيس الإيراد:

- % مصادر لها Source Passport  
- % AI runs مسجّلة  
- % مخرجات لها governance status  
- % إجراءات خارجية بموافقة قبل التنفيذ  
- % ادعاءات لها proof status  
- PII flags · redactions · blocked actions · حوادث · زمن الإغلاق  

## عتبات تشغيلية (أمثلة)

تسجيل AI runs < 100% → لا enterprise.  
تغطية Passport < 100% → لا scale.  
إجراء خارجي بلا موافقة → مراجعة حادث.  
ادعاء غير مدعوم في مخرج عميل → فشل QA.
