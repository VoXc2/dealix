# دليل استخدام الشعار | Dealix Logo Usage Guide

## الشعار الرئيسي

الشعار الرئيسي لـ Dealix يتكون من رمز أيقوني (Icon Mark) إلى جانب النص الإنجليزي "Dealix" والنص العربي "ديليكس". يجب استخدام الشعار الرئيسي حيثما أمكن.

### عناصر الشعار
1. **الرمز الأيقوني**: مربع كحلي دائري الزوايا (10px) يحتوي على حرف D ذهبي داخل شكل ماسي
2. **النص الإنجليزي**: "Dealix" بخط Inter عريض، مقاس 22px، بلون كحلي #0c2742
3. **النص العربي**: "ديليكس" بخط Noto Kufi Arabic، مقاس 11px، بلون رمادي #64748b

---

## الأبعاد والمسافات

### المساحة الآمنة (Clear Space)
يجب أن لا يقل الحد الأدنى للمسافة حول الشعار عن ارتفاع الرمز الأيقوني (40px) من جميع الجهات. هذا يضمن وضوح الشعار وتأثيره.

```
├─ 40px ─┤
┌─────────┬──────────────────────────┐
│         │                          │ ← 40px
│  ICON   │  Dealix                  │
│         │  ديليكس                   │
├─────────┴──────────────────────────┤
│                                    │ ← 40px
└────────────────────────────────────┘
```

### الحد الأدنى للحجم
- **الشعار الكامل (طباعة)**: لا يقل عن 25mm عرض
- **الشعار الكامل (رقمي)**: لا يقل عن 120px عرض
- **الرمز الأيقوني فقط**: لا يقل عن 24px × 24px
- **الرمز الأيقوني (فافيكون)**: 32px × 32px

## الاستخدامات الصحيحة ✅

الاستخدام الصحيح للشعار يحافظ على تناسق العلامة التجارية:

| السيناريو | الشعار المناسب | الخلفية |
|-----------|---------------|---------|
| موقع Dealix (هيدر) | أساسي | بيضاء/فاتحة |
| موقع Dealix (فوتر داكن) | أبيض | كحلي/داكن |
| بطاقة عمل | أساسي | أبيض |
| عرض تقديمي (شريحة عنوان) | أبيض | كحلي |
| عرض تقديمي (شريحة محتوى) | أساسي | أبيض |
| واتساب بزنس | أيقونة | دائرة |
| LinkedIn (صورة شخصية) | أيقونة | كحلي |
| ميرش (تيشيرت) | أيقونة أو شعار | كحلي/أسود |
| لافتة | أساسي | أبيض/شفاف |

## الاستخدامات غير الصحيحة ❌

لا يجوز تحت أي ظرف:

| الاستخدام غير الصحيح | الوصف |
|---------------------|-------|
| ❌ تغيير الألوان | لا تستخدم ألواناً غير #0c2742 و #d4a843 |
| ❌ إعادة تلوين | لا تجعل الشعار كله ذهبياً أو كله كحلياً |
| ❌ تدوير الشعار | لا تدير الشعار بزاوية |
| ❌ إمالة أو انحراف | لا تطبق تحويل منظور أو انحراف |
| ❌ تغيير الخط | لا تستبدل الخطوط بأي خط آخر |
| ❌ إضافة ظلال | لا تضف ظلالاً مسقطة أو توهجات مبالغ فيها |
| ❌ حدود خارجية | لا تضف إطاراً حول الشعار |
| ❌ خلفية مزدحمة | لا تضع الشعار على خلفية ذات أنماط مزدحمة أو صور |
| ❌ تغيير النسب | لا تسحب أو تضغط الشاعر (حافظ على Aspect Ratio) |
| ❌ إزالة عناصر | لا تحذف الرمز الأيقوني أو النص |
| ❌ إعادة ترتيب | لا تغير ترتيب النص الإنجليزي والعربي |
| ❌ شعار جزئي | لا تستخدم جزءاً من الشعار فقط |
| ❌ خلفية منخفضة التباين | لا تضع الشعار الأساسي على خلفية كحلي أو داكنة |
