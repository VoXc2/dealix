# Dealix Presentation Style Guide

## Deck Rules

- 12 slides maximum for first sales deck.
- One message per slide.
- No dense paragraphs.
- Use proof, not hype.
- Every offer must show: pain, outcome, deliverables, price, next step.

## Slide Structure

1. Title
2. Problem
3. Cost of inaction
4. Solution
5. P1 offer
6. Proof Pack
7. P2 monthly
8. P3 command center
9. Governance
10. Pricing
11. Demo
12. Close

## Visual Direction

- Dark executive background.
- Cyan/emerald accents.
- Big numbers.
- Proof cards.
- Decision maps.
- Pipeline flow.
- Arabic-first, English-ready.
