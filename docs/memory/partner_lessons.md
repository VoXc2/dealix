# Strategic memory — Partners (قالب)

سجّل الدروس **بعد كل تعاون شراكة أو قناة** (بدون PII):

- أي شريك/نوع شراكة **يفتح فرصًا** (قطاع، حجم، نموذج عرض)؟  
- ما الذي **لم يعمل** في التفعيل أو التسعير المشترك؟  
- شروط **حوكمة الثقة** (عدم واتساب بارد، عدم DMs تلقائية، مصدر بيانات)؟  
- روابط لمحتوى/صفحات ثقة تم تحديثها نتيجة السؤال الشائع من الشركاء  

**لا تنسخ بيانات عملاء أو PII هنا.**
