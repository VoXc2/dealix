# Client Journey (Dealix)

```text
1. Awareness — content or referral.

2. Diagnostic — where to start is clear.

3. Sprint — first proof is delivered.

4. Review — value and evidence are visible.

5. Pilot — a workflow or capability runs ~30 days.

6. Retainer — monthly operating cadence.

7. Expansion — add capabilities: support, knowledge, governance, reporting.

8. Enterprise — multiple teams, systems, permissions.
```

## One CTA per stage

```text
Awareness CTA: Book Diagnostic
Diagnostic CTA: Start Sprint
Sprint CTA: Start Pilot
Pilot CTA: Monthly Retainer
Retainer CTA: Expand Capability
```
