# رحلة العميل — ثماني خطوات، دعوة واحدة لكل مرحلة

**الطبقة:** L7 · محرك التنفيذ
**المالك:** المؤسس / CSM
**الحالة:** مسودة
**آخر مراجعة:** 2026-05-13
**النسخة الإنجليزية:** [CLIENT_JOURNEY.md](./CLIENT_JOURNEY.md)

## السياق
تُعرّف هذه الوثيقة رحلة عميل ديليكس المعتمدة بثماني خطوات، مع دعوة
واحدة محدّدة لكل مرحلة. تُزيل أكثر تسرّبات الإيراد شيوعاً في شركات
الخدمات — وجود عدّة دعوات متنافسة في نفس المرحلة، مما يُنتج التردد.
ترتبط بدليل نجاح العملاء في `docs/CUSTOMER_SUCCESS_PLAYBOOK.md` وبمواد
المبيعات في `docs/sales/DEMO_SCRIPT.md`. هذا تنفيذ في الطبقة L7 لأن
انضباط الرحلة يمتد عبر كل حلقة نشطة.

## الخطوات الثماني

```
الوعي ─► التشخيص ─► السبرنت ─► المراجعة ─► التجربة ─► الاحتفاظ ─► التوسعة ─► المؤسسي
```

لكل خطوة دعوة واحدة فقط. لا يرى المشتري أكثر من خيار في مرحلة واحدة.
مهمّة المؤسس أو CSM جعل هذه الدعوة الواحدة واضحة وسهلة القبول.

## مرحلة بمرحلة

### 1. الوعي
يلتقي المشتري بديليكس عبر المحتوى أو الإحالة أو التواصل الخارجي.
- **الدعوة:** احجز تشخيصاً.
- **المالك:** المؤسس.
- **الأصل:** One-pager، فيديو تعريفي.

### 2. التشخيص
ارتباط مدفوع لتوضيح الحالة (SAR 3.5k - 7.5k). يخرج المشتري بتشخيص مكتوب.
- **الدعوة:** ابدأ سبرنت.
- **المالك:** المؤسس.
- **الأصل:** تقرير التشخيص.

### 3. السبرنت
ارتباط 2-4 أسابيع بسعر ثابت لإثبات أوّل (SAR 7.5k - 25k).
- **الدعوة:** ابدأ تجربة.
- **المالك:** المؤسس.
- **الأصل:** حزمة إثبات، أصل عملي.

### 4. المراجعة
مراجعة مشتركة لنتيجة السبرنت. ليست مرحلة بيع، بل نقطة وضوح لتحديد ما
إذا كانت الدعوة التالية ستُقدَّم.
- **الدعوة:** (لا توجد — بوّابة جاهزية)
- **المالك:** المؤسس + راعي المشتري.
- **الأصل:** ملخص المراجعة.

### 5. التجربة
تحقّق تشغيلي 4-8 أسابيع داخل بيئة المشتري (SAR 22k - 60k).
- **الدعوة:** ابدأ احتفاظاً شهرياً.
- **المالك:** المؤسس / CSM.
- **الأصل:** تقرير التجربة بمقاييس تشغيلية.

### 6. الاحتفاظ
ارتباط شهري مستمر (SAR 8k - 60k / شهر).
- **الدعوة:** وسّع القدرة.
- **المالك:** CSM.
- **الأصل:** حزمة إثبات شهرية وتقرير قيمة.

### 7. التوسعة
يُفتَح مجال قدرة ثانٍ (مثل: المشتري الأصلي كان نموّ الإيراد؛ الآن نقترح
دماغ الشركة).
- **الدعوة:** أضِف قدرة (سبرنت جديد أو سطر احتفاظ جديد).
- **المالك:** CSM + المؤسس.
- **الأصل:** ملخص التوسعة.

### 8. المؤسسي
ارتباط متعدد القدرات ومتعدد الأرباع (SAR 85k - 300k+).
- **الدعوة:** جدّد بشروط مؤسسية.
- **المالك:** المؤسس.
- **الأصل:** اتفاقية مؤسسية، حزمة حوكمة.

## انضباط الدعوة الواحدة

إذا رُفِضت دعوة المرحلة، لا يتراجع المشتري إلى مرحلة سابقة، بل يتوقّف
مؤقتاً. الدعوة التالية هي نفس الدعوة، تُقدَّم لاحقاً، بدليل جديد. لا
يرى المشتري دعوة ثانية بديلة في نفس المحادثة.
