# Phase E — First Customer Execution Kit

The Phase E kit is the **operational playbook** for taking Dealix from
"production live" to "first paying pilot" without flipping any live
gate, scraping, cold-WhatsApp-ing, or fabricating proof.

Everything in this folder assumes the founder is running
**warm-intro-only**, manual-WhatsApp (inbound/opt-in), test-mode
Moyasar, and approval-required content.

## Hard rules — never break

- ❌ NO live WhatsApp send / Gmail send / LinkedIn DM / Moyasar live charge
- ❌ NO scraping, cold WhatsApp, fake customers, fake proof
- ❌ NO automation of customer messages — every send is manual + approved
- ❌ NO public claim of revenue, ROI, or testimonial without customer-signed permission
- ❌ NO "نضمن" / "guaranteed" / "blast" / "scrape" tokens in any artifact
- ✅ Arabic primary, English secondary
- ✅ Every external action: `blocked` / `draft_only` / `approval_required`
- ✅ Pilot pricing 499 SAR until ≥3 proof events justify a tier change

## Founder's operating rule of thumb

If a thing in this kit can be done by a tool, the tool prepares a
**draft** and the founder sends. If a thing requires a customer's
private data, it stays in your head/phone — never a screenshot, never
the repo, never an AI prompt.
