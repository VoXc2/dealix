# Core OS — القلب المشترك للقابضة

**التعريف:** طبقة مشتركة لكل وحدات Dealix — **الثقة، البيانات، الـ AI، الإثبات، رأس المال، الجودة، مساحة العميل.**

## مكونات مرجعية

- Data OS · Governance OS · LLM Gateway · AI Workforce Registry  
- Proof Ledger · Capital Ledger · Quality OS · Client Workspace  
- Founder / Holding Command Center  

## قواعد

```text
لا وحدة تبني حوكمتها المعزولة عن السياسة الموحدة.
لا استدعاء LLM يتجاوز بوابة التوجيه/التكلفة.
لا تسليم بلا Proof Pack وQA حيث ينطبق المسار.
```
