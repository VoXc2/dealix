# 90-Day Plan

See cadence: [`CEO_OPERATING_CADENCE_AR.md`](CEO_OPERATING_CADENCE_AR.md).

## Month 1 — Service readiness

- 3 ready services (folders + readiness >= 80)
- 3 demos (CSV, report sample, brain / workflow sample)
- Core modules wired (`data_os`, `governance_os`, `reporting_os`, `delivery_os`, `revenue_os`)
- Sales assets minimum set

## Month 2 — First paid projects

- 2–3 paid sprints
- proof packs on every delivery
- refine delivery checklists
- automate repeated internal steps

## Month 3 — Retainer conversion

- 1–2 retainers
- client workspace MVP (product roadmap)
- monthly reporting rhythm
- governance dashboard usage
- customer success checklist

## Days 1–7 (bootstrap)

- `docs/company`, `docs/strategy`, first 3 `docs/services/*`
- `docs/templates/*`
- Run `python scripts/verify_full_mvp_ready.py --skip-tests` until green

## Days 8–30

- Expand governance + quality docs
- Harden verify scripts in CI
