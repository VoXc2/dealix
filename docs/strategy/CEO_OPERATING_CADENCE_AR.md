# إيقاع تشغيل المؤسس (يومي / أسبوعي / شهري)

## يومي

- الموافقات المعلّقة
- عوائق التسليم
- رسائل العملاء الحرجة
- تكلفة AI / ميزانية النماذج
- فشل QA
- متابعة مبيعات (مسودات فقط حتى الموافقة)

## أسبوعي

- الإيراد المغلق
- Pipeline
- جودة التسليم وإعادة العمل
- proof packs المكتملة
- رضا العميل (qualitative)
- خطوات يدوية متكررة → مرشّحة لمنتج

## شهري

- أفضل خدمة مبيعاً وهامشاً
- أفضل قطاع استجابة
- أفضل مسار retainer
- ميزة تقلل وقت التسليم
- مخاطر تحتاج تحسين حوكمة
