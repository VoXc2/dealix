# Change Request System

Protects against **scope creep** and silent discounting.

## Template

```markdown
# Change Request

## Original Scope
(link to `01_scope.md` or SOW)

## Requested Change


## Impact
- timeline:
- cost:
- quality:
- risk:

## Decision
- Accept free (rare — document why)
- Paid change
- New sprint
- Retainer item
- Reject

## Price
SAR:

## Approvals
Client: ______   Dealix: ______   Date: ______
```

## CEO rule

Do not let “طيب نسويها” become habit. **Every change** has a cost or an explicit **no**.

Log material scope changes in `clients/<client>/<project>/04_governance_log.md` if they affect risk or claims.
