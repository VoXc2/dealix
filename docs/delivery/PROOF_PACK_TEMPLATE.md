# Proof Pack Template — Delivery

**المرجع الاستراتيجي:** [`../strategy/DEALIX_FULL_OPS_MASTER_PLAN_AR.md`](../strategy/DEALIX_FULL_OPS_MASTER_PLAN_AR.md)

## الهيكل (10 أقسام)

1. Executive Summary  
2. Workflow Map  
3. Source Quality Findings  
4. Approval Boundary Findings  
5. Evidence Trail Gaps  
6. Revenue Leakage Points  
7. Top 3 Governed Decisions  
8. Recommended Sprint  
9. Evidence Appendix  
10. 30-Day Action Plan  

## قواعد الجودة

- كل **finding** له **مصدر** (ملف، مقابلة، لقطة CRM، سياسة داخلية — حسب التوفر)  
- كل رقم: مصدر أو `is_estimate=true`  
- أي **بيان ناقص** يُسمّى صراحة «مفقود»  
- لا ادّعاء أمني بدون مصدر  
- لا إصدار **نهائي** بدون مراجعة بشرية (Founder/lead)

## محتوى يجب أن يثبت (positioning)

Source clarity · Approval boundaries · Evidence trail · Decision passport scaffolding · Value report — **بدون** KPI وهمية أو «proof» مزيّف.
