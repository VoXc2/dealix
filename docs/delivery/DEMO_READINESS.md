## 1) Lead Intelligence Demo

- [ ] `demo` CSV أو قائمة حسابات (حتى 50–500 صف)
- [ ] import preview + أعمدة معروفة
- [ ] data quality score
- [ ] top accounts مرتبة + سبب مختصر
- [ ] outreach drafts (مسودات فقط)
- [ ] mini CRM / لوحة أو جدول مراحل
- [ ] تقرير تنفيذي مختصر
- [ ] proof pack أو شريحة «ماذا أدخلنا / ماذا أخرجنا»

سكربت حديث: [`../sales/DEMO_SCRIPT.md`](../sales/DEMO_SCRIPT.md).

## 2) AI Quick Win Demo

- [ ] process map (قبل)
- [ ] workflow مقترح + خطوة موافقة بشرية
- [ ] before/after وقت أو خطوات
- [ ] تقدير وقت موفر
- [ ] SOP أو checklist تشغيل
- [ ] تقرير مختصر

## 3) Company Brain Demo

- [ ] ~10 وثائق عينة (بدون أسرار حقيقية)
- [ ] بحث / استرجاع
- [ ] إجابة **باقتباس مصدر**
- [ ] مثال **لا مصدر = لا إجابة**
- [ ] جودة عربي/إنجليزي
- [ ] eval أو ملاحظات جودة مختصرة
