# مدخلات التسليم المطلوبة

> لا يبدأ التسليم بلا مدخلات.

## نظام تشغيل الإيرادات

- قائمة الخدمات والأسعار
- سجل الفرص الحالية
- صلاحية بريد/واتساب للتواصل

## نظام القيادة التنفيذية

- مصادر الأرقام الحالية
- أهداف الربع
- صلاحية قراءة البيانات

## نظام استعادة المتابعات

- سجل العملاء/المهتمين
- قنوات التواصل المسموح بها
- قائمة عدم التواصل (إن وجدت)

## نظام عملاء واتساب

- رقم/حساب واتساب للأعمال
- قائمة الخدمات
- صلاحية الردود

## نظام العروض والإثبات

- الخدمات والنطاقات
- هيكل التسعير
- أمثلة نتائج سابقة (إن وجدت)
