# رسالة الترحيب بعد الدفع (قالب)

مرحبًا،

سعداء ببدء العمل معكم ضمن الـSprint المتفق عليه.

سنحوّل البيانات/العملية المتفق عليها إلى **مخرجات تنفيذية واضحة**، مع مراجعة جودة وحوكمة، و**Proof Pack** يثبت المدخلات والمخرجات والأثر في النهاية.

**الخطوات القادمة:**

1. تأكيد الهدف النهائي للمشروع.
2. استلام الملفات أو البيانات المطلوبة (حسب `data_request` الخاص بالخدمة).
3. مراجعة مصدر البيانات وحدود استخدامها (PDPL / موافقات).
4. التنفيذ والبناء.
5. مشاركة نسخة أولية للمراجعة الداخلية ثم معكم.
6. تسليم التقرير النهائي وProof Pack واقتراح المرحلة التالية.

**نحتاج منكم:**

- مسؤول واحد للمشروع (Owner).
- تسليم الملفات في الموعد المتفق.
- أي قيود على البيانات أو الصياغة أو القنوات الخارجية.
- جدولة جلسة مراجعة نهائية.
