# جدول أعمال جلسة المراجعة (30–45 دقيقة)

1. **تلخيص الهدف** (2 د) — هل ما زال مطابقًا لما اتفقنا؟
2. **المدخلات** (5 د) — ما استلمناه، ما الناقص، جودة المصدر.
3. **المخرجات** (15 د) — عرض التقرير / المسودات / اللوحة / المساعد.
4. **الجودة والحوكمة** (5 د) — QA، PII، قيود القنوات، «لا مصدر لا إجابة» إن انطبق.
5. **Proof Pack** (5 د) — ماذا تغير، ماذا التالي.
6. **القرارات** (5 د) — موافقات مطلوبة، تعديلات ضمن النطاق vs change request.
7. **Upsell / المرحلة التالية** (3 د) — Pilot، Retainer، أو Sprint لاحق.

سجل القرارات في ملاحظات المشروع أو أداة التسليم الداخلية.
