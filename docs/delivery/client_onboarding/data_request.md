# طلب بيانات — Onboarding

استخدم **قالب الخدمة** في `docs/services/<service>/data_request.md` (أو `document_request.md`) كمصدر رسمي للحقول.

**عام:**

- صيغة الملفات (CSV / PDF / …) والترميز.
- ما يُعتبر سجلًا واحدًا (صف، مستند، تذكرة، …).
- أي أعمدة إلزامية واختيارية.
- قيود الاستخدام والأساس النظامي إن وُجدت بيانات شخصية.

لا تبدأ البناء قبل **تأكيد Owner + استلام أول دفعة بيانات** أو اتفاق واضح على «تشخيص بدون داتا».
