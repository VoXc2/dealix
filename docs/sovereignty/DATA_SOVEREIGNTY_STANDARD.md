# Data Sovereignty Standard

سيادة البيانات = **كل مصدر له غرض وحالة استخدام** قبل أي AI.

- Source Passport (انظر [`SOURCE_PASSPORT_STANDARD.md`](SOURCE_PASSPORT_STANDARD.md))
- Allowed Use · Sensitivity · PII · Retention · AI access · External use

**اتجاه صناعي:** قرارات وصول **قابلة للتدقيق**، **policy-aware**، ومبدأ **deny-by-default** للبيانات الحساسة — إطار تعبيري: [NIST Zero Trust Architecture](https://www.nist.gov/publications/zero-trust-architecture).
