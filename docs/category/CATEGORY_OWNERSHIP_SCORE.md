# Dealix Category Ownership Score

قِس مدى **امتلاكك للفئة** قبل أن تبالغ في «الريادة الفكرية».

| الإشارة | المعنى |
|---------|--------|
| العملاء يستخدمون **مفرداتك** | language adoption |
| الشركاء يكررون **منهجيتك** | ecosystem adoption |
| يُطلب **Proof Pack** | منتج ثقة |
| يُطلب **Capability Score** | إطار استراتيجي |
| طلبات وارد على **Revenue Intelligence** | وضوح الإسفين |
| طلبات **benchmark** | سلطة سوقية |
| اهتمام **Academy** | طلب تعليم |
| Enterprise يسأل عن **Governance Runtime** | نضج الثقة |

## قرار تشغيلي

- إذا السوق **لا يستخدم لغتك** → ركّز على **education + proof**.  
- إذا السوق **يستخدم لغتك** → عزّز **standards + academy**.
