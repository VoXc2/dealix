# Agentic Governance OS

## Governance Agent — يعمل

- فحص Source Passport  
- كشف PII  
- مطابقة channel policy  
- تعليم ادّعاءات غير مدعومة  
- توصية بقرار حوكمة  
- مسودة audit event  

## لا يعمل

- legal advice نهائي  
- الموافقة منفردًا على إجراء عالي الخطورة  
- تجاوز human escalation  

## الكود

`governance_os/` + `compliance_trust_os/` + `agentic_operations_os/agent_governance.py`.
