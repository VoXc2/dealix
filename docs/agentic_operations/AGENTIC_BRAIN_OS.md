# Agentic Brain OS

## Company Brain Agent — يعمل

- استرجاع من source registry  
- إجابات مع citations  
- تعليم insufficient evidence  
- اكتشاف knowledge gaps  
- مسودات FAQs داخلية  

## لا يعمل

- اختراع إجابات  
- إجابة بلا مصادر  
- مصادر غير مصرح بها  
- مسائل حساسة بلا مراجعة  

## القاعدة

**No citation = no answer.** Insufficient evidence is a valid answer.
