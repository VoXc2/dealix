# Agent-to-Proof Loop

كل وكيل **يساهم في Proof** لا في مخرجات فقط.

## أسئلة Proof Pack

- أي وكلاء ساهموا؟  
- ماذا أنتجوا؟  
- ماذا راجع البشر؟  
- ماذا حُظر؟  
- ما القيمة الملاحظة؟  

## أمثلة مساهمات (مرجعية)

- **RevenueAgent:** scoring، duplicates، أولويات، إجراءات غير آمنة محجوبة.  
- **BrainAgent:** إجابات مع استشهاد، فجوات معرفة، insufficient evidence.  
- **GovernanceAgent:** قواعد مطابقة، مخرجات محجوبة، موافقات مطلوبة، ادّعاءات مزالة.
