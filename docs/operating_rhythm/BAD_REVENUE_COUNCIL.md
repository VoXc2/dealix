# Bad Revenue Council — رفض الإيراد الذي يدمّر الثقة

**مرة أسبوعيًا:** راجع الفرص التي **يجب رفضها أو إعادة تأطيرها**.

## إشارات إيراد سيئ

- العميل يطلب **scraping**  
- العميل يطلب **cold WhatsApp automation**  
- العميل يريد **مبيعات مضمونة**  
- مصدر البيانات **غير واضح**  
- النطاق **مفتوح بدون سقف**  
- **هامش ضعيف**  
- **لا مسار proof**  
- **لا مسار retainer**  

## قرارات المجلس

Reject، Reframe as Diagnostic، Reprice، Add strict scope، Refer out.

## الجملة

**Dealix لا ترفض المال؛ Dealix ترفض الإيراد الذي يدمّر الثقة والتركيز.**

## الكود

- `auto_client_acquisition/operating_rhythm_os/bad_revenue_council.py` — إشارات المجلس + توصية.
- `auto_client_acquisition/operating_finance_os/bad_revenue_filter.py` — فلتر إيراد سيئ أوسع (هامش، نطاق، إلخ).
