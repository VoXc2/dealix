# Dealix Execution Council — أدوار، لا أشخاص بالضرورة

حتى لو كنت وحدك اليوم، **تصرّف كأن لديك مجلس تنفيذ داخلي**. المجلس يتكوّن من **أدوار**؛ لاحقًا يصبح التفويض طبيعيًا لأن كل دور له سؤال مختلف.

## المقاعد

| الدور | السؤال الأساسي |
|--------|----------------|
| **CEO / Strategist** | ما أهم قرار هذا الأسبوع؟ |
| **Revenue Owner** | ما الذي نبيعه الآن؟ |
| **Delivery Owner** | هل نستطيع تسليمه بتكرار وجودة؟ |
| **Product Owner** | ما الذي تكرّر بما يكفي لنُنتجه؟ |
| **Governance Owner** | ما المخاطر التي يجب منعها؟ |
| **Proof Owner** | ما الدليل على القيمة؟ |
| **Client Success Owner** | هل اعتمد العميل النظام؟ |
| **Finance / Capital Owner** | هل هذا إيراد جيد أم سيئ؟ |

## قاعدة التشغيل

في البداية قد يمسك المؤسس معظم المقاعد، لكن **وجود المقاعد** يمنع ذوبان كل شيء في «قائمة مهام» واحدة بلا ملكية.
