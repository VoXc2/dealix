# Dealix CEO Operating Rhythm — إيقاع التنفيذ

الاستراتيجية لا تنفّذ نفسها. هذه الطبقة تحوّل Dealix من **معمارية ووثائق** إلى **شركة تُدار بإيقاع**: يوميًا قرارات، أسبوعيًا مجلس تنفيذ ومراجعات، شهريًا مذكرة مجلس، ربعيًا رهانات استراتيجية.

## السلسلة الذهبية للتشغيل

**Strategy → Weekly execution → Proof → Decision → Capital allocation → Next bet**

بدون هذا الإيقاع، تتحوّل الطبقات السابقة إلى مستندات جميلة فقط.

## لماذا الإيقاع؟

أغلب مشاريع الذكاء الاصطناعي لا تفشل لضعف الفكرة، بل لضعف **تصميم العمل والأدوار والتكامل** مع القياس والملكية. Dealix تعالج ذلك بـ **Operating Rhythm** صريح.

## المستويات

| المستوى | الغرض |
|---------|--------|
| يومي | تركيز CEO — قرارات وليس صندوق وارد فقط |
| أسبوعي | حقيقة التنفيذ — قرارات، التزامات، مخاطر، proof |
| شهري | حقيقة الأعمال — إيراد، هامش، أصول، أرشيف قرار |
| ربع سنوي | حقيقة الاستراتيجية — رهان، توسيع، قتل، سردية سوق |
