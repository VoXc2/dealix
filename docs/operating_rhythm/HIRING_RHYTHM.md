# Hiring Rhythm — التوظيف عند تكرار الاختناق

**لا توظف من الضغط فقط.** وظّف عندما يتكرر **اختناق** يثبت أن الدور سيدفع الإيقاع وليس الازدحام فقط.

## محفزات (مرجعية)

| الدور | متى تفكر جديًا |
|--------|----------------|
| **Delivery Analyst** | 3+ مشاريع متكررة بقائمة تحقق مستقرة |
| **Product Engineer** | خطوات يدوية 5+ وطابور تمنتج واضح |
| **Customer Success / AI Ops** | 2+ retainer بإيقاع شهري |
| **Governance Reviewer** | طلب مؤسسي + مراجعات PII/حوكمة متكررة |
| **Partner Manager** | 3+ شركاء نشطون يولّدون leads مؤهلة |

## القواعد

- **No hire before playbook.**  
- **No delegation without QA rubric.**  
- **No growth hire before offer–message–proof fit.**
