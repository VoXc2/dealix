# Quarterly Strategic Review — رهان الربع

كل ربع، **أبطئ التنفيذ اليومي** واسأل أسئلة أكبر.

## أسئلة استراتيجية

- هل **Revenue Intelligence** ما زال أفضل wedge؟
- هل **Governance Runtime** أصبح منتجًا مستقلًا؟
- هل **Client Workspace** عليه pull كافٍ؟
- هل هناك **Business Unit** جاهزة؟
- هل هناك **خدمة يجب قتلها**؟
- هل السوق بدأ يستخدم **لغتنا**؟
- هل يجب **رفع الأسعار**؟
- هل نحتاج **شريكًا** أو **موظفًا** أو **venture candidate**؟

## مخرجات إلزامية للربع

1. **رهان استراتيجي** واحد  
2. **خدمة للتوسيع**  
3. **خدمة للقتل**  
4. **وحدة منتج** للبناء  
5. **مخاطرة للتخفيض**  
6. **سردية سوق** للدفع
