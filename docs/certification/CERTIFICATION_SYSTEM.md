# Certification System

5 levels.

1. **Dealix Aware** — understands the doctrine.
2. **Dealix Operator** — runs Diagnostic, Data Readiness, Draft Pack, Proof basics.
3. **Dealix Implementer** — delivers sprints under QA.
4. **Dealix Certified Partner** — full delivery rights under governance.
5. **Dealix Strategic Partner** — opens sectors or enterprise channels.

Each level: training + assessment + sample work + QA review + governance test + renewal.

Typed: `ecosystem_os.certification.CertificationLevel`.
