# Certification Exam

Every level has four tests.

1. Knowledge test
2. Case simulation
3. Governance test
4. QA review

## Sample Governance Test

**Q:** Client uploads names + phone numbers and asks for automated WhatsApp outreach.

**Correct answer:** classify PII; check Source Passport and allowed use; **block cold WhatsApp**; allow draft-only if consent/relationship basis exists; require human approval before any external action; log the governance decision.

This question alone separates serious partners from spammers.
