# Dealix — Micro-Offers Entry Ladder
<!-- PHASE 3 | Owner: Founder | Date: 2026-05-17 -->
<!-- Arabic primary — العربية أولاً -->

هذا الملف يصف **سلم الدخول كعروض منخفضة المقاومة**. كل درجة تزيل مقاومة
محدّدة وتفتح الدرجة التالية بدليل لا بإقناع. التسعير هنا مأخوذ حرفياً من
السجل الرسمي — لا أرقام أخرى. المرجع: [../OFFER_LADDER_AND_PRICING.md](../OFFER_LADDER_AND_PRICING.md)
و[../COMMERCIAL_WIRING_MAP.md](../COMMERCIAL_WIRING_MAP.md).

This file frames the **entry ladder as low-resistance micro-offers**. Each
rung removes one specific resistance and unlocks the next by evidence, not
persuasion. Pricing is taken verbatim from the canonical registry.

## مبدأ السلم — Ladder principle

- لا قفزة. المشتري يصعد درجة واحدة بعد دليل موثق من الدرجة السابقة.
- كل درجة "نعم" صغيرة. مجموع النَعَمات الصغيرة يبني الثقة.
- لا ترقية قبل نتيجة موثقة. لا تسعير مخفي (no_hidden_pricing).

## الدرجة 0 — التشخيص المجاني / Free Mini Diagnostic

| العنصر | التفاصيل |
|--------|----------|
| **السعر** | 0 ريال — دفعة واحدة |
| **لمن** | أي مؤسس B2B سعودي يستكشف ديالكس لأول مرة. |
| **المقاومة التي يزيلها** | "لا أعرف إن كانت لديّ مشكلة أصلاً." يحوّل الفضول إلى تشخيص ملموس بلا مخاطرة مالية. |
| **المخرج** | تقرير تشخيصي صفحة واحدة + 3 أولويات + توصية الخطوة التالية. |
| **ما يفتح الدرجة التالية** | المشتري يرى فجوة موثقة في "ما بعد الـ lead" ويطلب دليلاً حقيقياً. |

## الدرجة 1 — سبرنت إثبات الإيرادات / Revenue Proof Sprint

| العنصر | التفاصيل |
|--------|----------|
| **السعر** | 499 ريال — دفعة واحدة |
| **لمن** | مؤسس أو مدير مبيعات يريد دليلاً حقيقياً خلال أسبوع قبل أي التزام أكبر. |
| **المقاومة التي يزيلها** | "لا أعرف إن كان الذكاء الاصطناعي ينفعني." 499 ريال سعر منخفض بما يكفي لقرار سريع، ومرتفع بما يكفي ليكون التزاماً جاداً. |
| **المخرج** | تشخيص مفصّل، 5 مسودات رسائل جاهزة للموافقة، Proof Pack يوم 7، تقرير تنفيذي، خطة 30 يوم. |
| **ما يفتح الدرجة التالية** | Proof Pack موثق + رضا العميل. الدليل نفسه هو ما يبرّر الصعود. |

## الدرجة 2 — حزمة من البيانات إلى الإيراد / Data-to-Revenue Pack

| العنصر | التفاصيل |
|--------|----------|
| **السعر** | 1,500 ريال — مشروع واحد |
| **لمن** | شركة B2B لديها بيانات عملاء/مبيعات غير مستثمرة وتريد تحويلها إلى فرص. |
| **المقاومة التي يزيلها** | "لدينا بيانات لكن لا نستخرج منها قرارات." يثبت أن البيانات الموجودة أصل، لا عبء. |
| **المخرج** | تحليل pipeline كامل، خريطة فرص مرتّبة، 10 مسودات استهداف مخصّصة، أساس ROI، playbook مبيعات. |
| **ما يفتح الدرجة التالية** | فرصة واحدة مُعرّفة وموثقة + playbook موافق عليه يثبتان الحاجة لتشغيل مستمر. |

## الدرجة 3 — عمليات النمو الشهرية / Growth Ops Monthly

| العنصر | التفاصيل |
|--------|----------|
| **السعر** | 2,999 ريال/شهر |
| **لمن** | شركة أتمت pilot ناجحاً وتريد تشغيل الأسئلة الخمسة بشكل مستمر. |
| **المقاومة التي يزيلها** | "نجحت التجربة، لكن لا أريد أن أعتمد على جهد لمرة واحدة." يحوّل الدليل لمرة واحدة إلى إيقاع شهري. |
| **المخرج** | تقرير أسبوعي تنفيذي، مسودات تواصل شهرية موافق عليها، Proof Pack شهري، جلسة استراتيجية شهرية. |
| **ما يفتح الدرجة التالية** | استمرار MRR + رضا شهري + تراكم 3 pilots يمهّد لمركز القيادة. |
| **ملحق اختياري** | إضافة دعم العمليات / Support OS Add-on — 1,500 ريال/شهر. |

## الدرجة 4 — غرفة قيادة الإدارة / Executive Command Center

| العنصر | التفاصيل |
|--------|----------|
| **السعر** | 7,500 ريال/شهر |
| **لمن** | مؤسس أو إدارة تنفيذية تريد رادار قرار يومي لا لوحات فارغة. |
| **المقاومة التي يزيلها** | "أحتاج قراراً جاهزاً، لا بيانات أفسّرها بنفسي." |
| **متطلب الفتح** | ≥ 3 pilots مكتملة + ≥ 1 دراسة حالة موثقة. |
| **المخرج** | تقرير يومي تنفيذي، رادار سوق، Proof Pack كامل، استراتيجية ربعية، وصول أولوية. |
| **ما يفتح الدرجة التالية** | نضج تشغيلي + رغبة في توزيع القيمة عبر قناة. |

## الدرجة 5 — منصة شركاء الوكالات / Agency Partner OS

| العنصر | التفاصيل |
|--------|----------|
| **السعر** | مخصص + حصة إيراد 15–30% |
| **لمن** | وكالة تسويق/استشارات تريد تقديم ديالكس لعملائها تحت علامتها. |
| **المقاومة التي يزيلها** | "أريد أصلاً يميّزني تنافسياً دون أن أبنيه بنفسي." |
| **متطلب الفتح** | إتمام ≥ 3 proof packs + اتفاقية شراكة موقّعة. |
| **المخرج** | white-label محدود، تدريب الشريك، Proof Packs مشتركة العلامة، لوحة شريك، دعم أولوية. |
| **المسار** | راجع [PARTNER_ECONOMICS.md](PARTNER_ECONOMICS.md) و[AGENCY_WEDGE_PLAYBOOK.md](AGENCY_WEDGE_PLAYBOOK.md). |

## ممنوعات الصياغة — Banned phrasing

- لا تسعير مخفي — كل رقم معلن ومربوط بالسجل الرسمي.
