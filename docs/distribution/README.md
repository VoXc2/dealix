# Dealix — Distribution Operating Layer Index
<!-- PHASE 3 | Owner: Founder | Date: 2026-05-17 -->
<!-- Arabic primary — العربية أولاً -->

هذه الطبقة هي **الطبقة التشغيلية الاستراتيجية فوق sales-kit**. الـ sales-kit يحتوي
الأصول التنفيذية (رسائل، سكربتات، قوالب). هذه الطبقة تفسّر *لماذا* و*بأي ترتيب*
و*لمن*. لا تكرّر الأصول — تربطها.

This layer is the **strategic operating layer above the sales-kit**. The sales-kit
holds execution assets; this layer explains the *why*, the *sequence*, and the
*audience*.

## التموضع الجوهري — Core positioning

> **Dealix = Post-Lead Revenue & Governed AI Operations OS**
> ديالكس = نظام تشغيل الإيرادات بعد وصول العميل المحتمل، والذكاء الاصطناعي
> المحكوم.

ديالكس ليس "أداة مبيعات بالذكاء الاصطناعي". الهدر الأكبر يبدأ **بعد** وصول
الـ lead: من ردّ؟ من تابع؟ من وافق؟ ما الرسالة التالية؟ أين الدليل؟

## آلة الذهاب-إلى-السوق — The GTM machine (layered)

```
[1]  Category Creation     → نخلق الفئة: تشغيل الإيرادات بعد الـ lead
[2]  Pain Education        → نُظهر للمشتري الهدر الذي لا يراه
[3]  Proof Asset           → نُنتج دليلاً حقيقياً قابلاً للفحص
[4]  Micro-offer           → باب دخول منخفض المقاومة (مجاني → 499)
[5]  Manual Sales          → بيع يقوده المؤسس، غرفة بيع واحدة
[6]  Partner Distribution  → الوكالات توزّع ديالكس لعملائها
[7]  Affiliate Layer       → إحالات محكومة، 5,000 ريال لكل صفقة مغلقة
[8]  Fast Delivery         → تسليم Sprint خلال 7 أيام، مُخرج جاهز
[9]  Customer Success      → تثبيت القيمة ومنع الـ churn
[10] Case Study Factory    → كل تسليم ناجح يصبح حالة آمنة قابلة لإعادة الاستخدام
[11] Benchmark Engine      → أنماط مجمّعة مجهولة الهوية تغذّي البيع
[12] Retainer/Platform Path→ المسار إلى Growth Ops و Command Center والشراكة
```

سطر واحد لكل طبقة — One line per layer:

| الطبقة | الغرض |
|--------|-------|
| Category Creation | نسمّي المشكلة قبل أن نبيع الحل. |
| Pain Education | المشتري لا يشتري قبل أن يرى الهدر. |
| Proof Asset | دليل حقيقي يسبق أي ادعاء. |
| Micro-offer | أصغر "نعم" ممكنة بأقل مخاطرة. |
| Manual Sales | المؤسس يبيع يدوياً حتى تظهر الأنماط. |
| Partner Distribution | الوكالة قناة توزيع، لا منافس. |
| Affiliate Layer | إحالة محكومة بموافقة، لا بريد بارد. |
| Fast Delivery | السرعة هي الدليل الأول على الكفاءة. |
| Customer Success | القيمة المُثبتة تمنع الانسحاب. |
| Case Study Factory | كل نجاح يصبح أصلاً بيعياً. |
| Benchmark Engine | الأرقام المجمّعة تبيع بلا كشف هوية. |
| Retainer/Platform Path | الترقية تُفتح بالدليل لا بالإقناع. |
