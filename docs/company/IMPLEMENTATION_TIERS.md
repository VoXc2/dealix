# Implementation Tiers

## Tier 1 — Diagnostic

Assessment, readiness, roadmap.

## Tier 2 — Sprint

Fixed-scope output and proof.

## Tier 3 — Pilot

Operate one workflow for 30 days.

## Tier 4 — Managed OS

Monthly operating cadence.

## Tier 5 — Enterprise OS

Multi-workflow, multi-team, governed platform.

## Per Tier

| Tier       | Goal                    | Outputs               | Pricing band |
| ---------- | ----------------------- | --------------------- | ------------ |
| Diagnostic       | clarity                 | roadmap               | low          |
| Sprint           | first proof             | output + proof        | medium       |
| Pilot            | operational validation  | workflow + metrics    | medium-high  |
| Managed OS       | continuity              | monthly value         | recurring    |
| Enterprise OS    | scale                   | platform + governance | high         |
