# Dealix AI Capability Factory

Dealix turns business problems into AI-enabled operating capabilities.

## Factory Loop (Capability Factory)

**Raw in → capability out → asset inside the holding:**

```text
Client Problem → Capability Built → Proof Created → Asset Captured → Module Improved
→ Playbook Updated → Retainer Offered → Venture Signal Detected
```

**الخطوات التشغيلية التفصيلية (9):** Diagnose → Scope → Build → Govern → Prove → Operate → Compound → Productize → Spin Out — انظر [`../group/DEALIX_OPERATING_GOVERNANCE_LAYER.md`](../group/DEALIX_OPERATING_GOVERNANCE_LAYER.md).

## Factory Flow (تسليم خدمة)

1. Intake  
2. Capability Assessment  
3. Data Readiness  
4. Workflow Design  
5. Governance Design  
6. AI Build  
7. QA + Evals  
8. Delivery  
9. Proof Pack  
10. Capability Roadmap  
11. Retainer / Expansion  

## Rule

Every service must improve at least one client capability:

- Revenue  
- Customer  
- Operations  
- Knowledge  
- Data  
- Governance  
- Reporting  

---

## What this means

Any service that is not defined as a capability should not be sold.

Examples:

- **Lead Intelligence** → builds **Revenue Capability**  
- **Company Brain** → builds **Knowledge Capability**  
- **AI Quick Win** → builds **Operations Capability**  
- **AI Governance** → builds **Governance Capability**  
- **Executive Reporting** → builds **Reporting Capability**
