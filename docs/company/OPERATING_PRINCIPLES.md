# Operating Principles

1. Outcome before tools.
2. No data without source.
3. No AI output without QA.
4. No external action without approval.
5. No source = no answer.
6. Every project ends with proof.
7. Every repeated manual step becomes a feature.
8. Every service has KPI.
9. Arabic quality is a core product feature.
10. Compliance is built in, not added later.
