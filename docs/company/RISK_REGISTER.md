# Risk Register (Company Executive View)

Operational risks Dealix manages explicitly. Detailed governance catalog: [`../governance/RISK_REGISTER.md`](../governance/RISK_REGISTER.md).

| Risk | Impact | Control |
|------|--------|---------|
| Overbuilding | burns time | build after 3 repetitions ([`PRODUCTIZATION_LEDGER.md`](../product/PRODUCTIZATION_LEDGER.md)) |
| Weak sales | no revenue | one ICP + proof-based sales |
| Scope creep | low margin | strict SOW + change request |
| Poor data | bad output | data readiness gate |
| Compliance issue | trust damage | governance OS + policy_check |
| Low-quality Arabic | weak trust | Arabic QA |
| No proof | no upsell | proof pack required |
| Founder bottleneck | no scale | delivery OS + hiring triggers |
| AI cost | margin loss | FinOps + gateway discipline |

Review weekly in operating review; escalate material items to governance ledger if needed.
