# Service Status Rules

| Status | Meaning |
|--------|---------|
| **Idea** | فكرة فقط — لا folder / لا offer → **ممنوع البيع** |
| **Designed** | offer + scope — **لا demo كامل أو QA** → ممنوع بيع رسمي |
| **Beta** | readiness **70–84** — pilot فقط بتسمية صريحة |
| **Sellable** | readiness **85+** + demo + QA + proof template |
| **Scalable** | **90+** + شخص آخر يمكنه التسليم + المنتج يغطي معظم العمل + QA متوسط عالي |

انظر [`SERVICE_READINESS_MATRIX.md`](SERVICE_READINESS_MATRIX.md) و `scripts/print_service_readiness_matrix.py`.
