# Productization Gate — بوابة تحويل التكرار إلى منتج

**لا تُبنَ ميزة من «إلهام فقط»** — من **التكرار** و**الإيراد** و**الاختبار**.

## شروط Gate H

| شرط | معنى |
|-----|------|
| تكرار | نفس الخطوة اليدوية **3+** مرات |
| وقت | توفير ساعات ملموس |
| خطر | تقليل خطر أو تباين جودة |
| إيراد | رابط صريح بعرض أو خدمة تُباع |
| اختبار | acceptance واضح |
| إعادة استخدام | أكثر من مشروع/عميل |

## دفتر المنتجات

سجّل كل خطوة يدوية: الوقت، التكرار، الخطر، رابط الإيراد، **القرار** — [`../product/PRODUCTIZATION_LEDGER.md`](../product/PRODUCTIZATION_LEDGER.md)
