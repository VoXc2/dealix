# Dealix Operating Flywheel

Every client must improve the **system**, not just deliver a one-off.

## Loop

See **closed-loop execution** in [`CLOSED_LOOP_EXECUTION.md`](CLOSED_LOOP_EXECUTION.md).

```text
Better services
→ Better delivery
→ Better proof packs
→ Better case studies / assets
→ Better sales
→ More clients
→ More learning
→ Better product
→ Better services
```

If a project does not improve at least one of:

- service quality (templates, checklists)
- delivery speed (automation candidate → feature)
- product capability
- **playbooks** (`docs/playbooks/`)
- **proof / sales** assets (`docs/assets/`)
- sales conversion (scripts, objections)

…it is **operationally incomplete** — see [`COMPOUNDING_SYSTEM.md`](COMPOUNDING_SYSTEM.md), [`POST_PROJECT_REVIEW`](../templates/LEARNING_LOOP.md) closure.

## Decision layer

All investments (sell, deliver, build, price, scale) flow through [`DECISION_OPERATING_SYSTEM.md`](DECISION_OPERATING_SYSTEM.md).
