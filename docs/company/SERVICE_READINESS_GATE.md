# Operating Gates — بوابات A إلى I

**القاعدة:** لا شيء ينتقل لمرحلة لاحقة **بدون gate** — يمنع الفوضى مع التوسع.

**المجلس والأسئلة:** [`CEO_OPERATING_SYSTEM.md`](CEO_OPERATING_SYSTEM.md) · **Capability Loop:** [`../group/DEALIX_OPERATING_GOVERNANCE_LAYER.md`](../group/DEALIX_OPERATING_GOVERNANCE_LAYER.md)

---

## Gate A — Diagnostic

**قبل بدء مشروع:** buyer · problem · data · workflow · risk · success metric · budget · decision maker

**قرار:** Accept · Diagnostic only · Reject

---

## Gate B — Scope

**قبل Build:** included · excluded · timeline · client inputs · approval owner · success metric

**قرار:** Build · Reduce scope · Rescope · Reject

---

## Gate C — Data

**قبل استخدام بيانات:** source · allowed_use · PII status · quality score · retention rule

**قرار:** Use · Use with review · Internal only · Redact · Block  

مرجع: [`../architecture/SOURCE_PASSPORT.md`](../architecture/SOURCE_PASSPORT.md) · [`../data/DATA_READINESS_STANDARD.md`](../data/DATA_READINESS_STANDARD.md)

---

## Gate D — Governance

**قبل خروج output:** risk level · policy decision · audit event · approval status

**قرار:** Allow · Draft only · Require approval · Redact · Block  

مرجع: [`../governance/GOVERNANCE_RUNTIME.md`](../governance/GOVERNANCE_RUNTIME.md)

---

## Gate E — QA

**قبل تسليم للعميل:** business score **≥ 85** · language **≥ 85** · data **≥ 80** · governance **= 100%** (حيث ينطبق المسار)

مرجع: [`../quality/QUALITY_STANDARD_V1.md`](../quality/QUALITY_STANDARD_V1.md)

---

## Gate F — Proof

**قبل إغلاق المشروع:** proof pack · value metric · next recommendation · capital asset

مرجع: [`../templates/PROOF_PACK_TEMPLATE.md`](../templates/PROOF_PACK_TEMPLATE.md)

## Gate G — Retainer

**قبل عرض retainer:** proof واضح · client health **≥ 70** · قيمة شهرية معرّفة · workflow متكرر

مرجع: [`../growth/PROOF_TO_RETAINER_SYSTEM.md`](../growth/PROOF_TO_RETAINER_SYSTEM.md)
