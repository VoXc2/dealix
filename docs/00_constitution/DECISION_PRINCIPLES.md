# Decision Principles — مبادئ القرار العليا

## القوانين العليا (تمنع الانزلاق لوكالة فوضوية)

1. No AI without **workflow**.  
2. No workflow without **owner**.  
3. No output without **QA**.  
4. No external action without **approval**.  
5. **No source, no trust.**  
6. **No proof, no claim.**  
7. No repeated demand, no **product feature**.  
8. No project without **capital creation**.  
9. **No governance, no scale.**

## بوابة القرار السريعة

قبل التنفيذ، نعم/لا:

1. **Does it sell?**  
2. **Does it deliver?**  
3. **Does it prove?**  
4. **Does it govern?**  
5. **Does it compound?**  
6. **Does it scale?**  

**قاعدة:** إن لم تحقق **3 من 6** على الأقل — **لا تنفّذ** (أو أعد الصياغة حتى تلتقي البوابة).
