# Partner Network — الشبكة المؤسسية للشركاء

**الهدف:** توزيع منظومي **بدون تفكيك الحوكمة أو الثقة**.

## الأنواع

Referral · Implementation · Certified · Strategic · **White-label** (أعلى مخاطر — شهادة، عينات، تدقيق، حق مراجعة عميل).

## شروط الشريك

- يفهم عمليات Dealix ويحترم non-negotiables  
- لا أنظمة spam / قنوات ممنوعة  
- يقبل **QA** و**Proof Pack**  
- يلتزم **Dealix Method** — [`../standards/DEALIX_METHOD.md`](../standards/DEALIX_METHOD.md)

## Partner Control

شهادة · مراجعات جودة · قواعد حوكمة · proof لكل تسليم مهم · **feedback عميل**

**السبب:** شريك سيء يدمّر الثقة أسرع من عميل صعب.
