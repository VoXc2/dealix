# Benchmark Engine

Aggregate **anonymized** operating metrics across clients → annual narrative (“State of AI Operations in Saudi SMEs”).

## Metrics (roll up monthly)

- Average **data quality** score (when measured)
- Common **missing fields**
- Common **AI use cases** accepted vs rejected
- Common **governance** risks
- Average **QA** scores
- Average **time saved** (estimated, honest)
- **Sprint → retainer** rate
