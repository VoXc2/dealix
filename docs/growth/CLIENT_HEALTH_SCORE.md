# Client Health Score

Predict **renewal**, **expansion**, and **churn risk** from behavior—not only NPS.

## Score (100)

| المعيار | النقاط |
|---------|-----:|
| يستخدم المخرجات فعليًا | 20 |
| يرى قيمة واضحة | 20 |
| لديه next workflow واضح | 15 |
| يتفاعل بسرعة (SLA للقرارات) | 15 |
| فرصة retainer / توسع real | 15 |
| لا توجد مخاطر علاقة كبرى | 15 |

## Bands

| Score | Meaning |
|-------|---------|
| **80+** | Expand |
| **60–79** | Nurture |
| **40–59** | Risk |
| **أقل من 40** | Likely churn — intervene or exit cleanly |

## Use

- Monthly on retainer clients
- End of sprint for project clients  
Log actions in [`CLIENT_REVIEW_MEETING.md`](../delivery/CLIENT_REVIEW_MEETING.md) follow-ups.
