# Case Study Factory

Turn **delivery evidence** into anonymized authority assets.

## Inputs

- Proof pack (client + internal)
- QA score
- Client feedback (consent for quotes)
- Before / after metrics (honest ranges)
- Sector insight (no identifiable firm)

## Structure

1. Client **type** (segment, size band)
2. **Problem**
3. What Dealix **did** (process, not tool fluff)
4. **Outputs** (artifacts)
5. **Proof** (numbers or qualitative gains)
6. **Next step** (retainer / pilot / self-serve)

## Rule

**No client identity** without written permission. Prefer [`../assets/case_studies/anonymized_case_study_template.md`](../assets/case_studies/anonymized_case_study_template.md).
