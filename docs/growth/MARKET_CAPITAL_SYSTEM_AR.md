# نظام رأس مال السوق — دليل التشغيل · CEO/CTO/CSO

**الطبقة:** دليل التشغيل · CEO/CTO/CSO
**المالك:** CSO + قائد النموّ
**الحالة:** مسودة
**آخر مراجعة:** 2026-05-13
**النسخة الإنجليزية:** [MARKET_CAPITAL_SYSTEM.md](./MARKET_CAPITAL_SYSTEM.md)

## السياق

رأس مال السوق هو الذاكرة العامة لـ Dealix. بدونه تبدأ كل مكالمة بيع
باردة وتبدأ كل حركة فئة بلا مرتكز. يحوّل نظام رأس مال السوق كل حزمة
إثبات إلى أصل عامّ آمن المشاركة. يُكمّل `docs/BRAND_PRESS_KIT.md`
و `docs/COMPETITIVE_POSITIONING.md` و
`docs/growth/AGENCY_RESELLER_PLAYBOOK.md`. وهو نظير
`docs/company/KNOWLEDGE_CAPITAL_SYSTEM.md` و
`docs/company/TRUST_CAPITAL_SYSTEM.md`.

## أنواع أصول السوق

| الأصل | التعريف |
|---|---|
| منشور محتوى | LinkedIn / X / Substack مربوط بإثبات أو رؤية |
| playbook | 5-10 صفحات قطاعية عامة |
| تقرير benchmark | تقرير مقاييس عبر العملاء مُجهَّل |
| علاقة شريك | شريك نشط مُسمّى بمسار عرض مشترك |
| مصدر إحالة | عميل أو شريك يقدّم صفقات |
| webinar | جلسة مسجَّلة بجمهور تنفيذي |
| قالب عام | أصل مجاني للتنزيل (SOP، prompt، نموذج استلام) |
| صفحة مرجعية | صفحة طويلة تملك مصطلحاً/موضوعاً |

## قاعدة 1-إلى-1

> **كل حزمة إثبات تنتج أصلاً واحداً آمناً للسوق.**

الأصل هو واحد من:

- منشور رؤية مجهَّل.
- ملاحظة درس قطاعي.
- نقطة بيانات benchmark.
- قصة من نوع دراسة حالة (موافق عليها + مجهَّلة + مُراجعة).

حزمة إثبات لا تُنتج أصل سوق هي ناقصة. لا يستطيع CSM إغلاق الارتباط
حتى يُنتج الأصل السوقي أو يُعفى صراحة من CEO بسبب موثَّق.

## لماذا الإجهال أولاً

الأصول المُجهَّلة قابلة للنشر فوراً. الأصول المُسمّاة تحتاج موافقة
صريحة وموقَّعة ومؤرَّخة (انظر
`docs/company/TRUST_CAPITAL_SYSTEM.md`). الافتراضي مجهَّل، والترقية
للمُسمّى تحدث لاحقاً.

## سطوح التوزيع

- حسابات LinkedIn الشخصية (CEO + القيادة).
- صفحة Dealix.
- نشرة بريدية إلى قائمة سعودية/MENA مؤهَّلة.
- قنوات الشركاء عبر `docs/growth/AGENCY_RESELLER_PLAYBOOK.md`.
- صفحات مرجعية على نطاق dealix.

## الإيقاع

- **عند إغلاق الارتباط** — أصل سوق واحد.
- **أسبوعي** — قائد النموّ ينقّح ويُطلق المحتوى.
- **شهري** — تحديث تقرير الـbenchmark إن كان الـcohort ≥5.
- **ربعي** — مراجعة الصفحة المرجعية.

## قواعد مضادّة للسبام

- لا ذكر أسماء مزوّدين بلا موافقة.
- لا أرقام كاذبة، الـbenchmark دائماً يذكر حجم العيّنة.
- لا لغة استدراج تفاعل ("علّق "موافق" إذا...").
- لا محتوى مترجم من الإنجليزية يكسر السجلّ العربي (وفق
  `docs/quality/ARABIC_BUSINESS_QUALITY.md`).

## الواجهات

| المدخلات | المخرجات | المالك | الإيقاع |
|---|---|---|---|
| حزم الإثبات | أصول سوق آمنة | قائد النموّ + CSM | لكل إغلاق |
| سجلّ المعرفة | مرشّحو مواضيع | CSO | شهري |
| الـcohort | تقرير benchmark | قائد النموّ | ربعي |

## المقاييس

- **أصول سوق لكل حزمة إثبات** — الهدف ≥1.
- **عدد الصفحات المرجعية حسب الموضوع** — يُتبع ربعياً.
- **الفرص الواردة من إحالة** — % من القمع (الهدف ≥20%).
- **عدد الـwebinars/ربع** — الهدف ≥1.
- **امتثال المحتوى العام** — % المُراجَع قبل النشر (الهدف 100%).
