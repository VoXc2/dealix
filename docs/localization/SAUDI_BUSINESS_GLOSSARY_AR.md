# مسرد الأعمال السعودي (Saudi Business Glossary)

| المصطلح | الاستخدام في Dealix |
|---------|---------------------|
| Revenue Intelligence Sprint | تشخيص 5 أيام لتسرّب الإيراد + خطة 30 يوم |
| Proof Pack | حزمة إثبات مُنقّحة بالأدلة |
| pipeline | مسار الصفقات |
| follow-up | المتابعة |
| conversion | التحويل (استفسار → عميل) |
| Retainer | اشتراك تشغيلي شهري |
| lead / prospect | عميل محتمل |
| churn | فقدان العملاء |
| MRR | الإيراد الشهري المتكرر |
| CAC | تكلفة اكتساب العميل |
| LTV | القيمة الدائمة للعميل |
| suppression | قائمة الكبح (لا تواصل) |
| do-not-contact | عدم التواصل |
| PDPL | نظام حماية البيانات الشخصية (SDAIA) |

## قاعدة
استخدم المصطلح بثبات؛ اشرح عند أول ظهور للعميل غير التقني.
