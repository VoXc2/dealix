# نظام التوطين السعودي (Saudi Localization OS)

Dealix **عربي-أولاً** موجّه للسوق السعودي B2B.

## الركائز
| الركيزة | المرجع |
|---------|--------|
| واجهة عربية أولاً (RTL) | `src/` (مراجعة UX) |
| نبرة عمل سعودية B2B | `SAUDI_B2B_TONE_GUIDE_AR.md` |
| صوت العلامة العربي | `ARABIC_BRAND_VOICE_AR.md` |
| اتساق ثنائي اللغة | `BILINGUAL_STYLE_GUIDE_AR.md` |
| مصطلحات الأعمال | `SAUDI_BUSINESS_GLOSSARY_AR.md` |
| واتساب سعودي | `WHATSAPP_SAUDI_UX_AR.md` |

## ثوابت
- التسعير بالريال السعودي (ر.س / SAR).
- المنطقة الزمنية: الرياض (UTC+3)؛ أوقات العمل السعودية.
- بلا مبالغة أو ادعاءات مضمونة (يتوافق مع `core/safety/claims.py`).
- مستويات رسمية/ودّية حسب السياق، مع احترام مهني.
