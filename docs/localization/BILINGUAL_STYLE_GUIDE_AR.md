# دليل الأسلوب ثنائي اللغة (Bilingual Style Guide)

عربي-أولاً مع مصطلحات إنجليزية تقنية شائعة عند الحاجة.

## قواعد
1. النص الأساسي عربي (RTL)؛ المصطلحات الإنجليزية تُبقى لاتينية ضمن النص العربي
   (مثل: pipeline، Sprint، CRM، Proof Pack).
2. اتساق المصطلح عبر كل القنوات (انظر `SAUDI_BUSINESS_GLOSSARY_AR.md`).
3. الأرقام والعملات: ر.س (SAR)، تنسيق عربي/لاتيني واضح.
4. الواجهة: دعم RTL، محاذاة يمين، أيقونات متناظرة.
5. التواريخ/الأوقات: منطقة الرياض (UTC+3).

## أمثلة اتساق
- "نبني لك pipeline واضح" ✅ (مصطلح شائع)
- خلط مزعج: "نعمل sales الـ تحسين" ❌ — أعد الصياغة بوضوح.

## التحقق
مراجعة بشرية + كاشف ادعاءات يعمل بالعربية والإنجليزية.
