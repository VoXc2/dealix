# سياسة تسخين صناديق البريد (Mailbox Warmup)

قبل أي تواصل بارد، يُسخَّن الصندوق لبناء سمعة إيجابية.

## المبادئ
- ابدأ بحجم منخفض جداً وزِد تدريجياً (انظر `SENDING_RAMP_PLAN_AR.md`).
- ركّز على تفاعلات إيجابية (ردود حقيقية) لا مجرد حجم.
- لا تسخين عبر أدوات تبادل آلي مزيّف تنتهك الشروط.

## فترة التسخين (إرشادي)
| المرحلة | المدة | النشاط |
|---------|-------|--------|
| تهيئة | أسبوع 0 | إعداد SPF/DKIM/DMARC + هوية مرسل حقيقية |
| تسخين مبكر | أسبوع 1–2 | محادثات حقيقية منخفضة الحجم |
| تصاعد | أسبوع 3+ | زيادة تدريجية مع مراقبة الارتداد/الشكاوى |

## شروط
- ارتداد < 2%، شكاوى سبام ≈ 0 قبل الزيادة.
- توقف فوري عند أي إشارة سلبية.
- احترام الكبح وإلغاء الاشتراك دائماً.
