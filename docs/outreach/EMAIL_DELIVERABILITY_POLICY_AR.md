# سياسة قابلية تسليم البريد — Dealix (المرحلة 8)

سياسة تشغيلية لضمان وصول البريد الصادر بطريقة نظيفة ومتوافقة مع نظام حماية البيانات الشخصية (PDPL) من الهيئة السعودية للبيانات والذكاء الاصطناعي (سدايا). المبدأ الأساسي: **الإرسال موقوف افتراضيًا** (`send_enabled=false`) حتى تجتاز كل بوابات القابلية ويوافق المؤسس.

---

## 1. المبادئ الحاكمة

| المبدأ | القاعدة |
|--------|---------|
| الإرسال الفعلي افتراضيًا | موقوف — `send_enabled=false` |
| التشغيل التجريبي افتراضيًا | مفعّل — `dry_run=true` |
| الموافقة | إلزامية — `approval_required=true` |
| الحد اليومي | 250 **مسودة**/يوم مسموحة · 250 **إرسالة**/يوم ممنوعة قبل اجتياز البوابات |
| القوائم المشتراة | ممنوعة منعًا باتًا |
| البيانات الشخصية | لا تُسجَّل في السجلات أو التقارير |

> الإرسال الفعلي مسموح فقط عندما يكون الحكم ≥ `LIMITED_SEND_READY` **و** توجد موافقة المؤسس.

---

## 2. قائمة بوابات القابلية (إلزامية)

| البوابة | المطلوب |
|---------|---------|
| SPF | اجتياز |
| DKIM | اجتياز |
| DMARC | اجتياز |
| Reply-To صالح | عنوان رد حقيقي يُراقَب |
| نقطة/عملية إلغاء الاشتراك | رابط أو عملية إلغاء اشتراك تعمل |
| قائمة الاستبعاد | مفعّلة وتُطبَّق قبل كل إرسال |
| معالجة الارتداد | آلية لرصد الارتداد والتصرف |
| هوية المُرسِل | اسم مُرسِل وجهة صادقان |
| Google Postmaster Tools (أو ما يعادله) | مُهيّأ ويُراقَب |
| فصل نطاق/نطاق فرعي للإرسال | نطاق فرعي مخصص للإرسال |
| تدرّج الحجم | خطة تدرّج مُتّبعة |
| لا قوائم مشتراة | مصدر بيانات أول-طرف فقط |

المرجع التفصيلي بصيغة نجاح/فشل: `docs/outreach/DELIVERABILITY_READINESS_CHECKLIST.md`.

---

## 3. أحكام الجاهزية

| الحكم | المعنى التشغيلي |
|-------|-----------------|
| `NOT_READY` | بوابات أساسية فاشلة — لا مسودات للإرسال، لا تدرّج |
| `DRY_RUN_ONLY` (الافتراضي) | تشغيل تجريبي فقط — خطط ومسودات بلا إرسال |
| `LIMITED_SEND_READY` | الحد الأدنى لبدء إرسال محدود بعد موافقة المؤسس |
| `RAMP_READY` | صحي بما يكفي للتقدم في خطوات التدرّج |
| `PAUSE_REQUIRED` | تجاوز مؤشر خطر — إيقاف فوري للإرسال |

تتبّع تعريفات الأحكام `docs/gtm/MARKET_PRODUCTION_NAMING_CONVENTIONS.md`.

---

## 4. القوائم المشتراة — ممنوعة

- لا يجوز استيراد أو شراء أو استئجار قوائم بريد.
- مصدر العناوين: بيانات شركات عامة وجهات اتصال عمل بدور وظيفي فقط.
- أي عنوان غير مبرَّر المصدر يُستبعَد.

## 6. القواعد غير القابلة للتفاوض

1. لا توجيهات للتحايل على مرشّحات السبام أو تجاوز إلغاء الاشتراك.
2. لا ادعاءات زائفة أو مضلّلة في الرسائل.
3. لا قوائم مشتراة.
4. تبقى `send_enabled=false` حتى الحكم ≥ `LIMITED_SEND_READY` وموافقة المؤسس.
