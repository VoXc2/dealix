# سياسة قابلية تسليم البريد (Email Deliverability Policy)

## الأركان
1. **المصادقة:** SPF + DKIM + DMARC إلزامية ومتوافقة.
2. **الهوية:** مرسل حقيقي + reply-to مُراقَب.
3. **المحتوى:** مخصّص ≥ P1، بلا ادعاءات، بلا عناوين مزيّفة، مع إلغاء اشتراك.
4. **القوائم:** لا قوائم مشتراة/مكشوطة؛ تنظيف دوري؛ احترام الكبح.
5. **الحجم:** تصاعد تدريجي (`SENDING_RAMP_PLAN_AR.md`).
6. **المراقبة:** Postmaster/ما يعادله؛ ارتداد وشكاوى ضمن الحدود.
7. **الفصل:** نطاق فرعي للتواصل البارد.

## الحُكم (Verdict)
يُحدِّده Deliverability Agent: حالياً **DRY_RUN_ONLY** حتى تُجهَّز البنية.

## التوقف الإلزامي
لا إرسال عند غياب أي ركن أو عند إشارات سمعة سلبية. الإرسال دائماً
بموافقة بشرية و`send_enabled` خارجي.
