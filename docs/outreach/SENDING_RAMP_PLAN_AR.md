# خطة تدرّج الإرسال — Dealix (المرحلة 8)

خطة الحجم الأسبوعية للإرسال الصادر. الهدف بناء سمعة نظيفة عبر تدرّج صبور ومراقَب — لا أي تحايل على المرشّحات. لا يبدأ أي حجم فعلي قبل اكتمال التهيئة والحكم ≥ `LIMITED_SEND_READY` وموافقة المؤسس. الإرسال موقوف افتراضيًا (`send_enabled=false`).

النظام المتحكِّم في تطبيق هذه الخطة: `docs/outreach/SENDING_RAMP_OS_AR.md`.

---

## 1. الجدول الأسبوعي

| الأسبوع | الحجم اليومي | الشرط |
|---------|--------------|-------|
| الأسبوع 0 | 0–20/يوم | تهيئة فقط؛ مؤشرات صحية |
| الأسبوع 1 | 25–50/يوم | استقرار الأسبوع 0 |
| الأسبوع 2 | 50–100/يوم | استقرار الأسبوع 1 |
| الأسبوع 3 | 100–150/يوم | استقرار الأسبوع 2 |
| الأسبوع 4+ | 150–250/يوم | **فقط إذا كانت المؤشرات صحية** |

> سقف 250 ينطبق على **المسودات** يوميًا دائمًا؛ أما 250 **إرسالة**/يوم فممنوعة قبل اجتياز كل بوابات القابلية.

---

## 2. شروط الانتقال بين الأسابيع

| الشرط | المطلوب |
|-------|---------|
| التهيئة | `warmup_status=complete` |
| المصادقة | SPF/DKIM/DMARC اجتياز |
| الارتداد | < 2% طوال الخطوة الحالية |
| شكاوى السبام | < 0.1% طوال الخطوة الحالية |
| الحكم | ≥ `LIMITED_SEND_READY` (و`RAMP_READY` للتقدّم) |
| الموافقة | موافقة المؤسس لكل خطوة |

تفاصيل التهيئة: `docs/outreach/MAILBOX_WARMUP_POLICY_AR.md`.

---

## 3. مُحفّزات الإيقاف (Auto-Pause)

| المؤشر | الحد | الإجراء |
|--------|------|---------|
| معدل الارتداد | ≥ 2% | `PAUSE_REQUIRED` |
| شكاوى السبام | ≥ 0.1% | `PAUSE_REQUIRED` |
| سمعة Postmaster | منخفضة | `PAUSE_REQUIRED` |

عند الإيقاف: ضبط `send_enabled=false` و`daily_cap=0`، والرجوع إلى `docs/outreach/DOMAIN_HEALTH_RUNBOOK.md`.

---

## 4. القواعد غير القابلة للتفاوض

1. لا قفز فوق خطوة؛ التقدّم تدريجي وبموافقة لكل خطوة.
2. لا تجاوز للحدود اليومية المعلنة في الجدول.
3. لا قوائم مشتراة في أي خطوة.
4. تبقى `send_enabled=false` حتى الحكم ≥ `LIMITED_SEND_READY` وموافقة المؤسس.
