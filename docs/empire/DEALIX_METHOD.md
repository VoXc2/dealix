# Dealix Method — المنهجية التي تُباع

**الغرض:** بيع **طريقة تشغيل** وليس قائمة ميزات. المنهجية متماثلة عبر: الموقع، العروض، الديمو، التسليم، الشراكات، والتدريب لاحقاً.

## Operating Doctrine (سلسلة القيمة)

```text
Signal
  → Source
  → Owner
  → Approval
  → Action
  → Evidence
  → Decision
  → Value
  → Asset
```

أي شيء لا يدخل هذه السلسلة يُعامل كـ **noise** حتى يُثبت غير ذلك.

### مثال تطبيقي (lead من حملة)

1. Lead من حملة → **Source** واضح  
2. مالك المتابعة → **Owner** (مثلاً مبيعات)  
3. الرسالة الخارجية → **Approval**  
4. الإرسال اليدوي بعد الموافقة → **Action**  
5. تسجيل الرد والحالة → **Evidence**  
6. تحديد الاجتماع / الـ scope → **Decision**  
7. قيمة للعميل → **Value**  
8. **Proof Pack** أو insight مجهّل → **Asset**

## Dealix Method — 8 خطوات

| # | EN | AR | ملخص |
|---|-----|-----|--------|
| 1 | Detect | اكتشاف | أين تضيع القيمة؟ |
| 2 | Diagnose | تشخيص | workflow، بيانات، قنوات، مخاطر |
| 3 | Decide | قرار | الأولوية، المالك، القرار التالي |
| 4 | Draft | مسودات | رسائل، next actions، هيكل Proof |
| 5 | Approve | موافقة | لا فعل خارجي حساس بدون موافقة |
| 6 | Deliver | تسليم | سرعة + وضوح (انظر [COMMERCIAL_GATES.md](COMMERCIAL_GATES.md)) |
| 7 | Prove | إثبات | مصدر، حالة، **Truth Label** |
| 8 | Expand | توسعة | Sprint، Retainer، إحالة، شريك، benchmark |

## المسار التشغيلي القياسي (Lead → Proof)

```text
Form / wa.me (inbound)
  → Operating Board (كرت لكل lead + next_step)
  → Mini Diagnostic ≤ 24h (3 فرص + رسالة + قناة + مخاطرة)
  → Pilot 499 (~48h من الاهتمام)
  → Delivery ~7 أيام
  → Proof Pack
  → Sprint / Retainer / Referral / Partner
```

مواءمة مع حزمة Full Ops: [FULL_OPS_CLOSE_ENGINE_AR.md](../commercial/FULL_OPS_CLOSE_ENGINE_AR.md).

## ديمو ~12 دقيقة (ملخص)

1. المشكلة: leads بعد الحملات.  
2. Dealix لا يستبدل الوكالة ولا CRM.  
3. الطبقة **بعد الـ lead**: تصنيف، متابعة، موافقة، Proof.  
4. مسار UI داخلي (مثال): `/ar/business-now#strategy` → قطاع → simulate → تركيز → GTM أول 10 → Sales Script → **Diagnostic Scope**.  
5. الختام: عميل واحد، ~10 فرص، Proof Pack واحد — لا اشتراك كبير كبداية.
