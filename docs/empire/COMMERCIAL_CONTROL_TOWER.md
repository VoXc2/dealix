# Commercial Control Tower — برج المراقبة التجارية

**الغرض:** لوحة **قرار** يومية/أسبوعية/شهرية — ليست «dashboard للعرض» فقط. بدونها يتحوّل التنفيذ إلى عشوائية.

## أساس القياس (مواءمة Launch)

مؤشرات مثل: Total leads، inbound (مثلاً WhatsApp حسب السياسة)، **Diagnostics sent**، **Pilots offered/paid**، **Proof delivered**، **Needs data**.

## يومياً (Daily)

| مجال | أمثلة |
|------|--------|
| نشاط | Messages sent (حسب السياسة)، follow-ups، replies |
| قمع البيع | Demos، scopes، invoices |
| نتائج | Paid / commitments، Proof Packs delivered |
| شركاء | Partner conversations، affiliate leads |
| مخاطر | Blocked risks (قنوات غير آمنة، cold WA، إلخ) |
| تعلّم سريع | Best message، worst channel |
| غد | Tomorrow priority (شريحة واحدة، حركة واحدة) |

## أسبوعياً (Weekly)

- أفضل **شريحة ICP**  
- أفضل **رسالة** و**عرض** و**قناة**  
- أين يتوقف الـ funnel؟  
- أي **اعتراض** تكرر؟  
- أي **partner** جلب quality؟ أي **affiliate** جلب noise؟  
- **نضاعف / نوقف / لا نبنيه** (قرار صريح)

## شهرياً (Monthly — Board Pack)

- Revenue، Pipeline، Delivery، Support  
- Partners، Governance، Learning  
- **Next strategic bet** (رهان واحد بأعلى leverage)
