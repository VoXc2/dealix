# Affiliate Network — شبكة المسوّقين بالعمولة
<!-- Layer: empire | Owner: Founder | Date: 2026-05-17 -->

## English

The Dealix affiliate network is **governed — not open to everyone**. An open
affiliate program rewards volume; a governed one rewards compliant quality. It
starts small and stays controlled:

- 5–10 trusted affiliates only
- Approved scripts only
- Manual review of every submission
- Mandatory disclosure on every affiliate post
- Commission paid only after `invoice_paid`

### The affiliate workflow

```text
affiliate_submits_copy
   → compliance_review
       → approved  |  edit_required  |  blocked
   → tracking_link
   → lead_submitted
   → qualified
   → invoice_paid
   → commission_eligible
```

### Do NOT pay before all of these are true

- `invoice_paid`
- No refund issue
- Lead is not a duplicate
- Copy is compliant
- Disclosure is present

If any one is false, commission is not eligible. There is no exception.

### Forbidden — these void the affiliate immediately

- Guaranteed ROI claims
- Guaranteed compliance claims
- "Dealix sends automatically" (Dealix never sends without human approval)
- Spam
- Cold WhatsApp
- Fake proof
- Missing disclosure
- Unapproved claims

An affiliate who uses any forbidden tactic is removed from the network. The
network protects the Dealix Trust Promise; an affiliate who breaks it is not
an affiliate.

## العربية

شبكة المسوّقين بالعمولة في Dealix **محوكَمة — وليست مفتوحة للجميع**. البرنامج
المفتوح يكافئ الكمّ؛ والمحوكَم يكافئ الجودة الممتثلة. تبدأ صغيرة وتبقى مضبوطة:

- 5–10 مسوّقين موثوقين فقط
- سكربتات معتمدة فقط
- مراجعة يدوية لكل تقديم
- إفصاح إلزامي في كل منشور للمسوّق
- العمولة تُدفَع فقط بعد `invoice_paid`

### مسار عمل المسوّق

```text
affiliate_submits_copy
   → compliance_review
       → approved  |  edit_required  |  blocked
   → tracking_link
   → lead_submitted
   → qualified
   → invoice_paid
   → commission_eligible
```

### لا تدفع قبل تحقّق كل هذه

- `invoice_paid`
- لا مشكلة استرداد
- العميل المحتمل ليس مكرَّراً
- النصّ ممتثل
- الإفصاح موجود

إذا كان أي واحد منها غير صحيح، فالعمولة غير مستحقّة. لا استثناء.

### ممنوع — هذه تُلغي المسوّق فوراً

- ادعاءات عائد استثمار مضمون
- ادعاءات امتثال مضمون
- "Dealix تُرسل تلقائياً" (Dealix لا تُرسل أبداً دون موافقة بشرية)
- البريد المزعج (spam)
- واتساب بارد
- إثبات مزيّف
- غياب الإفصاح
- ادعاءات غير معتمدة

المسوّق الذي يستخدم أي تكتيك ممنوع يُزال من الشبكة. الشبكة تحمي وعد الثقة من
Dealix؛ ومن يكسره ليس مسوّقاً.
