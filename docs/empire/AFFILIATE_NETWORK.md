# شبكة الأفلييت — The Affiliate Network (controlled)

> **AR:** الأفلييت العام مبكراً خطر — الأفلييت المغلق مفيد.
> **EN:** An open affiliate program early is a risk — a closed one is useful.

---

## البداية / The Start

- 5–10 trusted affiliates فقط
- Approved scripts — نصوص معتمدة فقط
- Manual review — مراجعة يدوية
- Mandatory disclosure — إفصاح إلزامي
- Commission after `invoice_paid` only — عمولة بعد دفع الفاتورة فقط

---

## مسار العمل / Workflow

```text
affiliate_submits_copy
→ compliance_review
→ approved / edit_required / blocked
→ tracking link
→ lead submitted
→ qualified
→ invoice_paid
→ commission_eligible
```

---

## ممنوع / Forbidden

- ❌ ROI مضمون / guaranteed ROI
- ❌ Compliance مضمون / guaranteed compliance
- ❌ Dealix يرسل تلقائياً / Dealix auto-sends
- ❌ Spam · Cold WhatsApp
- ❌ Fake proof · No disclosure
- ❌ Claims غير معتمدة / unapproved claims

> هذه القائمة هي تطبيق مباشر للـ11 «غير القابل للتفاوض» على قناة الأفلييت.
> **EN:** This list is a direct application of the 11 non-negotiables to the affiliate channel.
