## 1. The rungs / الدرجات

| Rung / الدرجة | Offer / العرض | Price / السعر |
|---|---|---|
| 0 — Free / مجاني | Risk Score, Sample Proof Pack, Decision Passport template | Free / مجاني |
| 1 — First paid / أول مدفوع | 7-Day Sprint / 10-Lead Audit | **499 SAR — firm** / ثابت |
| 2 — Operating Sprint / سبرنت تشغيلي | Deeper workflow sprint | `recommended_draft` |
| 3 — Monthly Operating Layer / طبقة تشغيل شهرية | Retainer | `recommended_draft` |
| 4 — Partner / Agency co-branded / شراكة بعلامة مشتركة | Co-branded delivery | `recommended_draft` |
| 5 — Larger AI-ops program / برنامج تشغيل أكبر | Multi-workflow program | `recommended_draft` |

> Only **499 SAR** (rung 1) is a firm, committed price. Rungs 2–5 are
> `recommended_draft`: a real number is set only after **3+ paid pilots**
> inform it. There is **no firm "Custom Enterprise" tier** yet.
>
> فقط **499 ريال** (الدرجة 1) سعر ثابت ملتزم به. الدرجات 2–5 هي

## 2. The golden rule / القاعدة الذهبية

> **Don't cut price first — cut scope first.**
>
> **لا تخفّض السعر أولاً — خفّض النطاق أولاً.**

Price is the last lever, not the first. When a prospect hesitates, the
disciplined move is to make the offer smaller, not cheaper. A smaller scope
still produces real proof; a discount only trains the buyer to expect less for
less effort.

السعر هو الرافعة الأخيرة لا الأولى. حين يتردد العميل المحتمل، التصرف المنضبط هو
تصغير العرض لا تخفيضه. النطاق الأصغر لا يزال يُنتج إثباتاً حقيقياً؛ أما الخصم
فيدرّب المشتري على توقّع الأقل مقابل جهد أقل.

---

## 3. Cut scope, not price — examples / أمثلة: خفّض النطاق لا السعر

| Instead of / بدلاً من | Start with / ابدأ بـ |
|---|---|
| A 50% discount / خصم 50% | 10 leads only / 10 عملاء محتملين فقط |
| Running the whole company / تشغيل الشركة كاملة | One workflow / سير عمل واحد |
| A full Sprint / سبرنت كامل | One Proof Pack / حزمة أدلة واحدة |

Each smaller version is still a paid, complete unit of proof. None of them is a
favor or a free trial.

كل نسخة أصغر تظل وحدة إثبات مدفوعة كاملة. لا واحدة منها معروف أو تجربة مجانية.

---

## 4. No skipped rungs / لا تخطّي للدرجات

A prospect cannot jump from rung 0 to rung 5. The ladder is sequential because
each rung produces the evidence that justifies the next. Selling a large
program before proof exists is selling on hope.

العميل المحتمل لا يقفز من الدرجة 0 إلى الدرجة 5. السلم تسلسلي لأن كل درجة تُنتج
الدليل الذي يبرّر التالية. بيع برنامج كبير قبل وجود إثبات هو بيع على الأمل.
