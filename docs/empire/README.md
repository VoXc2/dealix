# Empire Layer — Index — طبقة الإمبراطورية — الفهرس
<!-- Layer: empire | Owner: Founder | Date: 2026-05-17 -->

## English

`empire/` is Dealix's commercial operating blueprint — the layer that turns a
product into a company. It defines the **category** Dealix owns, the **method**
it runs, the **standard** it enforces, the **offers** it sells, the **proof** it
produces, the **trust** it promises, the **partners and capital** it compounds.
It is an additive strategic layer: it does not redefine doctrine. The canonical
doctrine — the 11 non-negotiables — lives in
[`docs/00_constitution/`](../00_constitution/NON_NEGOTIABLES.md) and remains the
source of truth. Where this layer and the constitution disagree, the
constitution wins.

## العربية

`empire/` هو مخطّط Dealix التشغيلي التجاري — الطبقة التي تحوّل المنتج إلى شركة.
تُحدّد هذه الطبقة **الفئة** التي تملكها Dealix، و**المنهج** الذي تُشغّله،
و**المعيار** الذي تفرضه، و**العروض** التي تبيعها، و**الإثبات** الذي تُنتجه،
و**الثقة** التي تَعِد بها، و**الشركاء ورأس المال** الذي تُراكمه. إنها طبقة
استراتيجية إضافية: لا تُعيد تعريف الدستور. الدستور المرجعي — البنود الـ11 غير
القابلة للتفاوض — موجود في
[`docs/00_constitution/`](../00_constitution/NON_NEGOTIABLES.md) ويبقى مصدر
الحقيقة. وعند أي تعارض بين هذه الطبقة والدستور، الدستور هو الحاكم.
