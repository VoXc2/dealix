> **قاعدة حاكمة:** Dealix لا يبيع AI. Dealix يبيع تحويل الفوضى بعد الـlead وبعد
> تجربة AI إلى workflow محكوم: مصدر، مالك، موافقة، دليل، وخطوة تالية.

هذا الفهرس هو **طبقة المعيار التشغيلي** لـDealix — وثائق عقيدة تلخّص ما هي
Dealix كشركة. كل وثيقة هنا **طبقة ملخّص**، والمرجع القانوني يبقى في الوثائق
المُشار إليها في تذييل كل ملف. عند أي تعارض — الدستور هو الحاكم:
[`../institutional/DEALIX_CONSTITUTION.md`](../institutional/DEALIX_CONSTITUTION.md).

**عمل سابق ذو صلة:** [`../operating_empire/`](../operating_empire/) — مخططات
الإمبراطورية التشغيلية الأقدم؛ تبقى كما هي، وهذه الطبقة تختصرها وتحدّثها.

```
Market Confusion
```
