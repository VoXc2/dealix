## Read this first / اقرأ هذا أولاً

This set is **strategic narrative**. It describes direction and intent, not
shipped commitments. Where a topic has a binding rule, the canonical document
governs and is linked. In particular:

- **Pricing** is governed by [`../COMPANY_SERVICE_LADDER.md`](../COMPANY_SERVICE_LADDER.md).
  Only **499 SAR** (the 7-Day Sprint) is a firm price. Every higher tier is a
  `recommended_draft` until 3+ paid pilots inform a real number. No firm
  "Custom Enterprise" tier exists.
- **Proof Packs** are governed by [`../PROOF_PACK_V6_STANDARD.md`](../PROOF_PACK_V6_STANDARD.md).
- **Hard limits** are governed by
  [`../00_constitution/NON_NEGOTIABLES.md`](../00_constitution/NON_NEGOTIABLES.md).

هذه الحزمة **سرد استراتيجي** يصف الاتجاه والنية، لا التزامات منشورة. حيثما
يوجد لكل موضوع قاعدة مُلزِمة، فالوثيقة المرجعية هي الحاكمة ومربوطة أعلاه.

## 90-day priorities / أولويات الـ90 يوماً

The core decision: **do not build more product now.** Use what exists to close
the first paid Diagnostics, prove value with Proof Packs, then expand.

القرار الأساسي: **لا تبنِ منتجاً أكثر الآن.** استخدم ما بُني لإغلاق أول
تشخيصات مدفوعة، أثبت القيمة بحِزم الأدلة، ثم توسّع.

| Window | Focus / التركيز |
|---|---|
| Now (7 days) | Sample Proof Pack, Agency Proof Pilot one-pager, the Control Tower, 10 targets/day, 5 follow-ups/day, 1 partner conversation/day, **no feature building**. |
| 30 days | 1 paid pilot, 1 Proof Pack, 1 partner loop, 3 case-style insights, a clear winning message and ICP. |
| 60 days | 3 paid pilots, 1 Sprint proposal, 1 retainer candidate, 5 partners, a Proof Pack library, an objection engine. |
| 90 days | 5–8 paid pilots, 2 diagnostics, 1 Sprint, 1 retainer candidate, 10 partners, a mini benchmark report, a repeatable sales script. |

## Doctrine alignment / المواءمة مع الدستور

Every doc in this set honors the 11 non-negotiables: no scraping, no cold
WhatsApp, no fake proof, no live charge without approval, no guaranteed
outcomes, human approval for external actions, evidence trail, data
minimization. Strategy never overrides doctrine.

كل وثيقة في هذه الحزمة تحترم القواعد الإحدى عشرة غير القابلة للتفاوض.
الاستراتيجية لا تتجاوز الدستور أبداً.
