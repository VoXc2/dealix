# مسار التحويل إلى منتج — The Productization Path

> **AR:** لا تقفز إلى SaaS — اتبع التدرّج.
> **EN:** Don't jump to SaaS — follow the gradient.

---

## المراحل / The Stages

| المرحلة / Stage | الوصف / Description |
|-----------------|----------------------|
| Stage 1 | Internal tools — أدوات داخلية |
| Stage 2 | Client-visible reports — تقارير يراها العميل |
| Stage 3 | Client workspace — مساحة عمل للعميل |
| Stage 4 | Partner portal — بوابة الشركاء |
| Stage 5 | Self-serve modules — وحدات ذاتية الخدمة |
| Stage 6 | SaaS |

---

## قاعدة البناء / The Build Rule

```text
workflow جديد        → checklist
تكرّر مرتين          → template
تكرّر 3 مرات         → automation
تكرّر مع عميلين      → internal module
تكرّر مع 3 عملاء + retainers → product feature
```

---

## القاعدة / The Rule

> ❌ لا SaaS قبل **3 retainers** أو workflows مكرّرة مُثبتة.
> **EN:** No SaaS before 3 retainers or proven repeated workflows.

البناء يتبع التكرار، لا الطموح: لا feature قبل تكرار workflow، ولا منتج قبل
client pull حقيقي.
