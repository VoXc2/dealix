# Hiring Gates — بوابات التوظيف

**الغرض:** لا توظّف لأنّك مشغول. وظّف عندما **workflow تكرر** وأصبحت بوابات الإيراد واضحة.

## الآن

- **Founder:** Sales + Delivery + Approval  
- **System:** مسودات، scorecards، قوالب Proof، Operating Board

## بعد ~3 paid pilots

- **Part-time Ops Assistant** — جدولة، تنسيق، دخول بيانات، جودة ملفات

## بعد ~5–8 paid pilots

- **Delivery Operator** — يشغّل مسار التسليم القياسي تحت إشراف المؤسس

## بعد أول Retainer

- **Partner Manager / Growth Assistant** — شركاء، كو-بيع، دفاتر إحالات

## بعد ~3 Retainers

- **Customer Success + Product Engineer** — توسعة، جودة منتج، أتمتة آمنة

## ممنوع

- توظيف «مؤسس مبيعات» قبل وجود **Proof Pack متكرر**  
- توسيع فريق قبل **Commercial Control Tower** أسبوعي فعّال
