# بوابات التوظيف — Hiring Gates

> **AR:** لا توظّف لأنك مشغول — وظّف لأن workflow تكرّر.
> **EN:** Don't hire because you're busy — hire because a workflow has repeated.

---

## البوابات / The Gates

| البوابة / Gate | متى توظّف / When to hire |
|----------------|---------------------------|
| الآن / Now | Founder = Sales + Delivery + Approval. النظام = Drafts + Scorecards + Proof templates |
| بعد 3 paid pilots | Part-time Ops Assistant |
| بعد 5–8 paid pilots | Delivery Operator |
| بعد أول Retainer | Partner Manager / Growth Assistant |
| بعد 3 Retainers | Customer Success + Product Engineer |

---

## القاعدة / The Rule

> التوظيف يتبع **تكرار الـworkflow** وإشارات الإيراد المؤكَّد — لا الانشغال.
> **EN:** Hiring follows *workflow repetition* and confirmed-revenue signals — not busyness.

كل بوابة توظيف يجب أن تستند إلى دليل (paid pilots / retainers) لا إلى توقّع.
