# Authority Engine — محرك السلطة السوقية

**الغرض:** تحويل كل Proof إلى **محتوى مرجعي** — بدون كسر الثقة أو خرق الخصوصية.

## Pipeline

```text
Proof Pack
  → anonymized insight
  → LinkedIn post / مقال قصير
  → newsletter
  → partner asset
  → case-style post
  → webinar slide
  → إجابة اعتراض (objection answer)
```

## قواعد السلامة

- **لا اسم عميل** بلا إذن صريح.  
- **لا أرقام** غير **Payment-confirmed** أو **Observed** بتوضيح المصدر.  
- **لا شعارات** بلا إذن.  
- لا «نتيجة مالية» غير مثبتة.

## مثال آمن (نمط)

> في workflow راجعناه **لوكالة محلية** (دون تسمية)، المشكلة لم تكن في الإعلان، بل في **غياب owner بعد وصول الـ lead** — وهذا يفسّر توقف المتابعة رغم تدفق الاهتمام.
