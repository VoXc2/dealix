# Unit Economics — اقتصاد الوحدة لكل عرض

**الغرض:** فهم **تكلفة الوقت** و**معدل التحويل** بين طبقات السلم؛ إن لم يتحوّل العرض إلى التالي، راجع الجذور.

## 10-Lead Audit (499–990 SAR)

| البند | قيمة تقريبية |
|-------|----------------|
| وقت تنفيذ | 1–3 ساعات |
| هدف | إثبات القيمة + فتح Diagnostic |
| نجاح | ~20–30% تحويل نحو Diagnostic (هدف تشغيلي — قيّس فعلياً) |

## Agency Proof Pack (990–4,999 SAR)

| البند | قيمة تقريبية |
|-------|----------------|
| وقت تنفيذ | 4–8 ساعات |
| هدف | شراكة + co-selling |
| نجاح | شريك يجلب **عميلاً ثانياً** أو pilot متكرر |

## Diagnostic (4,999–15,000 SAR)

| البند | قيمة تقريبية |
|-------|----------------|
| وقت تنفيذ | 2–5 أيام بنطاق متفق |
| هدف | Sprint أو Retainer |
| نجاح | ~30–50% نحو Sprint أو مرشح Retainer (قيّس فعلياً) |

## Sprint (25,000+ SAR)

| البند | قيمة تقريبية |
|-------|----------------|
| مدة | ~3–4 أسابيع |
| هدف | تشغيل + proof + تكرار workflow |
| نجاح | تحويل إلى Retainer |

## Retainer (4,999–35,000 SAR / شهر)

| البند | قيمة تقريبية |
|-------|----------------|
| هدف | تكرار تشغيلي شهري |
| نجاح | churn منخفض + expansion + referrals |

## إذا تعثر التحويل، راجع

1. الألم غير صحيح (ICP)؟  
2. السعر/النطاق غير متطابق؟  
3. **Proof** ضعيف؟  
4. العميل غير مناسب؟  
5. التسليم **بطيء** (time-to-proof)؟
