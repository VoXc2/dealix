# Company Brain Sprint — اختبار قبول (Acceptance)

الخدمة **PASS** إذا:

1. رفع ≥10 ملفات (أو ما يعادلها في النطاق).  
2. **Document inventory** محدّث.  
3. **Retrieval** يعمل على العينة.  
4. إجابة **مع اقتباس/مرجع** عند وجود مصدر.  
5. عند غياب الأدلة: رد **insufficient evidence** / لا مصدر = لا إجابة.  
6. **لا تسريب** لوثائق مقيّدة حسب قواعد الوصول.  
7. جودة عربي/إنجليزي مقبولة للعرض التنفيذي.  
8. **eval report** (حتى لو مختصراً).  
9. **Proof pack** كامل المنطق.

المرجع: [`answer_with_citations.py`](../../../auto_client_acquisition/knowledge_os/answer_with_citations.py)، API Company Brain.
