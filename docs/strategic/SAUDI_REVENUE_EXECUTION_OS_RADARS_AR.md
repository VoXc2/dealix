# Dealix — نظام تنفيذ الإيرادات السعودي والرادارات الـ25

هذا المستند يثبت **التميّز التنافسي** لـ Dealix: ليس لأن المنتج «فيه AI» أو «صفحات كثيرة»، بل لأنه يصبح **نظام تشغيل لنمو الشركات السعودية** (Revenue Execution OS): عميل يدخل من واتساب/نموذج/إحالة → يُفهم وضعه → يخرج قرار → تُسلَّم نتيجة → تُثبت القيمة → تُحوَّل العلاقة إلى اشتراك أو شراكة.

**الفرضية المركزية**

```text
Dealix = Revenue Execution OS للشركات والوكالات السعودية
```

**ليس:** chatbot فقط · CRM فقط · صفحة خدمات فقط · أداة رسائل فقط · مولّد قوائم فقط.

**القيمة الحقيقية**

```text
وضوح تجاري → من تستهدف → ماذا تقول → قناة آمنة → منع مخاطر → Proof Pack → تحويل النشاط إلى Pipeline
```

**انسجام مع الـ loops:** كل عملية = Trigger → Input → Action → Output → Status → Metric → Decision → Next Loop. إذا كل loop يُنتج **evidence**، فعندك نظام تشغيل لا ارتجال. راجع [full_ops_pack/LEVEL_1_ACCEPTANCE_CHECKLIST_AR.md](../ops/full_ops_pack/LEVEL_1_ACCEPTANCE_CHECKLIST_AR.md).

**مرافق — الفئة والمعمارية وخارطة 12 شهراً:** هذا الملف يغطي **الرادارات والحلقات والتشغيل اليومي**. للنسخة الشاملة (تصميم الفئة، 7 طبقات، GTM، منافسة Salesforce/HubSpot، لوحة CEO، خريطة ambition) راجع [CATEGORY_DESIGN_AND_MASTER_OPERATING_SYSTEM_AR.md](CATEGORY_DESIGN_AND_MASTER_OPERATING_SYSTEM_AR.md) والملخص الإنجليزي [DEALIX_CATEGORY_EXECUTIVE_SUMMARY_EN.md](DEALIX_CATEGORY_EXECUTIVE_SUMMARY_EN.md).

## 1) الرادار الاستراتيجي — Strategic Radar

**يسأل:** من نخدم؟ لماذا الآن؟ لماذا نحن؟

### ICP (لا «كل الشركات»)

1. وكالات التسويق والنمو  
2. شركات خدمات B2B  
3. شركات تدريب/استشارات  
4. متاجر إلكترونية بمنتج واضح  

**ترتيب التركيز المقترح:** الوكالات أولاً → B2B خدمات → تدريب/استشارات → المتاجر.  
**السبب:** الوكالة تفهم leads والرسائل وProof Pack بسرعة، وقد تشتري لنفسها أو لعميلها أو تصبح قناة توزيع.

### الوجع (لا تبِع «AI»)

```text
فرص تضيع؟ واتساب/إيميل بدون متابعة؟ لا تعرف من تستهدف؟
لا رسائل عربية قوية؟ لا Proof يثبت القيمة؟
```

### وعد Dealix (قابل للقياس)

```text
خلال 24 ساعة: Mini Diagnostic
خلال 7 أيام: 10 فرص + رسائل + متابعة + Proof Pack (حسب العرض)
```

### مؤشر نجاح الرادار

- العميل يفهم Dealix في **جملة واحدة**.  
- الوكالة: «يفيدني مع عملائي». الشركة: «يجيب وضوح وفرص».

### سياق السوق (مراجع رسمية)

- **Monsha'at:** نمو قاعدة المنشآت الصغيرة والمتوسطة يعكس طلباً كبيراً على أدوات تشغيل ونمو، لا أدوات تسويق سطحية فقط. [Monsha'at — بيان صحفي عن المنشآت](https://www.monshaat.gov.sa/en/node/53859)  
- **رؤية 2030:** إطار التحول الاقتصادي للمملكة كقوة استثمارية عالمية. [وزارة الخارجية — رؤية السعودية 2030](https://www.mofa.gov.sa/en/ksa/Pages/vision.aspx)  
- **PDPL (نظرة تشغيلية):** جمع البيانات مرتبط بالغرض ومحدود بالحد الأدنى؛ التسويق المباشر يتطلب موافقة وسجلات — يدعم positioning «نمو بدون حرق القنوات». ## 6) رادار المبيعات — Sales Radar

**ابدأ بنظام لا بفريق مبيعات كبير.**

**Pipeline Level 1:** New Lead → Qualified → Diagnostic Sent → Pilot Offered → Accepted → Invoice Sent → Paid/Commitment → Delivery Started → Proof Delivered → Upgrade/Referral.

**حقول:** `lead_status` · `diagnostic_status` · `pilot_status` · `invoice_status` · `delivery_status` · `proof_pack_status` · `customer_status` (وسّع الـ Sheet عند الحاجة).

**Progression:** Free Diagnostic → Pilot 499 → Growth OS شهري → Partner/Retainer.

**جملة بيع:** «خلّنا نثبتها على عميل واحد أو أسبوع واحد، وبعدها نقرر.»
