# Venture Graduation Gate — بوابة تخرج المشروع الاستثماري

**متى تفصل وحدة كـ BU مستقلة أو venture؟** فقط عند اجتياز هذه البوابة.

مرافق: [`BUSINESS_UNIT_MODEL.md`](BUSINESS_UNIT_MODEL.md) · [`../company/DEALIX_SOVEREIGN_OPERATING_MODEL.md`](../company/DEALIX_SOVEREIGN_OPERATING_MODEL.md)

## شروط Gate I (Venture)

| الشرط | عتبة |
|--------|------|
| عملاء مدفوعون | **5+** |
| Retainers | **2+** |
| تسليم | متكرر وقابل للتنبؤ |
| Product module | واضح **ومستخدم** من العملاء |
| Owner | معين ومسؤول |
| Playbook maturity | **~80+** |
| هامش | صحيّ ومستدام |
| مكتبة proof | موجودة غير فارغة |

## مثال: Dealix Revenue OS

Lead Intelligence يبيع + Monthly RevOps به retainers + **Revenue OS** فعلي عند العملاء + B2B playbook ناضج + proof packs قوية → مرشح **venture / SKU مستقل** تحت القابضة.
