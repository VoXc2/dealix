## قرارات المحفظة

| قرار | متى |
|------|-----|
| **Scale** | تبيع · تتكرر · retainer · product module واضح |
| **Build** | وعد قوي يحتاج proof إضافي |
| **Pilot** | فكرة قوية لكن data/QA غير ناضجين |
| **Hold** | ليست أولوية الآن |
| **Kill** | لا proof ولا هامش ولا capital |
