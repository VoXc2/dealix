# AI Usage Policy — استخدام الذكاء الاصطناعي

**الطبقة:** سياسة Level 1 في [`../institutional/GOVERNANCE_PYRAMID.md`](../institutional/GOVERNANCE_PYRAMID.md).

## المبادئ

1. **كل مخرج AI** يخص workflow تجاري معرّف ومالك بشري.  
2. **لا إجراء خارجي** (رسائل، CRM، عملاء) بدون **موافقة** صريحة حيث ينطبق.  
3. **لا تدريب** على بيانات عميل دون أساس قانوني وعقدي.  
4. **المصادر إلزامية** للإجابات المعرفية عند المطالبة بها — انظر `no_source_no_answer`.  
5. **تتبع التشغيل** — تكاليف، نماذج، نسخ prompts حيث تُفرض السياسة — [`AI_RUN_LEDGER.md`](AI_RUN_LEDGER.md).
