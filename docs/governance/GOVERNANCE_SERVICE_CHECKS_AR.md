# فحص حوكمة الخدمة — أسئلة قبل التسليم

قبل اعتبار أي Sprint جاهزًا للتسليم للعميل، جاوب **نعم/لا/غير applicable** مع أدلة:

1. هل للداتا **مصدر** موثّق؟  
2. هل توجد **بيانات شخصية**؟ ما الأساس النظامي؟  
3. هل المخرج يحتوي **ادعاء مبالغ** أو ضمانات؟  
4. هل يوجد **إرسال خارجي** (بريد، واتساب، إلخ)؟ هل يحتاج موافقة؟  
5. هل يوجد **audit log** للخطوات الحساسة (حسب المنتج)؟  
6. هل المسودات مررت على [`draft_gate`](../../auto_client_acquisition/governance_os/draft_gate.py) / فحص ممنوعات؟  

**PDPL:** وثّق الغرض والمصدر والأساس (موافقة، عقد، مصلحة مشروعة، أو بيانات علنية/من مصدر علني حيث ينطبق). مرجع: [`PDPL_DATA_RULES.md`](../governance/PDPL_DATA_RULES.md).

**ممنوعات:** [`FORBIDDEN_ACTIONS.md`](../governance/FORBIDDEN_ACTIONS.md).
