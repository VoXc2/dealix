# Reversibility & Rollback

AI-assisted workflows must define rollback before execution.

## For internal actions

- previous state stored  
- change log created  
- rollback owner assigned  

## For external actions

- approval required  
- preview required  
- recipient list confirmed  
- mitigation plan defined  

## For reports

- version history  
- correction process  
- client notification process if needed  

## Rule

No action without a rollback or mitigation plan.
