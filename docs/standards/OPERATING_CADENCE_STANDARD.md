# Operating Cadence Standard

```text
No retainer without cadence.
No cadence without owner.
No owner without metric.
No metric without proof.
```

تحديث بيانات · مراجعة مخرجات · موافقة مسودات · تسجيل proof events · رفع مخاطر.

## Monthly

تقرير قيمة · ملخص proof · تحديث capability · حوكمة · next actions.

## Quarterly

مراجعة قدرة · نضج workflows · نطاق retainer · تحديث خارطة استراتيجية.
