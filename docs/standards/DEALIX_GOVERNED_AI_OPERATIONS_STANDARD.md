# Dealix Governed AI Operations Standard (D-GAOS)

## Identity

- Short name: **D-GAOS**
- AR: معيار Dealix لتشغيل الذكاء الاصطناعي المحكوم

## Goal

Convert random AI use into operating capability: clear data, defined workflow, AI assistance, human approval, runtime governance, Proof Pack, monthly improvement.

## Short form

```
Data → Workflow → AI → Governance → Proof → Operating Cadence
```

## Components

9 sub-standards (see `standards_os.standards_index.SubStandard`).

## Public ownership

Owned and versioned by the Office of the Standard. Method changes trigger recertification.
