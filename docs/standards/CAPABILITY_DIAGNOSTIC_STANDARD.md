# Capability Diagnostic Standard

## Measures (7 capabilities × 0–5)

Revenue, Customer, Operations, Knowledge, Data, Governance, Reporting.

## Maturity

0 Absent, 1 Manual, 2 Structured, 3 AI-Assisted, 4 Governed Workflow, 5 Optimized OS.

## Output

Current Score, Target, Transformation Gap, Data Readiness, Governance Risk, Recommended Sprint, Expected Proof, Retainer Path.

## Rule

> No serious AI project without a Capability Diagnostic.

Typed: `global_grade_os.capability_index.DCIReport`.
