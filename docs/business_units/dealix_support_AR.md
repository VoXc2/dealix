# Dealix Support — وحدة أعمال

**الطبقة:** Holding · Compound Holding Model
**المالك:** المدير العام لوحدة Support
**الحالة:** مسودة
**آخر مراجعة:** 2026-05-13
**النسخة الإنجليزية:** [dealix_support.md](./dealix_support.md)

## السياق
Dealix Support هي وحدة الأعمال التي ترفع جودة وسرعة خدمة العملاء في قنوات العميل القائمة (البريد، واتساب، الدردشة، التذاكر). لا تستبدل فريق الدعم — بل **تُعطي الفريق مكتباً معزَّزاً بالذكاء**: تصنيف، اقتراح ردود، قواعد تصعيد، تتبّع SLA. تقع في طبقة الوحدات داخل [`docs/holding/DEALIX_HOLDING_OS.md`](../holding/DEALIX_HOLDING_OS.md)، ومقترنة بـ `docs/capabilities/customer_capability.md`، وكتيباتها في `docs/CUSTOMER_SUCCESS_PLAYBOOK.md`.

## الوظيفة
تستوعب Support التذاكر التاريخية والحيّة، تبني مصنّفاً للنيّة والمشاعر، تقترح ردوداً مرتبطة بقاعدة معرفة العميل (عبر Dealix Brain)، توجِّه التصعيد عبر Governance Runtime، وتُبلِّغ أداء SLA إلى العمليات.

## الخدمات

| الخدمة | المدّة | المخرج |
|---|---|---|
| AI Support Desk | 2–4 أسابيع | مصنّف + Suggested Replies لأهم النوايا |
| Feedback Intelligence | 1–2 أسبوع | مواضيع واتجاهات مشاعر آخر 6 أشهر |
| Monthly Support AI (احتفاظ) | مستمر | ضبط، نوايا جديدة، تحسين التصعيد |

## وحدات المنتج داخل Core OS

| الوحدة | الوظيفة |
|---|---|
| Message Classifier | نيّة + مشاعر + إلحاح |
| Suggested Replies | مسودات مرتبطة بالمعرفة |
| Escalation Rules | توجيه قابل للضبط مع تدقيق |
| SLA Tracker | SLAs لكل قناة وتنبيهات |

## المؤشرات

- **زمن أول استجابة** — وسيط.
- **اتساق الرد** — توافق بين المسودة والرد النهائي.
- **فئات التذاكر** — % تصنيف صحيح.
- **دقّة التصعيد** — % توجيه صحيح أول مرّة.
- **معدّل الحلّ التلقائي** — % رسائل تُحلّ بلا تدخّل بشري.
- **زيادة CSAT** — مقارنة بالأساس.

## اعتمادات Core OS

| الوحدة | الاستهلاك |
|---|---|
| Data OS | سجل التاريخ، وسم PII |
| LLM Gateway | المصنّف + المسودة عبر Gateway |
| Governance Runtime | توجيه + تدقيق لكل تذكرة |
| Proof Ledger | Support Proof Pack |
| Capital Ledger | مكتبات نوايا، قوالب ردّ قطاعية |
| AI Control Tower | الكلفة، التقييم، الانحراف |
| Client Workspace | واجهة موظّف فوق نظام الدعم القائم |
