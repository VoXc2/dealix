# Dealix Governance — وحدة أعمال

**الطبقة:** Holding · Compound Holding Model
**المالك:** المدير العام لوحدة Governance
**الحالة:** مسودة
**آخر مراجعة:** 2026-05-13
**النسخة الإنجليزية:** [dealix_governance.md](./dealix_governance.md)

## السياق
Dealix Governance هي وحدة الأعمال التي تُمكّن المؤسسات من استخدام الذكاء الاصطناعي **بأمان ومساءلة وتوافق مع PDPL**. هي الوحدة التي تُحوّل كل قلق مؤسّسي — الخصوصية، التدقيق، الموافقات، مخاطر النماذج — إلى سطح تحكّم ملموس داخل Core OS. تقع في طبقة الوحدات داخل [`docs/holding/DEALIX_HOLDING_OS.md`](../holding/DEALIX_HOLDING_OS.md)، ومقترنة بـ `docs/capabilities/governance_capability.md`، وتدمج الحزمة القانونية في `docs/legal/COMPLIANCE_CERTIFICATIONS.md` و`docs/DPA_DEALIX_FULL.md`.

## الوظيفة
تُقيّم Governance جاهزية العميل للذكاء الاصطناعي؛ تصمم وتنشر سياسة استخدام؛ تُجري مراجعة بيانات متوافقة مع PDPL؛ تُفعِّل سجل سياسات وموافقات وسجل مخاطر وسجل تدقيق داخل Core OS؛ وتدير برنامج حوكمة شهرياً.

## الخدمات

| الخدمة | المدّة | المخرج |
|---|---|---|
| AI Readiness Review | 1–2 أسبوع | بطاقة جاهزية + خطة معالجة |
| AI Usage Policy | 2–3 أسابيع | سياسة منشورة + تواصل داخلي |
| PDPL-Aware Data Review | 2–4 أسابيع | خريطة بيانات، إقامة، احتفاظ، جاهزية DSR |
| AI Governance Program (احتفاظ) | مستمر | تحديثات، تدقيقات، استجابة حوادث |

## وحدات المنتج داخل Core OS

| الوحدة | الوظيفة |
|---|---|
| Policy Registry | سياسات بإصدارات وارتباطات بالمسارات |
| Approval Matrix | توجيه موافقات لكل سياسة |
| AI Risk Register | مخاطر مرتّبة بمالكين ومُخفّفات |
| Audit Log | سجل ثابت لنداءات النموذج والموافقات والتصدير |

## المؤشرات

- **سياسات منشأة.**
- **مخاطر مكتشفة.**
- **موافقات مسجَّلة.**
- **حوادث ممنوعة.**
- **حوادث حرجة** — الهدف 0.
- **زمن الاستجابة لطلب الأصحاب (DSR)** — وسيط.

## اعتمادات Core OS

| الوحدة | الاستهلاك |
|---|---|
| Governance Runtime | المستهلك الأصلي |
| Data OS | وسم PII، إقامة، احتفاظ |
| LLM Gateway | معرّف سياسة لكل استدلال |
| Proof Ledger | Governance Proof Pack |
| Capital Ledger | قوالب سياسات، تصنيف مخاطر |
| AI Control Tower | الانحراف، الشذوذ، انتهاكات السياسة |
