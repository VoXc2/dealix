# Dealix Brain — وحدة أعمال

## السياق
Dealix Brain هي وحدة الأعمال التي تحوّل معرفة الشركة — السياسات، الأدلّة، العقود، العروض، الأسئلة الشائعة، وثائق القطاع — إلى **إجابات بمصادر** للموظفين والعملاء والشركاء. هي الوحدة التي تجعل "اسأل الشركة" قدرة حقيقية محوكَمة قابلة للتدقيق. تقع في طبقة الوحدات داخل [`docs/holding/DEALIX_HOLDING_OS.md`](../holding/DEALIX_HOLDING_OS.md)، ومقترنة بـ `docs/capabilities/knowledge_capability.md`، ومرتكز الاسترجاع في `docs/EMBEDDINGS_PIPELINE.md`.

## الوظيفة
تستوعب Brain قاعدة معرفة العميل وتُصنّفها وتُفهرسها؛ تُوجِّه الأسئلة عبر LLM Gateway بالسياسة والبرومبت الصحيح؛ تُعيد إجابات بمصادر؛ تُبرز الفجوات حيث الأدلّة ناقصة؛ وتُشغّل مراقب حداثة يرصد المصادر القديمة أو المتناقضة.

## الخدمات

| الخدمة | المدّة | المخرج |
|---|---|---|
| Company Brain Sprint | 3–6 أسابيع | فهرسة 200–2000 وثيقة؛ سؤال وجواب بمصادر |
| Sales Knowledge Assistant | 2–4 أسابيع | سؤال وجواب لرجل المبيعات |
| Policy Assistant | 2–4 أسابيع | أسئلة السياسات / الموارد / الامتثال |
| Monthly Brain Management (احتفاظ) | مستمر | استيعاب، حداثة، تقييمات، تقرير فجوات |

## وحدات المنتج داخل Core OS

| الوحدة | الوظيفة |
|---|---|
| Source Registry | جرد وثيقة بوثيقة بمالك ونسخة وانتهاء |
| Answer with Citations | إجابات RAG بمراجع مدمجة |
| Knowledge Gaps | أسئلة مكتشفة بأدلّة غير كافية |
| Freshness Monitor | تنبيهات قديمة/متناقضة/منتهية |

## المؤشرات

- **عدد الوثائق المفهرَسة.**
- **إجابات بمصادر** — % إجابات بمصدر صحيح.
- **معدّل الأدلّة غير الكافية** — %.
- **خفض زمن البحث** — دقائق لكل سؤال.
- **التزام الحداثة** — % مصادر داخل النافذة.
- **تغطية المصادر** — % الوثائق الحرجة المفهرَسة.

## اعتمادات Core OS

| الوحدة | الاستهلاك |
|---|---|
| Data OS | سجل المصادر، وسم PII، الاحتفاظ |
| LLM Gateway | استدلال RAG، البرومبت، التقييمات |
| Governance Runtime | سياسات الوصول لكل وثيقة + تدقيق لكل استعلام |
| Proof Ledger | Brain Proof Pack: الوقت الموفَّر، عدد التدقيقات |
| Capital Ledger | خرائط معرفة قطاعية، تهيئات تضمين |
| AI Control Tower | كلفة الإجابة، التقييم، دقّة المراجع |
