# AI Control Tower

The AI Control Tower monitors AI work across Dealix.

## Monitors

- AI runs  
- cost  
- prompt versions  
- eval scores  
- governance flags  
- approval status  
- error rates  
- client-facing outputs  

## Alerts

- cost over budget  
- QA below threshold  
- governance block  
- PII detected  
- source missing  
- hallucination risk  
- approval overdue
