# AI Monitoring and Remediation

## Monitor

- AI run failures  
- low QA score  
- governance blocks  
- PII flags  
- source missing  
- cost overrun  
- hallucination risk  
- approval delays  

## Remediate

- pause workflow  
- request source  
- redact data  
- rerun with stronger model  
- escalate to human  
- update prompt  
- update rule  
- log incident
