# Product Telemetry

Track:

## Usage

- active clients  
- active workflows  
- reports generated  
- proof packs generated  
- approvals completed  

## Quality

- QA scores  
- eval pass rates  
- rework rate  

## Governance

- blocked actions  
- PII flags  
- approval delays  

## Value

- proof events  
- hours saved  
- opportunities identified  

## Economics

- cost per AI run  
- cost per project  
- margin per service  

## Why

SaaS without telemetry is blind.
