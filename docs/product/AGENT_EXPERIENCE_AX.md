# Agent Experience

## Good agent experience means

- user knows what the agent can do  
- user knows what the agent cannot do  
- agent explains uncertainty  
- agent shows sources  
- agent asks for approval when needed  
- agent logs actions  
- user can correct or override  

## Agent UI principles

1. Show task status.  
2. Show data sources.  
3. Show confidence/risk.  
4. Show required approval.  
5. Show audit trail.  
6. Allow human correction.
