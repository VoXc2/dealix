# Data Productization

Every recurring dataset becomes a data product.

## Examples

**Lead Dataset:**

- company_name  
- city  
- sector  
- source  
- relationship_status  
- notes  

**Support Dataset:**

- message  
- channel  
- customer_type  
- timestamp  
- category  
- resolution_status  

**Document Dataset:**

- title  
- owner  
- last_updated  
- sensitivity  
- source_type  
- allowed_users  

## Rule

Any service without a clear schema becomes chaos.
