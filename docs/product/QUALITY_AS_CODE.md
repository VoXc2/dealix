# Quality as Code

Quality standards should be testable.

## Quality Rules

- Every report must include executive summary.  
- Every report must include next action.  
- Every proof pack must include inputs and outputs.  
- Every Company Brain answer must include citation or insufficient evidence.  
- Every outreach draft must avoid guaranteed claims.  
- Every Arabic output must pass tone review.
