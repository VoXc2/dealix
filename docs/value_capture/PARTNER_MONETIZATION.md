# Partner Monetization

## Revenue models

Referral fee, revenue share, implementation fee, certification fee, partner-portal subscription (later), white-label fee (later).

## Do not start with white-label

It carries the highest brand risk.

## Start with

Referral → Advisory → Implementation under QA → Certified Partner.

## Partner Covenant

- No unsafe automation.
- No guaranteed claims.
- Proof Pack required.
- QA review accepted.
- Governance rules required.
- Audit rights accepted.
