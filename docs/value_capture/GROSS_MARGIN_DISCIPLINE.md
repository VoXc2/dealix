# Gross Margin Discipline

Track delivery hours, AI cost, review hours, QA hours, governance overhead, revision risk, communication cost.

## Targets

| Category | Min Margin |
| --- | ---: |
| Diagnostic | 75% |
| Sprint | 65–75% |
| Pilot | 55–70% |
| Retainer | 65% |
| Enterprise | 50–70% |
| Academy | 80% |
| Platform | 80% (mature) |

## If margin weak — choose one

raise price, reduce scope, convert custom to add-on, use templates, productize a step, refuse the client.
