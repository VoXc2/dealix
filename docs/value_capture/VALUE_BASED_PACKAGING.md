# Value-Based Packaging

Four tiers.

## Starter — Clarity
Capability Diagnostic, Data Readiness, basic Governance Review, Recommended Sprint.

## Growth — Operating Capability
Sprint, Proof Pack, Draft Pack / workflow, Mini Dashboard, Next-Step Recommendation.

## Scale — Managed Operating System
Monthly cadence, Client Workspace, Proof Timeline, Governance Review, Executive Report.

## Enterprise — Control & Assurance
AI Control Plane, Audit Exports, Agent Registry, Policy Registry, Risk Index, SLA.

## Principle

> Packages reflect the operating layer the client is ready to live with.
