# Platform Monetization

Do not launch a full platform too early.

## Conditions to start charging

- 3+ retainers
- client asks for a dashboard
- approval workflows repeat
- proof timeline repeats
- monthly reports repeat

## First platform fees

Client Workspace access, Proof Timeline, Approval Center, Value Dashboard, Governance Log.

## Do not add early

self-serve billing, marketplace, complex RBAC, white-label, deep integrations.

## Principle

> The platform price follows the pull — never leads it.
