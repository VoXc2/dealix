# Value Capture Ladder — السلم التجاري

```text
Free / low-friction education
→ Paid Diagnostic
→ Sprint
→ Pilot
→ Retainer
→ Managed Platform
→ Enterprise Program
→ Academy / Certification
→ Partner Revenue
→ Venture / Vertical OS
```

## طبقات موجزة

1. **Education** — لغة وثقة (Proof، Governance، Revenue Intelligence، Capability Score)  
2. **Paid Diagnostic** — Capability · Data Readiness · AI Governance Readiness  
3. **Sprint** — Revenue Intelligence · Company Brain · AI Quick Win  
4. **Pilot** — workflows أعمق للشركات الأكبر  
5. **Retainer** — Monthly RevOps / Brain / Governance / AI Ops  
6. **Managed Platform** — Client Workspace · Proof Timeline · Approval · Value Dashboard  
7. **Enterprise Program** — Control Plane · Audit · Agent/Policy Registry · Executive dashboard  
8. **Academy** — تعليم السوق  
9. **Partner** — توزيع  
10. **Venture** — وحدات ناضجة → منتج/شركة
