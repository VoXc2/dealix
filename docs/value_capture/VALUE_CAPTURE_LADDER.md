# Value Capture Ladder

Ten ordered stages.

```
Education → Paid Diagnostic → Sprint → Pilot → Retainer →
Managed Platform → Enterprise Program → Academy → Partner Revenue → Venture
```

## Stage purpose

1. Education — language and trust.
2. Diagnostic — buyer pays to understand.
3. Sprint — first capability.
4. Pilot — enterprise extension.
5. Retainer — operating cadence.
6. Platform — workspace fees.
7. Enterprise Program — control plane.
8. Academy — distribution.
9. Partner Revenue — referrals + revshare.
10. Venture — spinout.
