# Scope Control System

Scope creep kills service firms.

## Scope must define

Included, Excluded, Inputs Needed, Review Cycles, Delivery Format, Approval Owner, Success Metrics, Change Request Rules.

## Change Request Rule

Every new request is classified as one of:

```
Included
Minor Adjustment
Paid Add-on
New Sprint
Retainer Backlog
Rejected
```

## Example

> Client: "Can you send the messages on our behalf?"
>
> Dealix: "We produce drafts and an approvable outreach plan. We do not execute external sending without explicit approval, an allowed channel, and a permitted relationship/usage basis."

## Typed surface

`value_capture_os.scope_control.classify_change_request()`.
