# Value Capture Architecture

The architecture has two halves:

## Value creation

Cleaning data, ranking opportunities, reducing risk, accelerating reports, improving follow-up, generating Proof Packs, building governed workflows.

## Value capture

Sprint fee, Retainer, Platform fee, Enterprise contract, Academy revenue, Partner revenue, Venture upside.

## The fatal mistake

Creating large value and being paid like a task executor.

## The correction

Price for capability + proof + governance + operating cadence.
