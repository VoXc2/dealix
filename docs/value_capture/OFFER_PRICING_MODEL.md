# Offer Pricing Model

## Capability Diagnostic
Factors: company size, source count, workflow count, governance level. Outputs: Capability Score, Transformation Gap, Data Readiness, Governance Risk, Recommended Sprint, Expected Proof.

## Revenue Intelligence Sprint
Factors: account count, data quality, sectors / cities, scoring depth, draft pack type, executive report level. Message: "We create an initial Revenue Capability operable and measurable."

## Company Brain Sprint
Factors: source count, knowledge sensitivity, citation requirements, test question count, governance level.

## AI Governance Review
Factors: use case count, risk level, policy count, approval matrix complexity, audit needs.

## Monthly Retainer
Factors: workflow count, update frequency, report count, support level, governance level, review cycles.

## Principle

> Price reflects capability install, not deliverable count.
