# Proof-Driven Business Units

Each BU is measured by proof strength.

## Per BU

- **Dealix Revenue** — Revenue Proof Packs, pipeline clarity, account-scoring quality, retainer conversion.
- **Dealix Brain** — Knowledge Proof Packs, citation coverage, knowledge gaps closed, answer reliability.
- **Dealix Governance** — Risk Proof Packs, blocked actions, rules created, audit coverage.
- **Dealix Operations** — Time/Quality Proof Packs, hours saved, workflow repeatability, handoff success.

## Decision rule

```
Unit with strong proof + retainers + product module → scale.
Unit with weak proof → pilot only.
```
