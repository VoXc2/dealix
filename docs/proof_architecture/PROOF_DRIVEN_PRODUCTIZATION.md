# Proof-Driven Productization

The Proof Pack reveals what to build.

## Mappings

- Repeated proof report writing → `proof_pack_generator.py`.
- Repeated account scoring → `account_scoring.py`.
- Repeated source validation → `source_passport.py`.
- Repeated governance review → `policy_check.py`.
- Repeated retainer recommendation → `expansion_engine.py`.

## Rule

> Every repeated proof must produce a productization candidate.
