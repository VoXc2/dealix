# ROI Discipline

Three levels of value claim. Never mix.

## Levels

- **Estimated** — assumption-based forecast. Requires explicit assumption.
- **Observed** — measured during the engagement. Requires measurement method.
- **Verified** — confirmed by the client or measured after cadence. Requires client confirmation evidence.

## Rules

- Estimated never used as marketing.
- Only Verified powers public cases.
- Observed + Verified usable in sales.

## Typed surface

`proof_architecture_os.roi_discipline.ROIRecord` enforces field presence per level.
