# Value Metrics by Offer

Per-offer value metric catalog.

## Revenue Intelligence Sprint
records imported, duplicates removed, data quality score, accounts scored, top opportunities, drafts generated, unsafe outreach blocked, pipeline value estimated.

## Company Brain Sprint
sources registered, documents indexed, questions tested, citation coverage, insufficient evidence rate, knowledge gaps found, answer QA score.

## AI Governance Review
use cases inventoried, risk rules created, approval paths defined, unsafe actions blocked, policy gaps found, audit coverage designed.

## AI Quick Win Sprint
manual steps mapped, workflow created, approval path defined, hours saved estimate, QA pass rate, handoff completion.

## Executive Reporting Automation
manual reports reduced, sources connected, report freshness, decision recommendations generated, QA score, time-to-report reduction.

## Principle

> Each offer ships with the same proof skeleton but a different metric set; both must travel with the Proof Pack.
