# Enterprise Proof Architecture

Five proof levels, each a stronger evidence of impact.

## 1. Levels

1. **Activity Proof** — records imported, AI runs completed, drafts generated.
2. **Output Proof** — clean account list, ranked opportunities, executive report.
3. **Quality Proof** — QA score, Arabic quality, citation coverage, schema pass rate.
4. **Business Value Proof** — time saved, pipeline value, risk reduced, decision speed.
5. **Operating Capability Proof** — workflow owner, monthly cadence, approval path, dashboard, ongoing proof, retainer path.

## 2. Why Level 5 matters

Level 5 is what opens the retainer. Levels 1–4 are necessary but not sufficient.

## 4. The principle
