# Proof Dashboard

Aggregated proof state.

## Indicators

- Proof Packs delivered
- Average Proof Score
- Proof by type
- Proof-to-Retainer conversion
- Case candidate count
- Weak proof count
- Risk events blocked
- Value events recorded
- Client-confirmed value
- Estimated vs verified ratio

## Decision rule

Which proof deserves to become: retainer proposal / sales asset / benchmark input / productization signal / price increase justification.
