# مسار ترقية الخدمة (Service Upgrade Path)

```
P1 Sprint (تشخيص + خطة)
   │  قيمة مُثبتة + رغبة في التشغيل المستمر
   ▼
P2 Retainer صغير (3,000 ر.س)
   ▼  نمو الاحتياج/السعة
P2 متوسط (8,000) → P2 مؤسسي (20,000)
```

## قواعد الترقية
- الترقية تتبع **قيمة مُسلّمة** لا ضغط بيع.
- السعر النهائي لكل طبقة بموافقة بشرية.
- التجديد/الترقية يمران عبر فحص القيمة (`renewal_allowed`).
- لا وعود مضمونة عند الترقية.

## مسار التخفيض/الإيقاف
احترام طلب العميل؛ تسليم سلس؛ بيانات تُعامَل وفق سياسة الاحتفاظ/الحذف.
