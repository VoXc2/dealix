# Capability Blueprint: Knowledge Capability

## Business Purpose

Turn scattered knowledge into fast, source-backed answers the organization can trust.

## Typical Problems

- files in drives and inboxes  
- no canonical answers  
- repeated questions  
- outdated policies  
- no citation discipline  

## Required Inputs

- document inventory and owners  
- allowed vs restricted corpora  
- sensitivity labels  
- top question themes  

## AI Functions

- ingest and index approved docs  
- retrieve with citations  
- summarize with sources  
- flag stale or conflicting docs  
- insufficient-evidence responses when needed  

## Governance Controls

- permission mirroring for retrieval  
- no-source-no-answer where policy requires  
- redaction for outward-facing use  
- human review for regulated or external answers  

## KPIs

- docs indexed  
- citation rate  
- time to answer  
- insufficient-evidence rate (healthy when honest)  
- reduction in repeated questions  

## Services

- Knowledge Diagnostic  
- Company Brain Sprint  
- Sales / Ops Knowledge Assistant  
- Monthly Brain Management
