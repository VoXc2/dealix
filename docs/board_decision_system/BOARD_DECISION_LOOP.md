# Board Decision Loop

```
Collect signals → Score signals → Generate decision options →
Choose action → Assign owner → Execute → Measure proof → Update system
```

Inputs: sales, client adoption, proof, governance, data patterns, workflow friction, productization candidates, partner signals, market trends, financial metrics.

Outputs: Scale, Build, Pilot, Hold, Kill, Raise Price, Offer Retainer, Create Playbook, Create Benchmark, Create Business Unit, Create Venture Candidate.
