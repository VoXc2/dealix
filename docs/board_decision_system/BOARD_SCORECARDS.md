# Board Scorecards

Three scorecards: Offer, Client, Productization. Each on a shared 4-tier band: 85+ Scale/Build now, 70–84 Improve/MVP, 55–69 Pilot/Template, <55 Hold/Kill.

Typed: `board_decision_os.board_scorecards.compute_offer_scorecard / compute_client_scorecard / compute_productization_scorecard`.
