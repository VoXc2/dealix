## المبدأ

دعم **AI-first** لكن **bound إلى مصادر معتمدة**؛ أي إجابة عالية المخاطر تمر بموافقة بشرية أو تصعيد.

## الطبقات

| Tier | الوصف |
|------|--------|
| 0 | FAQ ذاتي |
| 1 | إجابة من knowledge base معتمد |
| 2 | مسودة AI + موافقة بشرية |
| 3 | مؤسس/اختصاصي |
| 4 | سياسة / أمن / قانون |

## التدفق

سؤال العميل → تصنيف intent → بحث KB → مسودة → risk score → إجابة آليّة إذا منخفضة المخاطر → تصعيد إذا متوسطة/عالية → تسجيل أدلة → كشف فجوة معرفة

## إجابة آلية مسموحة (منخفضة المخاطر)

نظرة خدمة · نطاق تسعير (عام) · عملية diagnostic · جدول تسليم مبدئي · شرح proof pack · مساعدة حجز · حالة فاتورة (حسب النظام) · FAQ عام

## تصعيد إلزامي

أمن/امتثال · استرداد · خصم · نطاق مخصص · تشخيص عميل محدد · شكوى · إذن دراسة حالة · حذف/تصدير بيانات (DSR)

## السياق السعودي

وضوح **PDPL**: الجمع، الأساس، الاحتفاظ، الحقوق، الشكاوى — في الواجهة وفي ردود الدعم القياسية؛ المراجعة القانونية للنص النهائي.
