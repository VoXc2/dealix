# Ecosystem Governance

## Partner Governance

Certification required, QA review required, approved messaging only, no unsafe promises, audit rights.

## Academy Governance

Curriculum tied to real Method, no hype, proof-backed examples, privacy-safe case studies.

## Benchmark Governance

Anonymized data, aggregated only, no confidential data, methodology disclosed, limitations stated.

## Platform Governance

Source passports, AI run logs, approval records, audit events, role permissions.

## Principle

> Ecosystem governance is identical in spirit to internal governance — the rules don't soften because the actor is external.
