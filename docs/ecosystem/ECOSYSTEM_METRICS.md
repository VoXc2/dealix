# Ecosystem Metrics

## Partner

partner leads, win rate, QA score, compliance incidents, revenue.

## Academy

registrations, completion rate, assessment pass rate, certified partners, course-to-lead conversion.

## Benchmark

downloads, inbound diagnostics, media mentions, partner usage, enterprise inquiries.

## Platform

workspace active clients, proof timeline views, approval completion, monthly active stakeholders, audit export requests.

## Venture

unit revenue, unit retainers, proof library size, module usage, owner readiness, margin.
