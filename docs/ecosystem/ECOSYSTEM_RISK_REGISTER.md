# Ecosystem Risk Register

## Risks

- Partner overpromises.
- Certification becomes weak.
- Benchmarks expose confidential info.
- Academy launches before method maturity.
- Platform overbuilt before pull.
- Venture spinout too early.

## Controls

Partner covenant, QA audits, benchmark anonymization policy, academy launch gate, platform pull signals, venture gate.

## Venture Gate v2

5+ paid clients, 2+ retainers, 10+ Proof Packs, avg proof ≥ 80, repeatable delivery, product module, owner, healthy margin, Core OS dependency.

Typed: `ecosystem_os.venture_gate.evaluate_venture_gate_v2()`.
