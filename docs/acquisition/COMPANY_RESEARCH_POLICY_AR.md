# سياسة بحث الشركات (Company Research Policy)

> الغرض: ضبط كيفية بحث الشركات المستهدفة: مصادر عامة أو بيانات يقدّمها المؤسس فقط، بلا كشط بيانات خاصة، مع انضباط مستوى الدليل وتسجيل المصادر.

## 1) مصادر البحث المسموحة

- معلومات عامة منشورة: موقع الشركة، صفحات عامة، إعلان وظيفة منشور، خبر منشور، حساب أعمال عام.
- بيانات يقدّمها المؤسس مباشرة (مستوى الدليل L4).
- ممنوع: قوائم مشتراة، كشط بيانات خاصة، استخراج بيانات تواصل غير معلنة، أي أتمتة على LinkedIn (بحث آلي، رسائل آلية، طلبات اتصال آلية).

## 2) الخصوصية وحماية البيانات

- لا نكشط بيانات خاصة ولا نخترق صفحات محمية أو خلف تسجيل دخول.
- لا نكتب أسرارًا أو بيانات شخصية حساسة (PII) في التوجيهات (prompts) أو السجلات (logs).
- لا نخزّن أرقام تواصل خاصة غير معلنة؛ نكتفي بالقنوات العامة المنشورة.
- نلتزم بنظام حماية البيانات الشخصية السعودي (PDPL) كحدّ أدنى.

## 3) احترام قائمة عدم التواصل

- قبل أي بحث موسّع أو تواصل، نتحقق من `do_not_contact`.
- أسباب الكبت تشمل: طلب عدم التواصل، عميل حالي، منافس، لا قناة عامة، مكرر، خارج النطاق.
- السجل المكبوت لا يُعاد تفعيله إلا بقرار موثّق.

## 4) انضباط مستوى الدليل

| المستوى | المصدر | كيف يُصاغ |
|--------|--------|-----------|
| L0 | تخمين قطاعي | احتمال فقط: «في هذا النوع من الشركات غالبًا...» |
| L1 | موقع الشركة | احتمال: «يبدو من الموقع أن...»، «قد...» |
| L2 | صفحة/وظيفة/خبر محدد | يمكن الإشارة للمصدر المحدد بوضوح |
| L3 | عدة مصادر عامة متوافقة | أقوى، مع ذكر تعدد المصادر |
| L4 | بيانات مقدّمة من الشركة | الأقوى، يُستخدم كأساس |

- قاعدة إلزامية: ما كان L0 أو L1 يُكتب كاحتمال، لا كحقيقة مؤكدة. ممنوع تحويل تخمين قطاعي إلى ادعاء جازم عن وضع الشركة.
- ممنوع أي وعد بإيراد مضمون مهما كان مستوى الدليل.

## 5) تسجيل المصادر

لكل معلومة مؤثرة نسجّل:

| الحقل | المثال |
|-------|--------|
| نوع المصدر | موقع الشركة / إعلان وظيفة / خبر / بيانات المؤسس |
| الوصف | «صفحة الدورات تستقبل تسجيلات عبر نموذج» |
| مستوى الدليل | L0–L4 |
| تاريخ الاطلاع | 2026-06-03 |

- يُسجَّل المصدر في حقل `source` (انظر `schemas/contact_target.schema.json`) ومستوى الدليل في `evidence_level`.
- عند تعارض المصادر، نخفض مستوى الثقة ونصوغ بحذر أكبر.

## 6) قائمة تحقق الامتثال

- [ ] كل المعلومات من مصدر عام أو بيانات المؤسس.
- [ ] لا كشط بيانات خاصة ولا أتمتة LinkedIn ولا قوائم مشتراة.
- [ ] لا أسرار أو PII في التوجيهات أو السجلات.
- [ ] `do_not_contact` مُراجَع.
- [ ] L0/L1 مصاغة كاحتمال.
- [ ] المصدر ومستوى الدليل مسجّلان.
- [ ] لا أي وعد بإيراد مضمون.
