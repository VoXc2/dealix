# بطاقة احتياج العميل (Client Need Card)

> الغرض: ملخّص مضغوط لاحتياج الشركة المرجّح والنظام المطابق، مشتق من حزمة استخبارات الشركة، لاستخدامه في القرار السريع والإسناد.

المخطط المرجعي: `schemas/client_need_card.schema.json`.

## متى تُنتَج

- بعد اكتمال حزمة استخبارات الشركة (Company Intelligence Pack) ومرورها بمراجعة الجودة.
- قبل صياغة العرض المصغّر أو إسناد المكالمة البشرية، كي يرى المؤسس والمتصل احتياج الشركة في بطاقة واحدة.
- تُحدَّث إذا وصلت بيانات جديدة من الشركة (L4) تغيّر النظام الموصى به أو الألم المرجّح.

## شرح الحقول

| الحقل | المعنى | إلزامي |
|------|--------|:------:|
| `company` | اسم الشركة | نعم |
| `sector` | القطاع | لا |
| `recommended_system` | معرّف أحد الأنظمة الخمسة | نعم |
| `likely_pain` | الألم المرجّح (صياغة احتمالية لا جازمة) | نعم |
| `why_this_system` | سبب مطابقة النظام للألم | لا |
| `first_mission` | أول مخرج صغير يقدّمه النظام | نعم |
| `proof_angle` | زاوية الإثبات الأولى | لا |
| `starter_price_sar` | السعر الابتدائي للنظام (يبدأ من) | لا |
| `risk_level` | `low` / `medium` / `high` | لا |
| `evidence_level` | `L0`–`L4` | نعم |

> ملاحظة: السعر يُذكر دائمًا بصيغة «يبدأ من»، لا كسعر نهائي مقفل. الأسعار الابتدائية: نظام تشغيل الإيرادات 4,500 SAR، نظام القيادة التنفيذية 5,500 SAR، نظام استعادة المتابعة 3,500 SAR، نظام عملاء واتساب 4,500 SAR، نظام العروض والإثبات 3,000 SAR.

## قواعد الصياغة

- الألم يُربط بإشارة عامة، ويُصاغ كاحتمال إن كان الدليل L0/L1.
- لا وعود مضمونة بالإيراد.
- النظام الموصى به يجب أن يطابق `recommended_system` في الحزمة الأصلية.

## مثال مُعبّأ

```json
{
  "company": "SkillUp Arabia",
  "sector": "تدريب",
  "recommended_system": "follow-up-recovery-os",
  "likely_pain": "في شركات التدريب المشابهة غالبًا تدخل استفسارات عبر واتساب والنموذج وقد تُترك بعضها دون متابعة في وقتها",
  "why_this_system": "اعتماد واضح على الاستفسارات الواردة بلا قائمة متابعة موحدة — وهو جوهر نظام استعادة المتابعة",
  "first_mission": "قائمة متابعة أولية + مجموعة رسائل استعادة خلال 7 أيام",
  "proof_angle": "كم استفسار متروك بلا متابعة، وكم منه قابل للاستعادة",
  "starter_price_sar": 3500,
  "risk_level": "low",
  "evidence_level": "L2"
}
```

## في سطر واحد (صيغة عرض سريعة)

> SkillUp Arabia — تدريب — نظام استعادة المتابعة — غالبًا استفسارات متروكة بلا متابعة — أول مهمة: قائمة متابعة + رسائل خلال 7 أيام — يبدأ من 3,500 SAR — دليل L2 — مخاطرة منخفضة.
