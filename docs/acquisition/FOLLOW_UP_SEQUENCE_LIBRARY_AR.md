# مكتبة تسلسلات المتابعة (Follow-up Sequence Library)

> الغرض: تسلسلات متابعة على هيئة مسودات بريد فقط (لا واتساب بارد ولا أتمتة)، لكل نظام، مع شروط التوقف.

المخطط المرجعي: `schemas/follow_up_sequence.schema.json`.

## قواعد

- القناة `email` فقط (أو `human_call_note` كملاحظة لمكالمة بشرية). لا واتساب بارد ولا إرسال آلي.
- كل خطوة مسودة تبقى مسودة حتى موافقة المؤسس قبل الإرسال اليدوي.
- لا لغة ضغط ولا وعود مضمونة. الألم يُطرح كاحتمال إن كان دليله L0/L1.
- يتوقف التسلسل فورًا عند تحقّق أي شرط توقف.

## شروط التوقف العامة

- رد العميل (إيجابًا أو سلبًا).
- طلب عدم التواصل → ضبط `do_not_contact = true`.
- اكتمال خطوات التسلسل دون رد.
- ظهور سبب كبت (منافس، عميل حالي، خارج النطاق).
