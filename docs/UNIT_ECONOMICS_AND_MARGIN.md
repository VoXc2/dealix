## 3. تكلفة الاستحواذ (Customer Acquisition Cost)

### الوضع الحالي (قبل pilots)
```
CAC = وقت المؤسس في البيع + تكلفة التسويق
    = ~3 ساعات/عميل محتمل × 50 SAR/ساعة
    = ~150 SAR CAC (warm intro فقط)

LTV (Diagnostic Starter only) = 4,999 SAR
LTV:CAC = ~33x (جيد لـ one-time)

LTV (→ Governed Ops Retainer، 6 شهور بمتوسط 9,000 SAR) = 4,999 + (9,000 × 6) = 58,999 SAR
LTV:CAC = ~393x (تقديري — ممتاز)
```
