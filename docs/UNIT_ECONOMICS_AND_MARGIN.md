## 3. تكلفة الاستحواذ (Customer Acquisition Cost)

### الوضع الحالي (قبل pilots)
```
CAC = وقت المؤسس في البيع + تكلفة التسويق
    = ~3 ساعات/عميل محتمل × 50 SAR/ساعة
    = ~150 SAR CAC (warm intro فقط)

LTV (Sprint only) = 499 SAR
LTV:CAC = ~3.3x (جيد لـ one-time)

LTV (→ Managed Ops، 6 شهور) = 499 + (3,999 × 6) = 24,493 SAR
LTV:CAC = ~163x (ممتاز)
```
