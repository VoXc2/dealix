# Revenue Wedge v2

The wedge offer remains the **Revenue Intelligence Sprint**. This document specifies the v2 execution — the version operated under the Command Model.

## 1. Why it remains the wedge

- Closest to money.
- Easy to explain.
- Produces a Proof Pack quickly.
- Opens the Monthly RevOps OS retainer.
- Exercises `data_os`, `revenue_os`, and `reporting_os` in one engagement.

## 2. Outputs

- **Data Quality Score** — duplication, completeness, freshness, ICP fit.
- **Cleaned Account List** — deduplicated, segmented.
- **Top 50 Ranked Accounts** — ranked with feature attribution.
- **Top 10 Next Actions** — owned and dated.
- **Draft Pack** — draft-only outreach.
- **Mini Pipeline** — instrumented stages with definitions.
- **Proof Pack** — Revenue Proof Pack with before/after.

## 3. Difference vs. lead generation

| Lead generation | Dealix Revenue Intelligence |
| --- | --- |
| Lists + scraping | Permissioned data only |
| Cold outreach | Draft-only outputs |
| Vague promises | Proof Pack with metrics |
| No governance | Runtime governance + audit |
| No pipeline | Mini Pipeline + discipline |

## 4. Command Model wiring

- **Sense:** the diagnostic identifies the revenue capability gap.
- **Decide:** the sprint is the structured decision.
- **Govern:** every output passes through the governance runtime.
- **Execute:** delivery is productized — same playbook every time.
- **Prove:** Revenue Proof Pack closes the engagement.
- **Compound:** ICP scoring patterns, sector objections, and draft templates land in the Capital Ledger.
- **Scale:** retainer conversion converts the proof into recurring revenue.

## 5. Pricing principles

- Fixed price, productized.
- Priced above the value of one month of Monthly RevOps OS.
- Multi-region, multi-product, multi-language as priced add-ons.

## 6. Anti-patterns

- Scope drift into auto-sent campaigns.
- Treating the sprint as a list build.
- Skipping the mini pipeline because the buyer “already has a CRM.”
- Closing without a Proof Pack.

## 7. The principle

> The wedge is not a discount. It is a *capability install* sold at fixed scope and fixed proof.
