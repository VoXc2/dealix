# Dealix Support FAQ

## General Questions

### What is Dealix?
Dealix is an AI Operating System for Saudi B2B companies. It's not a chatbot or CRM—it's a daily operating platform that connects revenue, follow-up, decisions, compliance, and client delivery into one system.

### Who is Dealix for?
Dealix is designed for:
- B2B companies with 10-500 employees
- Teams managing complex sales pipelines
- Founders who need visibility and control
- Companies using WhatsApp for business communication
- Organizations requiring compliance and auditability

### How is Dealix different from a CRM?
Traditional CRMs are databases. Dealix is an operating system:
- **CRM**: Stores data, requires manual updates
- **Dealix**: Generates insights, creates drafts, suggests actions
- **CRM**: Tracks what happened
- **Dealix**: Tells you what to do next
- **CRM**: No built-in approval flows
- **Dealix**: Every sensitive action requires review

## Features & Functionality

### What is the Command Room?
The Command Room is your daily operational cockpit:
- **Pipeline View**: See all deals and their stages
- **Draft Queue**: Review AI-generated messages before sending
- **Actions Panel**: See what needs your attention today
- **WhatsApp Inbox**: Monitor all conversations

### What is Brain OS?
Brain OS is your strategic decision system:
- **Signals**: Market, competitor, customer insights
- **Decisions**: Log and track important decisions
- **Risks**: Identify and mitigate threats
- **Opportunities**: Capture and evaluate chances
- **Metrics**: Track decision outcomes

### How does WhatsApp integration work?
1. **Connection**: Connect your WhatsApp Business API account
2. **Templates**: Create message templates
3. **Draft Mode**: All messages start as drafts
4. **Approval**: You review and approve before sending
5. **Tracking**: Monitor delivery and responses

### What does "draft-only mode" mean?
Draft-only mode means:
- No message is sent automatically
- AI generates drafts for your review
- You approve, edit, or reject each message
- Nothing leaves the system without your explicit approval
- This ensures quality and compliance

### Can I customize pipeline stages?
Yes:
1. Go to Command Room → Settings
2. Click "Manage Stages"
3. Add, edit, or reorder stages
4. Set stage-specific actions
5. Save changes

## Data & Privacy

### Where is my data stored?
- All data is stored in Saudi Arabia
- Encrypted at rest and in transit
- Regular backups with 30-day retention
- Compliant with Saudi PDPL regulations

### Is my data secure?
Yes:
- **Encryption**: AES-256 for data at rest, TLS 1.3 for data in transit
- **Access Control**: Role-based permissions
- **Audit Logs**: All actions logged and reviewable

Yes:

4. Download export

## Troubleshooting

### I can't login
Try these steps:
1. Check email and password (case-sensitive)
2. Clear browser cache and cookies
3. Try different browser
4. Reset password via email
5. Contact support if issue persists

### Messages aren't sending
Check these:
1. Verify WhatsApp connection in Settings
2. Ensure messages are approved (not in draft)
3. Check WhatsApp API status
4. Review error messages in Command Room
5. Contact support if issue persists

### Dashboard is slow
Try these:
1. Clear browser cache
2. Check internet connection
3. Try different browser
4. Disable browser extensions
5. Contact support if issue persists

### I'm not receiving notifications
Check these:
1. Go to Settings → Notifications
2. Verify notification preferences
3. Check email spam folder
4. Verify WhatsApp notifications enabled
5. Contact support if issue persists
