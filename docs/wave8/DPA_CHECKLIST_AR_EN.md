## قبل التوقيع / Before Signing

### الطرف المُعالِج (Dealix) / Data Processor (Dealix)
- [ ] اسم الشركة مُسجَّل صحيحًا / Company name registered correctly
- [ ] عنوان الشركة صحيح / Company address correct
- [ ] بيانات التواصل محدّثة / Contact details updated
- [ ] نطاق المعالجة محدَّد بدقة / Processing scope clearly defined

### الطرف المتحكّم (العميل) / Data Controller (Customer)
- [ ] اسم الشركة / رقم السجل التجاري / CR number present
- [ ] ممثّل قانوني مخوَّل يوقّع / Authorized legal representative signs
- [ ] توقيع إلكتروني أو خطّي مقبول / Electronic or physical signature accepted

### البيانات المُعالَجة / Data Being Processed
- [ ] تصنيف البيانات محدَّد / Data categories specified
- [ ] الغرض من المعالجة مُوثَّق / Processing purpose documented
- [ ] مدة الاحتفاظ محدَّدة / Retention period defined
- [ ] مكان التخزين مذكور (Railway / KSA region) / Storage location noted

### الحقوق والالتزامات / Rights & Obligations
- [ ] حق الوصول للبيانات / Right of access
- [ ] حق التصحيح / Right to rectification
- [ ] حق المحو (DSR) / Right to erasure (DSR)
- [ ] الإخطار بالاختراق خلال 72 ساعة / Breach notification within 72h
- [ ] حظر النقل خارج المملكة بدون موافقة / No cross-border transfer without consent

### التوافق مع PDPL / PDPL Compliance
- [ ] مراجعة المحامي ✅ / Lawyer reviewed ✅
- [ ] تاريخ السريان محدَّد / Effective date specified
- [ ] آلية النزاع محدَّدة / Dispute resolution mechanism specified
- [ ] القانون المُطبَّق: نظام PDPL السعودي / Governing law: Saudi PDPL

- [ ] تاريخ التوقيع مُسجَّل في Consent Record / Signing date logged in Consent Record
