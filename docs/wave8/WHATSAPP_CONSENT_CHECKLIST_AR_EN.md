# قائمة تحقّق موافقة WhatsApp / WhatsApp Consent Checklist
<!-- Wave 8 — draft_only — REQUIRES LAWYER REVIEW BEFORE USE -->

> ⚠️ يجب الحصول على موافقة صريحة قبل إرسال أي رسالة WhatsApp للعميل.
> Explicit consent required before sending any WhatsApp message to a customer.

---

## المتطلّبات الأساسية / Core Requirements

### قبل أي رسالة WhatsApp / Before Any WhatsApp Message

- [ ] العميل أعطى رقمه طوعًا / Customer provided phone number voluntarily
- [ ] العميل طلب التواصل عبر WhatsApp / Customer requested WhatsApp contact
- [ ] تم إخباره بأن Dealix سيتواصل عبر WhatsApp / Informed Dealix will contact via WhatsApp
- [ ] لم يُعطَ الرقم من مصدر خارجي / Number not obtained from external source

### الرسائل المسموح بها / Allowed Messages

- [ ] ردود على رسائل وارده من العميل / Responses to inbound messages
- [ ] إشعارات تمّت الموافقة عليها مسبقًا / Pre-approved notifications
- [ ] Templates معتمدة من Meta / Meta-approved templates only

### الرسائل المحظورة / Blocked Messages (Hard Gates)

- ❌ رسائل ترويجية بدون موافقة / Promotional messages without consent
- ❌ WhatsApp بارد لأرقام غير معروفة / Cold WhatsApp to unknown numbers
- ❌ رسائل جماعية / Bulk WhatsApp
- ❌ إرسال تلقائي بدون موافقة المؤسس / Auto-send without founder approval

### بعد الحصول على الموافقة / After Obtaining Consent

- [ ] تسجيل الموافقة في Consent Record / Log consent in Consent Record
- [ ] تاريخ الموافقة مُسجَّل / Consent date logged
- [ ] طريقة الموافقة مُوثَّقة (شفهي / مكتوب / نموذج) / Method documented
