## معالجة الطلب / Request Processing

### المرحلة ١: التحقّق من الهويّة / Identity Verification
- [ ] طلب وثيقة هويّة / Request ID document
- [ ] تأكيد أن الطالب هو صاحب البيانات / Confirm requester is data subject
- [ ] تاريخ التحقّق: ___________

### المرحلة ٢: البحث في البيانات / Data Search
- [ ] بحث في قاعدة البيانات / Search database
- [ ] بحث في Google Sheets / Search Google Sheets
- [ ] بحث في ملفات العميل / Search customer files
- [ ] قائمة البيانات الموجودة: ___________

### المرحلة ٣: التنفيذ / Execution
- [ ] **Access:** إرسال نسخة من البيانات
- [ ] **Rectification:** تصحيح البيانات الخاطئة
- [ ] **Erasure:** حذف جميع البيانات + تأكيد
- [ ] **Portability:** تصدير بيانات بصيغة JSON/CSV

### المرحلة ٤: الإغلاق / Closure
- [ ] إخطار صاحب البيانات بالنتيجة
- [ ] تسجيل الطلب في Consent Record
- [ ] تاريخ الإغلاق (يجب خلال 30 يوم): ___________

## ملاحظات PDPL / PDPL Notes
- المهلة القانونية: 30 يومًا من تاريخ الاستلام
- Legal deadline: 30 days from receipt date
- إذا تعذّر الإغلاق خلال 30 يومًا: إخطار الجهة التنظيمية
