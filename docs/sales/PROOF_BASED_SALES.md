# Proof-Based Sales

Sell with evidence—not vague promises.

## Before formal case studies

Use:

```text
sample proof pack
demo CSV
sample report
before/after mock (labeled as illustration)
```

## After first client

Use:

```text
anonymized proof pack
case-style post
numbers without exposing identity
```

## After ~3 clients

Use:

```text
vertical benchmark themes
common problems report
industry playbook excerpts
```

## Rules

```text
No fake proof.
No guaranteed results.
No invented numbers.
```
