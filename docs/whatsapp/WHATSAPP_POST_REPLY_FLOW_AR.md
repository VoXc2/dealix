# تدفق ما بعد الرد — WhatsApp بعد رد إيجابي
## Post-Reply Flow — WhatsApp After Positive Reply

---

## 1. الغرض

عندما يصل رد إيجابي من عميل محتمل، نوجهه إما لحجز اجتماع أو واتساب — بناءً على سياق العلاقة.

---

## 2. شجرة القرار

```
رد إيجابي من Reply Handling Agent
    ↓
هل عندنا موافقة واتساب سابقة؟
    ├── نعم → وجه لـ WhatsApp Concierge
    └── لا → اسأل: "تحب نحاول واتساب ولا نحجز اجتماع؟"
                ├── واتساب → اطلب موافقة واتساب → WhatsApp Concierge
                └── اجتماع → اطلب حجز → Booking Flow
```

---

## 3. قالب السؤال الأول بعد الرد الإيجابي

```
رائع نسمع منك! 😊

عندنا خيارين للتواصل:

📅 حجز اجتماع
   نتكلم بشكل رسمي ومهني
   — أفضل إذا عندك موضوع محدد

💬 واتساب
   نتابع بشكل سريع ومباشر
   — أفضل إذا تحب التواصل غير رسمي
   — يحتاج موافقتك

**ما أعرف — اقترح علي:**

أزرار:
- [أفضل اجتماع] → يرسل رابط الحجز
- [أفضل واتساب] → يطلب موافقة واتساب
- [ما أدري، اقترح] → يوصي اجتماع
```

---

## 4. قالب طلب موافقة واتساب

```
عندك رقم واتساب نتواصل فيه؟

قبل ما نبدأ واتساب، محتاج منك:

✅ موافقة صريحة على التواصل عبر واتساب
✅ رقم صحيح نقدر نوصلك فيه
✅ تفهم إننا لا نرسل رسائل تسويقية بدون إذنك

⚠️ ملاحظة مهمة:
- ما نرسل أي شيء بدون ما توافق
- تقدر توقف التواصل في أي وقت
- كل شيء يسجل للتدقيق

**ما أعرف — اقترح علي:**

أزرار:
- [موافق، ابدأ] → يسجل الموافقة → WhatsApp Concierge
- [أفضل اجتماع] → يرسل رابط الحجز
```

---

## 5. قالب الترحيب في واتساب (بعد الموافقة)

```
أهلاً! 👋

أنا [المساعد] من Dealix.
تواصلت معك بخصوص [موضوع الرسالة الأصلية].

قبل ما نبدأ، محتاج أطمن على نقطتين:

1️⃣ الموافقة
   أنت وافقت على التواصل عبر واتساب
   تقدر توقف في أي وقت

2️⃣ الخصوصية
   نحتفظ بسجل المحادثة للتدقيق
   لا نشارك بياناتك مع أطراف ثالثة

هل نبدأ؟ 👇

أزرار:
- [نبدأ] → يبدأ فحص الجاهزية
- [عندسؤال] → يحول لمتخصص
```

---

## 6. قالب "ما أعرف — اقترح علي" بعد الترحيب

```
ما أعرف عن وضعك حتى الآن:
- أنت شركة سعودية [القطاع]
- عندك [نوع المشكلة] تحتاج حل
- الميزانية تقريباً [المدى]

**ما أعرف — اقترح علي:**

أزرار:
- [Quick Win Sprint] → يشرح وينصح Quick Win
- [تشخيص كامل] → يبدأ فحص الجاهزية الشامل
- [أدلة ونتائج] → يعرض دليل الأسعار
```

---

## 7. فحص الجاهزية — السؤال الأول

```
🔍 فحص جاهزية الإيرادات

عندك دقيقة؟ أبي أفهم وضعك بشكل سريع
— عشان أقدر أساعدك بشكل أفضل.

السؤال الأول:
**كيف يجيك عملاء جدد اليوم؟**

أزرار:
- [تسويق رقمي] → يقود للتسويق الرقمي
- [مرجعين] → يقود للمرجعين
- [معارف] → يقود للمعارف
- [لا أدري] → يقود للتشخيص الكامل
```

---

## 8. ربط بالتدفقات الأخرى

```
ما بعد الرد الإيجابي:
    ↓
WhatsApp Concierge (عبر موافقة)
    ↓
Client Assessment Agent (Readiness Scan)
    ↓
Action Card Agent (Recommendation)
    ↓
Proposal/Proof/Payment (عبر البوابة الآمنة)
    ↓
Won Deal → Sales-to-Delivery Handoff
```

---

## 9. حالة عدم الرد

إذا العميل وافق على واتساب لكن ما رد:

```
📤 أرسلنا لك رسالة على واتساب
   تقدر ترد في أي وقت

🔄 إذا ما رديت خلال 48 ساعة
   نشوفك على البريد الإلكتروني

**ما أعرف — اقترح علي:**
أزرار:
- [رد الآن] → يفتح واتساب
- [أحسن] → يسجل وينتظر
```

---

## 10. رفض الاجتماع/واتساب

إذا العميل رفض كلا الخيارين:

```
فهمت! 😊

ما ضغط — نقدر نتواصل بالبريد.

📧 رسالتك راح تلقاها على البريد
   وأقدر أرد عليك هناك

**ما أعرف — اقترح علي:**

أزرار:
- [أرسل بريد] → يفتح البريد
- [أفضل أنتظر] → يسجل وينتظر
```
