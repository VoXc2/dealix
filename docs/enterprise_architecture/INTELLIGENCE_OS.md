# Intelligence OS — التعلم من التشغيل

## الهدف

جمع الأنماط من الأحداث والسجلات: إشارات مبيعات، إثبات، احتكاك، مخاطر، سوق، مالية → **قرارات** في الطبقات العليا.

## Signal → Decision (مرجعي)

| الإشارة | اتجاه القرار |
|---------|---------------|
| sales | عرض / توسعة |
| proof | retainer / تسويق متحفظ |
| friction | منتج |
| risk | حوكمة |
| market | موضعة |
| financial | تخصيص رأس مال |
