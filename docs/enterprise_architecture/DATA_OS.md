# Data OS — البيانات وجودتها

## الهدف

- **Source Passport** — من يملك البيانات، لأي استخدام، هل يوجد PII، الاحتفاظ.
- **جودة البيانات** — وضوح المصدر، اكتمال، ازدواجية، حقول ناقصة، اتساق التنسيق.
- **تصنيف PII** و**استخدام مسموح** و**سياسة احتفاظ**.

## عتبات جودة مقترحة (تشغيلية)

| النطاق | الإجراء |
|--------|---------|
| ≥ 85 | جاهز لسير عمل AI (مع الحوكمة) |
| 70–84 | تنظيف أولًا |
| 50–69 | تشخيص فقط |
| < 50 | سباق جاهزية بيانات |
