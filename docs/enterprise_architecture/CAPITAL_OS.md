# Capital OS — الأصول المتراكمة

## الهدف

تحويل التكرار إلى **أصول**: خدمة، منتج، معرفة، ثقة، سوق، معيار، شريك، منصة استثمارية (حسب نضج الشركة).

## أنواع أصول (مرجعي)

Service / Product / Knowledge / Trust / Market / Standard / Partner / Venture capital (معنى منظمي — ليس نصيحة استثمار).

## الحد الأدنى لمخرجات المشروع (سياسة)

- أصل ثقة واحد على الأقل
- أصل منتج أو معرفة واحد
- مسار توسعة واحد

## Productization Gate (مرجعي)

تكرار يدوي ≥ 3، تكلفة وقت ≥ ساعتين/مشروع، مرتبط بعرض مدفوع، يقلل مخاطر أو يحسن هامش، قابل للاختبار، قابل لإعادة الاستخدام.
