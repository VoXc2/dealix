# Revenue OS — أول wedge تجاري

## الهدف

تحويل رفع العميل لقائمة حسابات (CSV) + جوازات المصادر + حقول قطاع/مدينة + حالة العلاقة والموافقة إلى:

- درجة جودة بيانات
- حسابات منظّفة
- ترتيب Top N
- إجراءات تالية مقترحة
- حزمة مسودات (Draft Pack)
- خط أنابيب مصغّر
- حزمة إثبات إيرادات

## ممنوع

- محرك كشط (scraping engine)
- أتمتة واتساب باردة
- أتمتة لينكدإن
- ادّعاءات مبيعات مضمونة

## التنفيذ في الريبو

`auto_client_acquisition/revenue_os/` — يشمل `anti_waste.py`, `source_registry.py`, مسارات الـ API في `api/`.
