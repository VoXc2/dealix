# Proof OS — الإثبات

## الهدف

تحويل العمل إلى **حزمة إثبات** يمكن للعميل أو الإدارة مراجعتها — لا ادّعاء بلا دليل.

## Proof Pack (أقسام مرجعية)

ملخص تنفيذي، المشكلة، المدخلات، جوازات المصدر، العمل المنجز، المخرجات، درجات الجودة، قرارات الحوكمة، مخاطر محجوبة، مقاييس القيمة، القيود، الخطوة التالية، أصول رأسمالية مُنشأة.

## Proof Score (نطاقات مقترحة)

| النطاق | الاستخدام |
|--------|------------|
| 85–100 | مرشح لدراسة حالة (مع شروط إضافية) |
| 70–84 | دعم مبيعات |
| 55–69 | تعلم داخلي |
| < 55 | إثبات ضعيف |
