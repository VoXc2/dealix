# كيف نعمل — Dealix

نستخدم منهجية واضحة (**Dealix Method**): Discover → Diagnose → Design → Build → Validate → Deliver → Prove → Expand.

كل مشروع يشمل:

- نطاق واضح (`scope.md`)
- مراجعة QA (`qa_checklist.md`)
- فحص حوكمة (مسودات، منع إرسال تلقائي، PDPL حيث ينطبق)
- **Proof Pack** يثبت المدخلات والمعالجة والمخرجات والأثر
- خارطة للخطوة التالية (Pilot / Retainer / توسع)

التفاصيل التقنية للأمان والبيانات: [`data_handling.md`](data_handling.md)، [`ai_safety.md`](ai_safety.md)، [`delivery_process.md`](delivery_process.md).
