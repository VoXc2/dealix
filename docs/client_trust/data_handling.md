- لا نستخدم بيانات شخصية إلا بـ**أساس نظامي** واضح وموثّق حيث ينطبق PDPL.
- **مصدر كل جدول/ملف** يُسجَّل؛ بلا مصدر موثّق لا نبني ادعاءات تشغيلية حرجة.
- **PII:** كشف وتقليم أو عدم إرسال للنماذج وفق السياسة — [`../governance/PII_REDACTION_POLICY.md`](../governance/PII_REDACTION_POLICY.md).
- الاحتفاظ والحذف: [`../governance/DATA_RETENTION.md`](../governance/DATA_RETENTION.md).

النطاق التعاقدي يُحدَّد في `scope.md` لكل خدمة.
