# 06 — Reliability and Mission-Critical Program

## Objective

Move from stable operations to mission-critical reliability with explicit SLOs, drills, and closed-loop incident governance.

## Program pillars

1. SLO and error budget policy
2. Incident command model
3. Kill-switch and rollback drill program
4. Dependency resilience checks
5. Release reliability gates

## Reliability controls

- Service-level SLO ownership
- Error budget review in weekly operating cadence
- Drill scorecard with closure evidence
- Post-incident corrective action tracking
- Risk-based release freeze policy

## Drill catalog

- Control-plane rollback drill
- Runtime safety kill-switch drill
- Approval-center degradation drill
- Observability signal-loss drill
- Data pipeline integrity drill

## Exit criteria to mission-critical

- Drill pass rate at or above target for two consecutive cycles
- No unresolved critical post-incident actions
- SLO breaches tied to corrective plans with accepted risk decisions
