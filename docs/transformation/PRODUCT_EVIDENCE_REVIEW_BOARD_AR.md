# مجلس مراجعة أدلة المنتج (PERB) — أسبوعي

## الهدف

قرارات منتج مبنية على **أدلة** (L0–L5) وليس آراء.

## إيقاع

- **الأربعاء** — مراجعة 30–45 دقيقة
- المخرج يُلحق بـ Weekly Operating Proof Pack (قسم PERB)

## جدول أعمال

1. قرارات الأسبوع المقترحة (Go / No-Go / Defer)
2. مستوى الدليل لكل قرار
3. KPI متأثر + `source_ref`
4. تسجيل في `docs/transformation/evidence/perb_decisions.log`

## قواعد

- لا upsell خارجي تحت L4 (anti-waste)
- لا إجراء خارجي بدون Decision Passport
