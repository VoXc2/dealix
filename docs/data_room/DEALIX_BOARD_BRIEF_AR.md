# Dealix Board Brief

## القرار الحالي

البيع يبدأ من P1 فقط. لا نبيع P2/P3 إلا بعد Proof.

## مؤشرات النجاح

| Metric | Target |
|---|---:|
| Warm messages/day | 5 |
| Discovery calls/week | 3 |
| P1 proposals/week | 2 |
| Closed P1/week | 1 |
| First P2 conversion | 30–60 days |

## المخاطر

1. بناء features بدل البيع.
2. إرسال عشوائي يضر السمعة.
3. تخصيص مجاني.
4. وعود ROI بدون baseline.
5. تعدد عروض يشتت السوق.

## قرار الإدارة

كل أسبوع نراجع:
- أفضل niche
- أقوى objection
- عدد الرسائل
- عدد الردود
- عدد المكالمات
- عدد العروض
- cash collected
