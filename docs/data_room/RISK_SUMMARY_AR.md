# ملخص المخاطر (Risk Summary)

عرض صادق للمخاطر — مطلوب في أي مراجعة جادة.

| المخاطرة | الشدة | التخفيف |
|----------|------|---------|
| نقطة موافقة غير مُصادَقة (API) | عالية | إصلاح موصى به (auth) قبل الإنتاج |
| اعتماد على مؤسس واحد (سعة) | متوسطة | `SALES_CAPACITY_MODEL_AR.md`، تأهيل/أتمتة المسودات |
| سمعة النطاق عند الإرسال | عالية | بوابة قابلية تسليم + تصاعد + كبح |
| امتثال PDPL عند الإيراد | متوسطة | سياسات قائمة + تسجيل SDAIA عند الإيراد |
| إقامة بيانات/DPA مورّدين | متوسطة | تفضيل KSA + DPA (TBD) |
| حقن الأوامر (وكلاء) | متوسطة | مدخلات = بيانات؛ بوابات هيكلية |
| تكرار شجرة الحوكمة | منخفضة | توحيد (قرار مؤسس) |
| جذب غير مثبت بعد | — | TBD صريح، لا اختلاق |

> لا إخفاء للمخاطر. مستوى الأدلة لكل بند متاح في الوثائق المرتبطة.
