## Detection & Remediation

### Automated Detection
```python
from auto_client_acquisition.saudi_layer.forbidden_claims import (
    forbidden_arabic_claim_detected, safe_alternative
)

# Simple check
is_flagged = forbidden_arabic_claim_detected("نضمن لك نتائج مضمونة")  # True

# Detailed check
result = forbidden_arabic_claim_detected("نضمن لك أرباحاً مضمونة", detailed=True)

# Auto-remediate

```
