## Usage

```python
from auto_client_acquisition.saudi_layer.city_region_normalizer import (
    SAUDI_REGIONS, get_all_cities, normalize_city_name,
    get_region_for_city, is_valid_city, get_city_coordinates,
    get_cities_in_region
)

# Normalize a city name (Arabic or English)
normalize_city_name("جدة")  # "jeddah"
normalize_city_name("Jeddah")  # "jeddah"

# Check if valid
is_valid_city("الرياض")  # True
is_valid_city("New York")  # False

# Get region for a city
get_region_for_city("الدمام")  # "eastern"

# Get cities in a region
get_cities_in_region("makkah")
# Returns list of city dicts

# Get coordinates
get_city_coordinates("الرياض")  # (24.7136, 46.6753)

# Flat list of all cities
all_cities = get_all_cities()
```
