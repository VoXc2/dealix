## NAICS / ISIC Reference

- **NAICS**: North American Industry Classification System (6-digit codes)
- **ISIC**: International Standard Industrial Classification (4-digit codes)

## Usage

```python
from auto_client_acquisition.saudi_layer.saudi_sector_taxonomy import (
    SAUDI_SECTORS, get_all_sectors, sector_known, get_sub_sectors,
    get_sector_by_code, get_sector_naics, get_sector_isic,
    get_sector_names_ar, search_sectors,
)

# Check if sector exists
sector_known("technology")  # True

# Get sub-sectors
get_sub_sectors("healthcare")
# Returns: [{"name_ar": "المستشفيات", "name_en": "Hospitals"}, ...]

# Get NAICS code
get_sector_naics("finance")  # "522"

# Get ISIC code
get_sector_isic("construction")  # "410"

# Search
search_sectors("تقنية")  # matches technology sector
```
