# التموضع التنافسي — Dealix

## الخلاصة

Dealix لا يُباع كـ CRM؛ يُباع كـ **Revenue OS سعودي سياقي** مع **مشغّل عربي** وتنفيذ آمن.

المنصات العالمية تتحرك نحو **وكلاء AI داخل الـ CRM وسير العمل** (مثل Salesforce Agentforce مع التركيز على visibility/control، وHubSpot Breeze لإشارات الشراء والـ outreach). Dealix لا يفوز بمحاكاة سطح المكتب المؤسسي؛ يفوز كـ **طبقة تنفيذ إيرادات سعودية SMB-first**: عربي، واتساب-aware، موافقة أولاً، Proof Pack، وخدمة Pilot قبل الأتمتة الواسعة. التفصيل الكامل: [CATEGORY_DESIGN_AND_MASTER_OPERATING_SYSTEM_AR.md](strategic/CATEGORY_DESIGN_AND_MASTER_OPERATING_SYSTEM_AR.md) و [DEALIX_CATEGORY_EXECUTIVE_SUMMARY_EN.md](strategic/DEALIX_CATEGORY_EXECUTIVE_SUMMARY_EN.md).

## مصفوفة سريعة

| منافس | يفعلون جيداً | ضعف في السعودية/GCC | كيف يفوز Dealix |
|--------|---------------|---------------------|------------------|
| HubSpot | منصة واسعة، AI + context | ليس عربي-native كمشغل | موافقات + بريف عربي + PDPL story |
| Salesforce | عمق مؤسسات | تكلفة وتعقيد | سرعة المؤسس + سيناريوهات واتساب |
| Gong | ذكاء إيرادات وتوسع narrative | مركزية اتصال/إنجليزية | واتساب + إشارات + عربي |
| Apollo/ZoomInfo | بيانات واسعة | ثقافة بارد | تأهيل + قمع + موافقة |
| Zoho/Odoo | سعر وERP | ليس Revenue OS | طبقة إيرادات + proof |
| أدوات واتساب | وصول | بريد عشوائي | opt-in + أزرار + تدقيق |
| Boardy | UX قبول/تخطي | سياق سعودي/إيراد | ذاكرة + رادار + امتثال |
| SocraticCode-like | فهم كود | ليس GTM | مشروع + استراتيجية + تنفيذ |

## ما لا ننسخ

- إرسال تلقائي بارد.
- ادعاء «CRM كامل».
- مئة أداة AI بدون عقود.

## ما نستعيره

- سرد **context-rich AI** (اتجاه السوق).
- توسيع **Revenue OS** ليشمل نجاح العملاء والتوسع (فكرة Gong أوسع من المكالمات).
- **Workflows طويلة** عبر LangGraph لاحقاً عند استقرار النواة.

API: `GET /api/v1/business/competitors` و`GET /api/v1/business/differentiators`.
