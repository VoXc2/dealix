# Agent Transparency Card

Every output that touched an agent ships with its transparency card.

```json
{
  "agent": "RevenueAgent",
  "task": "rank_accounts",
  "autonomy_level": 2,
  "human_owner": "Dealix Revenue",
  "external_action_allowed": false,
  "approval_required": true,
  "audit_event": "AUD-003"
}
```

Typed: `client_os.agent_transparency.AgentTransparencyCard`.

The principle: clients trust agents that are named, scoped, and audited — not invisible ones.
