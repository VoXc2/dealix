# Client OS Modules

Seven panels.

1. Capability Dashboard — 7 axes × 0–5 maturity.
2. Data Readiness Panel — sources, passport status, quality, duplicates, missing, PII, allowed use.
3. Governance Panel — allow / draft / approval / redaction / blocked counts, audit events.
4. Workflow Panel — owner, inputs, AI step, approval path, QA, proof metric, cadence.
5. Draft Pack — drafts only; never auto-sent.
6. Proof Timeline — proof events, value events, blocked risks, before/after, confirmed vs estimated.
7. Next Actions — Continue / Expand / Pause linked to retainer or upsell.

Typed surfaces: `client_os.client_health_score`, `client_os.client_os_stage`, `client_os.expansion_signal`, `client_os.monthly_value_report`, `client_os.agent_transparency`.
