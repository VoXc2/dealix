# Monthly Value Report

8 sections, same shape every month.

1. What was done
2. What changed
3. What value was observed
4. What value is estimated
5. What risks were blocked
6. What needs approval
7. What capability improved
8. What should happen next

Typed: `client_os.monthly_value_report.MonthlyValueReport` + `render_monthly_value_report`.

The report converts the client's monthly investment into a visible operating return.
