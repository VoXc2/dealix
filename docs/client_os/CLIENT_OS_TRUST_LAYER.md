# Client OS Trust Layer

Each output exposes:

- source status
- governance status
- PII status
- approval status
- QA status
- proof linkage

## Example

```
Output: Draft email pack
Status: Draft-only
PII: Contains contact names
External sending: Not approved
Approval required: Yes
Audit event: AUD-002
```

## Rules

- No invisible agents.
- No unowned agents.
- No external action without approval.
- No agent output without audit.
