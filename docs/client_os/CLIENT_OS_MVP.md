# Client OS MVP

Start small.

## In MVP

Client profile, capability score, data readiness score, lead/workflow table, governance status, draft pack, proof pack, next actions.

## Not in MVP

SSO, complex RBAC, billing, marketplace, partner portal, deep integrations, white-label.

## Add later

Approval center, proof timeline, audit exports, role permissions, enterprise dashboard.

## Rule

Build only what the client pulls.
