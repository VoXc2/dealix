# Client OS Monetization

Three stages.

## Stage 1 — included with Retainer

Monthly Value Report, basic Proof Timeline, Draft Pack review.

## Stage 2 — Workspace fee

Client Workspace, Approval Center, Governance Log, Value Dashboard.

## Stage 3 — Enterprise fee

Audit exports, role permissions, AI Control Plane, executive dashboards.

## Rule

> Do not impose a platform fee before the client sees value from continuation.

The platform fee is what the client *asks* to keep, not what Dealix demands they pay.
