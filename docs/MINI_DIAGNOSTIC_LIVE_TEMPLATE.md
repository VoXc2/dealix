# Mini Diagnostic — Live Template

> Free deliverable. Sent within 24 hours of receiving the 10 intake
> answers. Single page, Arabic-first. NEVER claims guaranteed outcomes.

## Step 1 — collect intake (10 fields)

When a prospect says "yes, send the diagnostic", reply with:

```
ممتاز يا [الاسم]، أرسل لي ١٠ معلومات سريعة وأرجع لك خلال ٢٤ ساعة:

1. اسم الشركة
2. الموقع الإلكتروني
3. القطاع
4. المدينة الرئيسية / السوق
5. العرض الرئيسي في جملة (offer)
6. العميل المثالي (ICP)
7. متوسط قيمة الصفقة بالريال (إذا متاح)
8. القنوات الحالية (LinkedIn / إيميل / inbound / referrals / other)
9. أكبر تحدّي حالي
10. لو عندك قائمة، شو مصدرها وحالة الموافقة
```

Log every reply in `docs/FIRST_CUSTOMER_LIVE_BOARD.md`. Do NOT proceed
to the diagnostic until all 10 are answered.

## Hard rules for the diagnostic content

| Rule | Detail |
| --- | --- |
| 24-hour SLA | from receiving all 10 intake answers |
| 1 page max | A4 / single screen |
| Arabic first | unless prospect asked for English |
| Label estimates clearly | "تقدير" / "estimate" — never bare numbers |
| No "نضمن" / "guaranteed" | sweep your draft before sending |
| Recommend ONE safe channel | from {LinkedIn manual, Email draft with approval, inbound WhatsApp only} |
| Include ONE blocked risk | what NOT to do, with the reason |
| No live external action | the diagnostic is a document, not a campaign |

## How to write the 3 opportunities

Pick from intake field 6 (ICP) + field 7 (avg deal value) + field 8 (current channels).

- **Opportunity = a specific company name OR a tightly-defined segment.**
- Examples (good):
  - `Riyadh-based B2B SaaS with 50-200 employees and a sales team of 3-10`
  - `Healthcare clinics in Jeddah using HubSpot or Zoho`
  - `Real estate developers in Eastern Province with active inbound leads`
- Examples (bad — too vague, will look like spam):
  - `B2B companies in Saudi Arabia` ← way too broad
  - `Anyone with a budget` ← meaningless

## How to write the 1 Arabic message draft

Use the intake's "offer" + "ICP" + "current challenge" fields.
Format: 4-5 lines, first-person, no claims.

```
السلام عليكم [contact],
شفت أن [company] تشتغلون في [their sector / problem area].
عند عميلي [your prospect's company] حل بالعربي يحل [the specific
problem], بدون cold WhatsApp وبموافقة كاملة.
أقدر أرسل لك عينة قصيرة (٥ دقائق قراءة) لو تحب.
```

Founder fills `[...]` per prospect.

## Definition of done

The Mini Diagnostic is "delivered" when:

1. The 7-section Arabic doc is written and reviewed.
2. The doc is sent via LinkedIn DM (or email if prospect asked) — manually.
3. The board row is updated to `diagnostic_sent`.
4. A 24-hour follow-up reminder is set.

Move to `docs/PILOT_499_OFFER.md` for the next step if the prospect
expresses interest after reading the diagnostic.
