# Dealix Governance Stack — سبع طبقات

Dealix طبقة **تحكم وتشغيل** لا «أداة AI» فقط:

```text
AI Model
→ LLM Gateway
→ Agent Control
→ Governance Runtime
→ Workflow Owner
→ QA
→ Approval
→ Audit
→ Proof Pack
→ Operating Cadence
```

## 1. Source Governance
جواز كل مصدر — **Source Passport** (نوع، مالك، allowed use، PII، حساسية، احتفاظ، AI access، external use).

## 2. Data Governance
Data Quality Score · PII · dedup · allowed use · retention.

## 3. Model Governance
LLM Gateway · Model Router · Prompt Registry · Output Schema · Cost Guard · Eval · **AI Run Ledger**.

## 4. Agent Governance
هوية · مالك · غرض · أدوات مسموحة · ممنوعات · مستوى استقلالية · موافقة · تدقيق.

## 5. Workflow Governance
مالك بشري · مدخلات/مخرجات · مسار موافقة · rubric QA · مقياس proof · cadence.

## 6. Output Governance
قرارات: ALLOW · ALLOW_WITH_REVIEW · DRAFT_ONLY · REQUIRE_APPROVAL · REDACT · BLOCK · ESCALATE.

## 7. Proof Governance
**No proof, no claim / public case / retainer push قوي** — كل claim له proof وحدود (limitations).
