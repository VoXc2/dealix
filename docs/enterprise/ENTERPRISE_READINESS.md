# Enterprise Readiness

**لا تُباع Enterprise** قبل تحقيق حد أدنى تشغيلي وثقة — حتى لو صممت المعيار مبكراً.

## شروط البيع Enterprise

```text
10+ paid projects
3+ retainers
zero critical governance incidents
client workspace MVP (أو مسار وصول عميل واضح)
audit logs policy implemented in operations
role permissions model documented
support process + SLA template
data handling policy signed path with customer
```

## ما يلي بعد التحقق

- Multi-tenant، RBAC، SSO، تصدير audit، retention، تكاملات CRM/ERP، تدريب إداري.
