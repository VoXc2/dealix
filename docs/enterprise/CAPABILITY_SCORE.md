# Capability Score — المنتج الاستراتيجي للتشخيص

**Dealix Capability Score** يقيس سبع قدرات (مرتبطة بـ **DCI** في الطبقة الذكية):

Revenue · Customer · Operations · Knowledge · Data · Governance · Reporting

## المخرجات للعميل (قالب)

- Current State  
- Target State  
- Transformation Gap  
- Recommended Sprint  
- Expected Proof  
- Retainer Path  

## مثال نصي مختصر

Revenue Capability: 1/5 · Data: 2/5 · Governance: 0/5  
**موصى به:** Revenue Intelligence Sprint + AI Governance Review (مع مراجعة intake الفعلية).
