# Strategic Asset Classes — أصول استراتيجية

لا تقس الشركة بالمال فقط — قسها بما **تراكمه**.

## الفئات السبع

1. **Software** — Core OS، modules، dashboards  
2. **Service** — SOPs، checklists، templates  
3. **Knowledge** — Playbooks، اعتراضات، benchmarks  
4. **Trust** — Proof packs، شهادات audit آمنة  
5. **Market** — محتوى، category POV، شراكات  
6. **Standards** — Dealix Method، شهادات، QA standards  
7. **Venture** — وحدات جاهزة للانفصال
