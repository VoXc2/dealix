# Partner Operating System — تشغيل الشركاء

الشركاء **يوسّعون** — وقد **يضرون** الثقة إن لم تُضبط الامتثال.

## الطبقات

Referral · Implementation · Certified · Strategic · White-label

## كل شريك يحتاج

- certification  
- QA compliance  
- governance compliance  
- proof pack usage  
- audit rights (حيث ينطبق العقد)  
- client feedback loop  

## القاعدة

**No partner can sell unsafe automation under Dealix brand.**  
(لا واتساب بارد · لا DMs تلقائية · لا ادعاءات بلا proof — انظر سياسات الثقة في الريبو.)
