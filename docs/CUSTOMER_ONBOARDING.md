# Dealix Customer Onboarding Guide

## Welcome to Dealix!

Thank you for choosing Dealix as your AI Operating System. This guide will help you get started and maximize value from day one.

---

## Week 1: Getting Started

### Day 1-2: Environment Setup
1. **Receive Access Credentials**
   - You'll receive an email with your login credentials
   - Bookmark your Dealix URL: `https://yourcompany.dealix.sa`

2. **Initial Login**
   - Navigate to your Dealix URL
   - Login with provided credentials
   - Change password on first login

3. **Profile Setup**
   - Complete your company profile
   - Add team members who need access
   - Set up notification preferences

### Day 3-4: Data Import
1. **Import Existing Data**
   - Upload your current prospect list (CSV/Excel)
   - Import existing deals and pipeline data
   - Map your current sales stages to Dealix stages

2. **WhatsApp Integration (Optional)**
   - Provide WhatsApp Business API credentials
   - We'll configure the integration
   - Test with draft messages first

### Day 5-7: First Operations
1. **Command Room Tour**
   - Review your pipeline
   - Process any draft messages
   - Set up your daily review routine

2. **Brain OS Setup**
   - Log your first strategic decision
   - Add current risks and opportunities
   - Set up weekly review cadence

---

## Week 2: Building Habits

### Daily Routine (15 minutes)
**Morning Command Room Review:**
1. Check pipeline changes
2. Review draft messages queue
3. Approve/reject/edit drafts
4. Check WhatsApp conversations
5. Note any new signals for Brain OS

### Weekly Routine (30 minutes)
**Brain OS Review:**
1. Review logged signals
2. Update decision statuses
3. Assess new risks/opportunities
4. Update action ownership
5. Generate weekly insights
