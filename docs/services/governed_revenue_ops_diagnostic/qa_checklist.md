# QA Checklist

## Data QA

- [ ] Every finding carries a source_ref
- [ ] Missing fields and duplicates reported
- [ ] PII flagged, never logged

## Analysis QA

- [ ] Workflow map matches the interview record
- [ ] Pipeline risks are evidence-backed
- [ ] Follow-up gaps tie to real stages

## Governance QA

- [ ] No guaranteed-outcome language
- [ ] No fake proof or un-sourced claims
- [ ] No external send proposed without an approval step

## Report QA

- [ ] Decision Passport complete
- [ ] Recommendation is decision-ready
- [ ] Bilingual disclaimer present
