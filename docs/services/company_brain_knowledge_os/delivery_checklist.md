# Delivery Checklist

- [ ] document request issued and source list confirmed with client
- [ ] access and permission map agreed (who can see what)
- [ ] documents ingested into staging knowledge base
- [ ] source indexing verified (each chunk traceable to a document)
- [ ] retrieval build complete with citations and confidence scores
- [ ] no-source-no-answer enforcement enabled and tested
- [ ] role-based access controls implemented and verified
- [ ] internal assistants configured (employee, sales, support)
- [ ] board and executive summary view configured with source tracking
- [ ] evaluation run completed and report drafted
- [ ] usage dashboard live
- [ ] internal QA passed (see `qa_checklist.md`)
- [ ] client review and sign-off
- [ ] handoff completed (see `handoff.md`)
- [ ] first monthly proof pack scheduled
