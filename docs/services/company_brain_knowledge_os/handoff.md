# Handoff

What is handed to the client at the end of setup:

- live knowledge base with the agreed document estate ingested
- internal assistants for employees, sales, and support
- board and executive summary view with source tracking
- role-based access control configuration and permission map
- usage dashboard with access instructions
- evaluation report covering citation accuracy and no-source-no-answer behaviour
- operating notes: how to add documents, how to manage roles, where audit logs live
- escalation and change-request path
- retainer plan: monthly proof pack, evaluation cadence, and support scope

Client owns the knowledge base and its contents. Dealix retains no client documents outside the agreed environment.
