# Scope

## Included

- document ingestion across approved internal sources
- knowledge base build with source indexing
- retrieval with inline citations and confidence scores
- role-based access controls and permission mapping
- internal assistants for employees, sales, and support
- board and executive summaries with source tracking
- no-source-no-answer enforcement (refuse rather than guess)
- evaluation report and usage dashboard
- monthly proof pack and retainer support

## Excluded

- live external sending or charging
- cold WhatsApp or automated outreach
- scraping or bulk data harvesting
- answers without a cited source
- fabricated or unverified proof
- production integrations without security review (change request)
- multi-region rollouts beyond the agreed scope (change request)
