# Lead Intelligence Sprint — Scope

## Included
- CSV / Excel import preview (validation + dedupe).
- Saudi entity normalization (AR/EN names, CR, VAT, phone, domain).
- ICP scoring run (explainable A/B/C/D bands).
- Top 50 ranked accounts.
- Top 10 next-best-actions with bilingual outreach drafts.
- Mini CRM board with stages.
- Bilingual executive report + 60-min handoff session.
- Proof pack (before/after metrics).

## Excluded (out of scope)
- Live sending of any outbound message.
- Web scraping / unauthorized data harvesting.
- LinkedIn automation.
- Cold WhatsApp.
- Custom CRM integration (separate engagement).
- Paid data-provider subscriptions.
- Guaranteed meetings or guaranteed revenue.

## Customer responsibilities
- Provide seed list / CRM export (Day 1).
- Confirm lawful basis / consent posture for the data.
- Review top-10 drafts within 48 hours of delivery.
- Attend the handoff session.

## Hard stops (we will pause delivery if encountered)
- No documented data source for > 20% of records.
- Customer requests delivery of any forbidden action (see `docs/governance/COMPLIANCE_PERIMETER.md`).
- Quality Score below 80 at any QA gate — improvement before resume.
