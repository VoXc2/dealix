# Scope

## Included

- CSV import
- basic data cleanup
- deduplication
- account scoring
- segmenting
- outreach drafts (draft-only)
- executive report

## Excluded

- WhatsApp sending
- LinkedIn automation
- scraping
- complex integrations
- manual sales execution on behalf of client

## Client responsibilities

- provide data source
- confirm allowed usage
- review drafts
- join review call
