# Delivery Checklist

## Step 1: Intake

- [ ] Client goals confirmed
- [ ] Data/files requested
- [ ] Scope approved

## Day 1–2 (within sprint)

- [ ] Data received
- [ ] Source confirmed
- [ ] Import preview
- [ ] Bad rows flagged
- [ ] Duplicate report

## Step 2: Build

- [ ] scoring + segmentation + top accounts
- [ ] outreach drafts (draft-only)
- [ ] mini CRM board

## Step 3: QA

- [ ] Business QA
- [ ] Data QA
- [ ] AI QA
- [ ] Compliance QA

## Step 4: Delivery

- [ ] executive report finalized
- [ ] proof pack generated
- [ ] handoff completed
- [ ] upsell presented
