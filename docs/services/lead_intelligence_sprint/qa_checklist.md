# QA Checklist

## Data QA

- [ ] All rows have source or are flagged
- [ ] Duplicates identified
- [ ] Missing fields reported
- [ ] PII flagged

## Scoring QA

- [ ] Every score has explanation
- [ ] Top 10 accounts make business sense
- [ ] Scores are deterministic

## Outreach QA

- [ ] No guaranteed claims
- [ ] No fake proof
- [ ] No cold WhatsApp
- [ ] Draft-only

## Report QA

- [ ] Clear executive summary
- [ ] Top actions included
- [ ] Risks included
- [ ] Upsell path included
