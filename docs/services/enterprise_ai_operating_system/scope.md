# Scope

## Included

- AI audit across in-scope departments
- Company Brain v1 build (knowledge base + citation layer)
- draft-first AI agents for Sales, Support, and Operations
- Executive Command Center dashboard
- approval workflow design + append-only audit trail
- integration contracts for CRM, WhatsApp, Email, and Drive
- ROI dashboard wired to the value ledger
- governance policy pack + risk register
- monthly Proof Pack and one optimization cycle

## Excluded

- automatic external send or charge — action mode is approval-required
- cold WhatsApp messaging, LinkedIn automation, scraping, or blast outreach
- fabricated proof or revenue figures
- production integrations without a completed security review
- net-new application development outside the agent layer
- 24/7 staffed operations (separate retainer tier)

## Boundaries

All hard gates apply: `no_live_send`, `no_live_charge`, `no_cold_whatsapp`, `no_linkedin_auto`, `no_scraping`, `no_fake_proof`, `no_fake_revenue`, `no_blast`.

Scope changes are handled via written change request before work begins.
