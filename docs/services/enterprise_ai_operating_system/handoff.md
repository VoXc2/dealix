# Handoff

Delivered at the close of the Scale Plan phase (weeks 11-12).

## What the client receives

- AI Opportunity Map + 30/60/90 roadmap (final version)
- Company Brain v1 with documented knowledge sources and refresh cadence
- Sales/Support/Operations AI agents (draft-first), with approval workflows
- Executive Command Center and ROI dashboard, wired to the value ledger
- append-only audit trail and access log
- governance policy pack + risk register
- first monthly Proof Pack (see [`proof_pack_template.md`](proof_pack_template.md))
- Scale Plan: prioritized next agents and rollout sequence

## Operating knowledge transferred

- how to run and review each agent draft
- who approves, and the escalation path
- where audit logs and citations live
- how to add or refresh Company Brain sources
- how to request changes or scope additions

## Ongoing

- monthly retainer covers the optimization cycle and Proof Pack
- next-phase work is scoped via written change request
