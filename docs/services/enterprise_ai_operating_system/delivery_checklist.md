# Delivery Checklist

Mapped to the 6 phases of the Enterprise AI Transformation Program. Each phase ends with a gate; do not advance until the gate is signed off.

## Phase 1 — AI Audit (weeks 1-2)

- [ ] stakeholders and in-scope departments confirmed
- [ ] process and decision inventory captured
- [ ] AI Opportunity Map drafted
- [ ] 30/60/90 roadmap drafted
- [ ] **Gate:** Opportunity Map + roadmap approved

## Phase 2 — Foundation (weeks 3-4)

- [ ] Company Brain v1 knowledge sources ingested
- [ ] citation layer verified on sample queries
- [ ] governance policy pack drafted
- [ ] risk register opened
- [ ] **Gate:** Company Brain v1 + governance pack approved

## Phase 3 — First Agents (weeks 5-6)

- [ ] Sales/Support/Operations agents built (draft-first only)
- [ ] approval workflow wired to each agent
- [ ] append-only audit trail verified
- [ ] **Gate:** agents produce reviewable drafts, no live send

## Phase 4 — Integrations (weeks 7-8)

- [ ] integration contracts defined (CRM/WhatsApp/Email/Drive)
- [ ] security review completed
- [ ] staging integration tested with approval gates
- [ ] **Gate:** integration contracts signed off

## Phase 5 — Executive Layer (weeks 9-10)

- [ ] Executive Command Center dashboard live
- [ ] ROI dashboard wired to the value ledger
- [ ] **Gate:** leadership accepts dashboards

## Phase 6 — Scale Plan (weeks 11-12)

- [ ] optimization cycle completed
- [ ] first monthly Proof Pack delivered
- [ ] Scale Plan and retainer scope agreed
- [ ] handoff completed (see [`handoff.md`](handoff.md))
