## Promise

Stand up an enterprise-wide AI operating layer: a Company Brain, draft-first AI agents for Sales/Support/Operations, an Executive Command Center, and a governed approval and audit trail — delivered as a structured 90-day program.

## Price

Setup SAR 75,000–250,000 + monthly retainer SAR 15,000–60,000 (custom; scoped per organization).

## Duration

90-day Enterprise AI Transformation Program — 6 phases, each ending with a gate. See [`../../27_delivery_playbooks/ENTERPRISE_AI_TRANSFORMATION_DELIVERY_PLAYBOOK.md`](../../27_delivery_playbooks/ENTERPRISE_AI_TRANSFORMATION_DELIVERY_PLAYBOOK.md).

## Best For

- mid-to-large organizations with multiple departments
- teams with repeatable, knowledge-heavy decisions
- leadership wanting one governed AI layer, not point tools

## Deliverables

- AI Opportunity Map + 30/60/90 roadmap
- Company Brain v1 (knowledge base with citations)
- Sales/Support/Operations AI agents (draft-first only)
- Executive Command Center dashboard
- Approval workflows + append-only audit trail
- CRM/WhatsApp/Email/Drive integration contracts
- ROI dashboard wired to the value ledger
- Governance policy pack + risk register
- Monthly Proof Pack + optimization cycle

## Not Included

- automatic external send or charge (action mode is approval-required)
- cold WhatsApp, LinkedIn automation, scraping, or blast outreach
- live data migration without security review

## Success metric

We commit to a 90-day transformation across 5 gated phases. If a phase gate is not met, we keep working that phase at no charge.
