# Data Readiness Assessment

## Promise

Assess whether your data is **ready for AI workflows** and identify gaps **before** expensive implementation.

**Context:** most organizations report **data not fully ready** for AI; preparation challenges include silos, integration difficulty, strategy, quality, and regulatory constraints ([Cloudera & HBR Analytic Services report](https://www.cloudera.com/about/news-and-blogs/press-releases/2026-03-05-only-7-percent-of-enterprises-say-their-data-is-completely-ready-for-ai-according-to-new-report-from-cloudera-and-harvard-business-review-analytic-services-reveals.html)).

## Deliverables

- Data **source inventory** + owners  
- Data **quality** / readiness **score** ([`scoring_model.md`](scoring_model.md))  
- PII / **sensitivity** flags  
- Integration / access **gaps**  
- Readiness **report** ([`report_template.md`](report_template.md))  
- **Recommended next sprint** (cleanup vs diagnostic vs targeted AI sprint)  

## Out of scope (defaults)

- Building production integrations unless separately scoped  
- Guaranteed business outcomes  

See [`offer.md`](offer.md), [`scope.md`](scope.md), [`data_request.md`](data_request.md).
