# Offer — Data Readiness Assessment

## What the client gets

- Inventory of **sources**, fields, and owners  
- **Readiness score** with explicit gaps (not vanity numbers)  
- **Risk** flags (PII, silos, missing lineage)  
- **Prioritized** remediation path → next sprint recommendation  

## Outcomes (KPI)

- One **primary** “data readiness” metric improved or honestly baselined  
- Clear **go/no-go** for a downstream AI sprint  

## Exclusions

- No covert data collection; **client-provided** exports only  
- No scraping without explicit permission  
- No guaranteed ROI claims
