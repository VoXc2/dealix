# Scope — Data Readiness Assessment

## In scope

- Review **provided** exports and documentation  
- Score per [`scoring_model.md`](scoring_model.md)  
- Workshop / interviews as scheduled  
- Written report + next-sprint recommendation  

## Out of scope

- Fixing all data issues in-repo (separate cleanup sprint)  
- Building pipelines without SOW  

## Inputs required

See [`data_request.md`](data_request.md).

## Timeline

Typical **3–8 business days** after full intake.

## Acceptance

Client receives report + summary readout; sign-off on **baseline** score and **top 5 gaps**.
