# Approval Matrix — Client AI Policy Pack

| Action | Examples | Default approver | Notes |
|---|---|---|---|
| Use new AI tool | first-time vendor | Security / IT owner | must be inventoried |
| External send from AI draft | email, WhatsApp | Line manager + policy owner | consent/context required |
| Upload customer data | CRM export snippets | Data owner | minimize + redact |
| Production workflow change | CRM stage updates | Ops owner | rollback plan |
| Public claim / marketing | numbers, promises | Exec / brand owner | claims QA |

Client should assign named owners and adjust rows to their org.
