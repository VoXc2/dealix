# QA Checklist

## Evidence QA

- [ ] Every claim cites a source_ref
- [ ] No claim exceeds the supplied evidence
- [ ] Estimated figures labelled estimated, not verified

## Doctrine QA

- [ ] No guaranteed-outcome language
- [ ] No fake proof or un-sourced claims
- [ ] No external-action recommendation without an approval step

## Memo QA

- [ ] Decisions are clearly stated
- [ ] Build / hold / kill calls are explicit
- [ ] Bilingual disclaimer present
