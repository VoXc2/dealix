# Intake Questions

## Audience

- Who reads this memo (board, founders, investors)?
- What decision are they being asked to make?

## Evidence

- Which prior outputs feed this memo (diagnostic, sprint, retainer)?
- Are there source_refs for each key claim?

## Scope

- Time horizon for the recommendations?
- Any decisions explicitly out of scope?

## Governance

- Confirm: every recommendation must cite a source_ref.
- Confirm: no guaranteed-outcome language.
