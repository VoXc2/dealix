# ROI Estimate

This program's value is framed as **risk reduced** and **safe AI adoption
enabled**, not revenue. All figures below are estimates, not promises.

## How Value Is Framed

- **Risk reduced** — uncontrolled AI use replaced with approval classes and
  documented data-handling rules.
- **Audit readiness** — audit logging and human-approval workflows make AI
  decisions reviewable.
- **PDPL readiness** — a gap assessment reduces the cost and time of later
  compliance work (readiness, not certification).
- **Safe adoption** — clear policy lets teams use AI confidently instead of
  pausing or hiding usage.

## Estimate Inputs

- number of departments using AI
- number of high-risk use cases identified
- estimated hours currently spent on ad-hoc AI review
- estimated cost of a governance gap if unaddressed

## What We Commit

We commit to delivering an auditable, PDPL-aligned governance framework. If it
does not pass the agreed readiness review, we continue at no charge. We do not
promise specific financial returns.
