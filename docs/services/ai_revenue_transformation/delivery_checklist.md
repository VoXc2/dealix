# Delivery Checklist

Step-by-step delivery for the 60-day build sprint.

## 1. ICP and discovery

- [ ] kickoff held; sales goals and KPIs agreed
- [ ] ICP analysis completed and signed off
- [ ] data sources mapped (client-provided, consented)

## 2. Lead machine design

- [ ] lead-machine design documented
- [ ] enrichment fields and consented sources confirmed
- [ ] governance gates reviewed (no scraping, no blast)

## 3. Lead scoring and enrichment

- [ ] scoring model built and tuned on client data
- [ ] enrichment model validated against a sample
- [ ] score thresholds agreed with sales lead

## 4. Outbound strategy and messaging

- [ ] outbound strategy documented
- [ ] Arabic sales messaging library drafted
- [ ] tone and voice approved by client

## 5. Follow-up workflows

- [ ] WhatsApp/Email follow-up workflows built (draft-first)
- [ ] approval points confirmed before any external send
- [ ] escalation and opt-out handling documented

## 6. CRM pipeline

- [ ] CRM pipeline stages built
- [ ] AI-assisted follow-up cadence configured
- [ ] sales team trained on the pipeline

## 7. Executive dashboard

- [ ] revenue dashboard built and connected
- [ ] KPI definitions confirmed with client

## 8. Proof

- [ ] Proof Pack assembled (see [`proof_pack_template.md`](proof_pack_template.md))
- [ ] QA checklist passed (see [`qa_checklist.md`](qa_checklist.md))
- [ ] handoff completed (see [`handoff.md`](handoff.md))
