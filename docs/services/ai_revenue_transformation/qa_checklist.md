# QA Checklist

Quality checks completed before client handoff.

## Messaging

- [ ] Arabic messaging reviewed for voice and tone
- [ ] no forbidden terms in client-facing copy (e.g. guaranteed / نضمن)
- [ ] messaging matches the client's brand and approval

## Governance

- [ ] no scraping used in any data source
- [ ] no cold WhatsApp, no LinkedIn automation, no bulk blast
- [ ] all external-facing workflows are draft-first verified
- [ ] approval points present before every external send
- [ ] opt-out and escalation handling documented
- [ ] no live charge configured without approval

## Data and model

- [ ] PII handling documented
- [ ] scoring model validated against a sample
- [ ] enrichment uses consented sources only

## System

- [ ] CRM pipeline stages match agreed definitions
- [ ] dashboard KPIs match agreed definitions
- [ ] rollback path documented

## Proof

- [ ] Proof Pack figures traceable to source data (no fake proof, no fake revenue)
- [ ] success metric and pipeline KPIs agreed in writing
