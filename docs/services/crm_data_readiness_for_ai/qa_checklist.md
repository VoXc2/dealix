# QA Checklist

## Data QA

- [ ] Record counts reconcile with the export
- [ ] Duplicate logic is explained and deterministic
- [ ] Missing-field analysis is complete

## Scoring QA

- [ ] Readiness score components are explained
- [ ] Score is deterministic and reproducible

## Governance QA

- [ ] PII flagged, never logged
- [ ] No guaranteed-outcome language
- [ ] No fake proof or un-sourced claims

## Report QA

- [ ] AI readiness recommendation is decision-ready
- [ ] Bilingual disclaimer present
