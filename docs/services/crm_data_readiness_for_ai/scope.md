# Scope

## Included

- CRM / data export intake
- CRM hygiene review
- Source mapping and coverage check
- Missing-fields and duplicate-accounts analysis
- Lifecycle-stage review
- Data readiness score and AI readiness recommendation

## Excluded

- CRM migration or platform rebuild
- Bulk automated data cleaning without review
- AI workflow build (that is a Sprint or Retainer)

## Client responsibilities

- Provide a CRM / data export
- Confirm lawful basis for any contact data
- Join the review call
