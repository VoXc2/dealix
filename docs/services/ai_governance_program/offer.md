# AI Governance Program

## Promise

Create a practical AI governance system for safe, auditable, and approved AI use.

## Price

Scope-dependent (indicative ranges in [`../../company/SERVICE_CATALOG_V1.md`](../../company/SERVICE_CATALOG_V1.md) / proposal).

## Duration

3–6 weeks (typical)

## Best For

Organizations scaling AI use across teams without clear governance, inventory, or incident expectations.

## Deliverables

- AI use inventory
- risk register
- allowed/prohibited use policy
- approval matrix
- data handling rules
- incident response process
- employee guide
- governance report

## Outputs

- policy document
- control matrix
- approval workflow
- audit process
- training guide

## Not included

- external legal sign-off (counsel)
- full penetration test
- building all technical controls in-product (unless separately scoped)

## Success metric

Signed-off policy pack + completed inventories + recorded approval matrix + incident path understood.
