# AI Quick Win Sprint — Scope

## Included
- 30-min discovery + use-case selection from the curated list (Day 1).
- Build the automation in the customer's environment.
- Approval workflow + immutable audit log baked in.
- Two feedback rounds (Day 3 + Day 5).
- Runbook ≥ 3 pages + 1-hour recorded training session.
- ROI baseline measurement.

## Excluded
- More than one use case.
- Custom integrations outside the curated list.
- More than two training sessions.
- Autonomous external actions without human approval.

## Customer responsibilities
- Provide process owner + 30-min discovery slot.
- Sample inputs/outputs (5 examples).
- System access via sealed credentials vault.
- Review of automation outputs in feedback rounds.

## Hard stops
- Use case not in curated list (route to Workflow Automation Sprint).
- Any automation that would execute outbound communication without approval.
