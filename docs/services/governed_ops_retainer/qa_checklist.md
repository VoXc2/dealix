# QA Checklist (monthly)

## Data QA

- [ ] All rows have a source or are flagged
- [ ] PII flagged, never logged

## Review QA

- [ ] Revenue and pipeline figures tie to source data
- [ ] Risk register changes are evidence-backed
- [ ] Decision review cites source_refs

## Queue QA

- [ ] No guaranteed claims
- [ ] No cold WhatsApp / LinkedIn automation
- [ ] Follow-up queue is draft-only and approval-gated

## Report QA

- [ ] Value report uses estimated (not verified) language
- [ ] Board memo is decision-ready
- [ ] Bilingual disclaimer present
