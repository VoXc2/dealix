# Data Request (monthly)

## Required each cycle

- Updated CRM / pipeline export
- account_name, current_stage, source, last_contacted_at
- Closed-won / closed-lost since last cycle

## Optional

- deal_value
- next_step
- notes

## Rules

- Every row must carry a source.
- Personal contact data must have a lawful basis (PDPL).
- PII is flagged, never logged.
