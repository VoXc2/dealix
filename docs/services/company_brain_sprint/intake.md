# Company Brain Sprint — Intake

## Day-1 Discovery (45 min)

1. Which team is this assistant for?
2. How many people will use it?
3. What document types do you have? (PDF / DOCX / KB / policy)
4. Roughly how many documents per category?
5. Which documents contain sensitive data? (HR / finance / customer PII)
6. What questions do employees ask most often today?
7. What access tiers do you need? (default: Admin / Team / Read-only)
8. Who owns each document category? (we need an SME contact per category)

## Required from customer (Days 1–3)

- [ ] Document inventory list (file names + categories).
- [ ] Permissions matrix.
- [ ] 10 sample questions.
- [ ] Sensitive-doc flags.

## Compliance acknowledgements
- [ ] PDPL Art. 5 lawful basis (contract).
- [ ] PII redaction policy agreed.
- [ ] "No source = no answer" policy understood and accepted.
- [ ] Right-to-erasure SLA accepted (< 72 hours).
