# Adoption → Retainer Logic

لا تعرض retainer لأن المشروع انتهى فقط — اعرضه عندما **adoption** واضح.

## جاهز عندما

workflow owner · مخرجات مستخدمة · مسار موافقة يعمل · proof score ≥ 80 · العميل يطلب استمرارًا · قيمة شهرية واضحة · مخاطر حوكمة مضبوطة.

## غير جاهز عندما

مخرجات غير مستخدمة · owner غير واضح · بيانات ضعيفة · approvals معطلة · قيمة غير مرئية.
