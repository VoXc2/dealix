# Capital Creation — بناء رأس المال في كل مشروع

## أنواع الأصول (أمثلة)

| النوع | مثال |
|-------|--------|
| Service | checklist أقوى |
| Product | وحدة قابلة لإعادة الاستخدام |
| Knowledge | رؤية قطاعية آمنة |
| Trust | proof pack |
| Market | محتوى عام آمن |
| Partner | قناة إحالة |
| Standard | تحسين منهجية |

## الحد الأدنى لمشروع «استراتيجي»

**1 Trust Asset** + **1 Product أو Knowledge Asset** + **1 Expansion Path**

إلا فالمشروع **cash-only** وليس تراكمًا استراتيجيًا.
