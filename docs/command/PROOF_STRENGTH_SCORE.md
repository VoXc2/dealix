## المكوّنات

| المكوّن | الوزن |
|---------|------|
| Metric clarity | 20% |
| Evidence quality | 20% |
| Before/after comparison | 15% |
| Client relevance | 15% |
| Governance confidence | 15% |
| Retainer linkage | 15% |

## الاستخدام

| نطاق النتيجة | الاستخدام |
|--------------|-----------|
| 85+ | case candidate |
| 70–84 | sales support |
| &lt; 70 | internal learning only |
