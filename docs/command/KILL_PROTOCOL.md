# Kill Protocol — بروتوكول الإيقاف

## Kill Service

win rate منخفض · proof ضعيف · scope creep عالٍ · مخاطر حوكمة عالية · لا مسار retainer · لا تكرار · هامش منخفض — انظر `command_os/kill_criteria.py` (`kill_service_recommended`).

## Kill Feature

لا يُعاد استخدامه · غير مربوط بالإيراد · عبء صيانة · لا يقلل وقت التسليم (`reduces_delivery_effort=False`).

## Kill Market

مشترون غير واضحين · ميزانية ضعيفة · دورة مبيعات طويلة · بيانات خطرة · لا مسار proof.

## Kill Partner

يبيع **وعودًا غير آمنة** · يتجاوز الحوكمة · يضر العلامة · لا يلتزم QA — `kill_partner_recommended` في `command_os/kill_protocol.py`.
