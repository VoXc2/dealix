# First Customer Outreach Messages

> Manual warm LinkedIn DMs only. Founder sends from own account.
> No automation, no scraping, no cold WhatsApp, no email blast.

## Arabic — first warm LinkedIn DM

```
السلام عليكم [الاسم]،

أطلقت Dealix كـ Beta محدودة للشركات والوكالات السعودية.

الفكرة:
نساعدكم ترتبون النمو بشكل عملي:
فرص مناسبة، رسائل عربية، متابعة، وقنوات آمنة — وكل شيء يمر بموافقتكم.

أقدر أجهز لكم Mini Diagnostic مجاني:
3 فرص + رسالة عربية + توصية قناة + مخاطرة لازم تتجنبونها.

أرسله لك؟
```

## Forbidden phrases (must NEVER appear in any outbound)

- نضمن مبيعات
- نجيب لك عملاء مضمونين
- نرسل واتساب لكل الأرقام
- automation يرسل بدالك
- guaranteed revenue / guaranteed leads / guaranteed sales / guaranteed pipeline
- blast WhatsApp / cold WhatsApp / bulk WhatsApp
- scrape leads / scrape phones / scrape LinkedIn
- "we'll handle it for you" with the implication of automated send

## Style rules

| Rule | Why |
| --- | --- |
| Max 5 lines on the first DM | LinkedIn DMs over 5 lines have lower reply rates and look spammy |
| First-person, founder voice | "I'm launching ..." not "We at Dealix..." — warm, human |
| One CTA only ("Would you like me to send one?") | More than one ask kills conversion |
| Always "Mini Diagnostic مجاني" first | Free Diagnostic is the wedge; never lead with the 499 pilot |
| Never mention pricing in the first DM | Pricing comes after Mini Diagnostic delivered |
| Never claim outcomes | Use "نساعدكم ترتبون النمو" not "we'll grow your revenue 30%" |
| 24h before any follow-up | Earlier = pressure |
| Stop after 3 follow-ups, max | Then `nurture` status |

## What "OK send me the diagnostic" looks like

When a prospect replies positively, switch to `docs/MINI_DIAGNOSTIC_LIVE_TEMPLATE.md`
to collect the 10 intake fields, then deliver the Mini Diagnostic
within 24 hours.

## What "we already use X" looks like

That's a good outcome. Reply with:

```
ممتاز. لو حابب يوم تحب نشوف هل Dealix يضيف فوق X (مو يستبدله)،
أرسل لي عينة Mini Diagnostic بدون أي التزام. خذ وقتك.
```

Then move them to `nurture`. Don't push.
