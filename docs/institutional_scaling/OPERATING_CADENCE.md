# Institutional Operating Cadence

## 1. Daily

- Top revenue opportunity?
- Top delivery risk?
- Top governance risk?
- Top proof to create?

## 2. Weekly

- Sold / delivered / proved / learned / productize / retainer-ready.

## 3. Monthly

- Strongest offer / weakest offer / maturing unit / recurring risk / accumulated asset / service to kill.

## 4. Quarterly

- Closer to holding / MRR up / market language adoption / partner channel / venture candidate.

## 5. Discipline

- Each cadence has named owners.
- Each cadence ends in entries in the Decision Log.
- Skipped cadence is an organizational debt logged for the next review.

## 6. The principle

> Cadence is the operating heartbeat of an institutional firm.
