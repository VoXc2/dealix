# Enterprise Trust Engine

To serve enterprise, Dealix needs a **Trust Engine** — not just a privacy policy.

## 1. Components

- Source Passport
- Data Quality Score
- PII Detection
- Allowed Use Registry
- LLM Gateway
- Agent Control Plane
- Governance Runtime
- Approval Engine
- Audit Trail
- Proof Pack
- Incident Response

## 2. The six questions every output must answer

- What is the data source?
- Is PII present?
- Is the use allowed?
- Which agent or model was used?
- Is the output a draft or final?
- Who approved? What is the evidence? What are the risks?

## 3. The principle

> Enterprise sells trust. Trust is operated through the Engine.
