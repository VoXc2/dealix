# القاعدة المؤسسية — Institutional Rule

```text
Scale only what is:
proven,
governed,
repeatable,
profitable,
and owned.
```

بالعربي:

```text
لا توسع إلا ما كان:
مثبتًا، محكومًا، متكررًا، مربحًا، وله مالك واضح.
```

## لا توسع

خدمة بلا proof · feature بلا usage · partner بلا QA · agent بلا owner · workflow بلا approval · سوق بلا buyer واضح.

## وسّع

خدمة لها proof · retainer له value cadence · feature تقلّل وقت التسليم · partner يلتزم بالمنهجية · unit لها KPIs.
