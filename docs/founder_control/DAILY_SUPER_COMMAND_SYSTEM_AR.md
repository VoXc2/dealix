# نظام أمر المؤسس اليومي (Daily Super Command)

تقرير واحد يجيب كل صباح على أربعة أسئلة:

```
1) ماذا ترسل اليوم؟
2) من تتصل به؟
3) ماذا تعرض؟
4) ماذا تسلّم؟
```

## المخرج

`reports/founder/DAILY_SUPER_COMMAND.md` — مُولّد عبر
`python dealix.py founder-command` (أو ضمن `seed`).

## المحتوى

- **أعلى 5 مسودات بريد** للإرسال (بعد الاعتماد).
- **أعلى 5 مكالمات** مع Call Briefs جاهزة.
- **أعلى 3 عروض مصغّرة** للاعتماد.
- **معوّقات التسليم**: العملاء العالقون بانتظار مدخلات.
- **بوابات الاعتماد** المطلوبة من المؤسس.

## بوابات قرار المؤسس (إلزامية)

| الإجراء | يحتاج اعتماد؟ |
| --- | :---: |
| إرسال أي بريد | نعم |
| اعتماد Mini Proposal | نعم |
| تغيير تسعير | نعم |
| بدء أي تسليم | نعم (بعد اكتمال المدخلات) |
| اتصال / واتساب | يدوي دائمًا (لا أتمتة) |

## القاعدة

```
الوكيل يقترح. المؤسس يقرر. النظام يسجّل. لا فعل خارجي بلا اعتماد.
```
