# Dealix AI Inventory

المؤسسات لا تحكم ما لا تراه. **AI Inventory** يجعل use cases الوكلاء والنماذج والمصادر مرئية وقابلة للحوكمة.

## ماذا يحتوي السجل؟

لكل بند (سطر) في الجرد:

- use case  
- department  
- owner  
- data sources  
- agent/model used  
- risk level  
- approval path  
- audit status  
- proof metric  
- status (active / pilot / sunset)  

## العرض التجاري

**AI Inventory & Governance Map**

### مخرجات العرض

- قائمة use cases  
- تصنيف مخاطر  
- خريطة مصادر البيانات  
- فجوات موافقة  
- فجوات proof  
- خطة حوكمة موصى بها
