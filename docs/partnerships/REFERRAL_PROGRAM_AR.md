# برنامج الإحالات — Referral Program

## مقدمة

برنامج الإحالات في Dealix هو الطريقة الأساسية للشركاء لكسب العمولات من خلال إحالة عملاء جدد.

---

## نظرة عامة

| البند | التفاصيل |
|-------|----------|
| **الاسم** | Dealix Partner Referral Program |
| **البداية** | [التاريخ] |
| **العمولة** | 10-15% من قيمة الصفقة |
| **توقيت الدفع** | عند إغلاق الصفقة |
| **مدة الصلاحية** | 90 يوم من الإحالة |

---

## كيف يعمل البرنامج

### خطوات البرنامج

```
┌─────────────────┐
│  1. التقديم     │  ← الشريك يملأ نموذج الإحالة
└────────┬────────┘
         ▼
┌─────────────────┐
│  2. التأهيل     │  ← Dealix تتحقق من العميل
└────────┬────────┘
         ▼
┌─────────────────┐
│  3. البيع       │  ← Dealix تبيع للعميل
└────────┬────────┘
         ▼
┌─────────────────┐
│  4. الإغلاق     │  ← تأكيد الصفقة المغلقة
└────────┬────────┘
         ▼
┌─────────────────┐
│  5. العمولة     │  ← الدفع للشريك
└─────────────────┘
```

### أنواع الإحالات

| النوع | الوصف | العمولة |
|-------|-------|---------|
| **Warm Referral** | عميل تواصل معه الشريك | 10% |
| **Qualified Lead** | عميل مستوفي المعايير | 12% |
| **Premium Referral** | عميل مع حضور الشريك | 15% |

## معايير التأهيل

### معايير العميل المقبول

**✅ يُقبل:**
- شركة جديدة على Dealix
- حجم مناسب (5-100 موظف)
- في السوق المستهدف
-Budget موجود
- صانع قرار متاح

**❌ لا يُقبل:**
- عميل موجود في Dealix pipeline
- عميل تواصل مع Dealix خلال 90 يوم السابقة
- عميل في قطاع غير مستهدف
- منافس مباشر لـ Dealix

---

## العمولة والتسعير

### حساب العمولة

| حجم الصفقة | نسبة العمولة |
|------------|-------------|
| حتى 10,000 | 10% |
| 10,001 - 50,000 | 12% |
| أكثر من 50,000 | 15% |

### مثال على الحساب

```
صفقة بقيمة: 25,000 ريال
نسبة العمولة: 12%
العمولة: 25,000 × 12% = 3,000 ريال
```

### القيود

- العمولة الأدنى: 500 ريال
- العمولة الأقصى: 15,000 ريال (للصفقة الواحدة)
- لا يوجد سقف للإحالات

## شروط وأحكام

### الشروط العامة

1. **ملكية الإحالة**
   - Dealix تحتفظ بالحق في تحديد ملكية العميل
   - في حالات النزاع، سجل HubSpot هو المرجع

2. **السرية**
   - جميع معلومات العميل سرية
   - لا يجوز مشاركة المعلومات مع أطراف ثالثة

3. **السلوك**
   - لا ضغط على العملاء المحالين
   - لا وعود غير مصرح بها
   - احترام خصوصية العملاء

4. **الإنهاء**
   - يمكن إنهاء البرنامج بإشعار 30 يوم
   - يمكن لـ Dealix إنهاء البرنامج فوراً عند المخالفة

### استثناءات

| السيناريو | النتيجة |
|-----------|--------|
| صفقة ملغاة بعد الإغلاق | لا عمولة أو استرداد |
| عميل غير مؤهل بعد المراجعة | لا عمولة |
| انتهاك الشروط | إلغاء العمولات |
| إحالة مضاعفة | العمولة للشريك الأول |

## الأسئلة الشائعة

**س: متى أستحق العمولة؟**
ج: عند إغلاق صفقة مدفوعة مع عميل أحلته.

**س: هل يمكنني إحالة شركة موجودة في Dealix؟**
ج: لا، إذا كانت في pipeline Dealix خلال 90 يوم السابقة.

**س: ماذا لو أحلت نفس العميل لشريك آخر؟**
ج: الشريك الأول في السجل يحصد العمولة.

**س: هل العمولة تشمل التجديدات؟**
ج: لا، العمولة للصفقة الأولى فقط.

**س: ماذا لو ألغى العميل العقد؟**
ج: إذا حدث الإلغاء خلال 90 يوم، تُسترد العمولة.

**س: كيف أتتبع حالة إحالاتي؟**
ج: عبر لوحة تحكم الشركاء أو التواصل مع مدير الشراكات.
