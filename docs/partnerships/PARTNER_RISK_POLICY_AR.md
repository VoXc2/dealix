# سياسة مخاطر الشركاء — Partner Risk Policy

## مقدمة

سياسة مخاطر الشركاء تحدد كيفية تحديد وتقييم وإدارة المخاطر المرتبطة بشراكات Dealix.

---

## إطار إدارة المخاطر

```
┌─────────────────┐
│  التحديد         │  ← تحديد المخاطر المحتملة
└────────┬────────┘
         ▼
┌─────────────────┐
│  التقييم         │  ← تحليل الاحتمالية والتأثير
└────────┬────────┘
         ▼
┌─────────────────┐
│  التخطيط         │  ← وضع استراتيجيات التخفيف
└────────┬────────┘
         ▼
┌─────────────────┐
│  المراقبة        │  ← تتبع مؤشرات التحذير
└────────┬────────┘
         ▼
┌─────────────────┐
│  التصعيد         │  ← إجراءات التصعيد عند الحاجة
└─────────────────┘
```

## تصنيف المخاطر

### 1. مخاطر مالية

| المخاطر | الوصف | الاحتمالية | التأثير |
|--------|-------|-----------|--------|
| عدم تحصيل العمولات | الشريك لا يدفع العمولات المستحقة | منخفض | عالي |
| مطالبات ضخمة | مطالبة بـ sum كبير | منخفض | عالي |
| إرتفاع churn | عملاء الشريك يلغون بسرعة | متوسط | عالي |
| اتفاقيات غير مصرح بها | الشريك يوقع اتفاقيات باسم Dealix | منخفض | عالي |

### 2. مخاطر التشغيل

| المخاطر | الوصف | الاحتمالية | التأثير |
|--------|-------|-----------|--------|
| جودة leads سيئة | الشريك يحيل عملاء غير مؤهلين | متوسط | متوسط |
| ضعف engagement | الشريك لا يشارك بنشاط | متوسط | منخفض |
| تأخر التسليم | تأخير في تقديم الخدمة | منخفض | متوسط |
| فقدان البيانات | فقدان بيانات العملاء | منخفض | عالي |

### 3. مخاطر السمعة

| المخاطر | الوصف | الاحتمالية | التأثير |
|--------|-------|-----------|--------|
| سلوك غير لائق | الشريك يتصرف بشكل سيء | منخفض | عالي |
| تمثيل خاطئ | الشريك يبالغ في وصف Dealix | متوسط | عالي |
| محتوى مضلل | مواد تسويقية غير دقيقة | منخفض | عالي |
| تجارب سيئة | عملاء الشريك يشكون | متوسط | عالي |

### 4. مخاطر استراتيجية

| المخاطر | الوصف | الاحتمالية | التأثير |
|--------|-------|-----------|--------|
| اعتماد زائد | reliance كبير على شريك واحد | منخفض | عالي |
| منافسة | الشريك يبيع حلول منافسة | منخفض | عالي |
| انسحاب | انسحاب مفاجئ للشريك | منخفض | عالي |

## مؤشرات التحذير المبكر

### ⚠️ Yellow Flags (تحذيرات صفراء)

| المؤشر | الوصف | الإجراء |
|--------|-------|--------|
| انخفاض الإحالات | انخفاض 50%+ لمدة شهرين | مراجعة + تواصل |
| بطء الاستجابة | عدم الرد لأكثر من 7 أيام | إنذار |
| شكاوى العملاء | 1-2 شكاوى من عملاء الشريك | تحقيق + تدخل |
| مخالفة بسيطة | انتهاك بسيط للسياسات | توجيه + توثيق |
| تغيير القيادة | تغيير في الإدارة العليا للشريك | تقييم جديد |

### 🔴 Red Flags (تحذيرات حمراء)

| المؤشر | الوصف | الإجراء |
|--------|-------|--------|
| غياب مستمر | عدم الرد لأكثر من 30 يوم | إنهاء محتمل |
| مخالفات متكررة | انتهاكات متعددة للسياسات | تعليق + تصعيد |
| شكاوى خطيرة | شكاوى خطيرة من العملاء | تعليق فوري |
| مشاكل قانونية | مشاكل قانونية للشريك | تعليق + مراجعة |
| أداء سيء | 0 إحالة لمدة 6 أشهر | تقييم + قرار |

## استراتيجيات التخفيف

### للمخاطر المالية

| المخاطر | استراتيجية التخفيف |
|--------|-------------------|
| عدم تحصيل العمولات | شروط دفع واضحة + ضمانات |
| مطالبات ضخمة | حدود المسؤولية في العقد |
| إرتفاع churn | follow-up aggressive مع العملاء |
| اتفاقيات غير مصرح بها | تدريب صارم + consequences |

### للمخاطر التشغيل

| المخاطر | استراتيجية التخفيف |
|--------|-------------------|
| جودة leads سيئة | معايير تأهيل صارمة + feedback |
| ضعف engagement | برامج تحفيز + follow-up |
| تأخر التسليم | SLAs واضحة + عقوبات |
| فقدان البيانات | نسخ احتياطي + استعادة |

### للمخاطر السمعة

| المخاطر | استراتيجية التخفيف |
|--------|-------------------|
| سلوك غير لائق | قواعد سلوك + consequences |
| تمثيل خاطئ | تدريب على messaging + مراجعة |
| محتوى مضلل | موافقة مسبقة على المحتوى |
| تجارب سيئة | متابعة + استجابة سريعة |

### للمخاطر الاستراتيجية

| المخاطر | استراتيجية التخفيف |
|--------|-------------------|
| اعتماد زائد | تنويع الشركاء |
| منافسة | بنود عدم المنافسة |
| انسحاب | خطط طوارئ + توثيق |

## مستويات التصعيد

### المستوى 1: مراقبة

| الحالة | الشريك لا يفي بهدف واحد |
|--------|------------------------|
| الإجراء | تواصل + تحليل |
| المسؤول | مدير الشراكات |
| المدة | 30 يوم |

### المستوى 2: تدخل

| الحالة | فشل في تلبية الهدف لثلث |
|--------|------------------------|
| الإجراء | خطة تحسين + follow-up صارم |
| المسؤول | مدير الشراكات + مدير المبيعات |
| المدة | 60 يوم |

### المستوى 3: تعليق

| الحالة | مخالفات خطيرة أو red flags |
|--------|------------------------|
| الإجراء | تعليق مؤقت + مراجعة |
| المسؤول | مدير الشراكات + المؤسسين |
| المدة | 30-90 يوم |

### المستوى 4: إنهاء

| الحالة | فشل في التصحيح + red flags خطيرة |
|--------|------------------------|
| الإجراء | إنهاء العقد + نقل العملاء |
| المسؤول | المؤسسين |
| المدة | 30-60 يوم |

## إجراءات الطوارئ

### عند اكتشاف Red Flag

```
1. توثيق الحادث
   ├── وصف ماذا حدث
   ├── التاريخ والوقت
   ├── الأطراف المعنية
   └── الأدلة المتاحة

2. تصعيد داخلي
   ├── إبلاغ مدير الشراكات
   ├── إبلاغ المدير المالي (إن لزم)
   └── إبلاغ المؤسسين (إن لزم)

3. الاتصال بالشريك
   ├── طلب توضيح
   ├── إبلاغ بالقلق
   └── طلب خطة تصحيح

4. اتخاذ القرار
   ├── استمرار
   ├── تعليق
   └── إنهاء
```

### عند إنهاء الشراكة

| الإجراء | التفاصيل |
|---------|----------|
| إشعار | إرسال إشعار رسمي |
| نقل العملاء | ضمان استمرارية العملاء |
| تسوية المستحقات | دفع العمولات المستحقة |
| استرجاع المواد | استرجاع مواد العلامة التجارية |
| التوثيق | إغلاق ملف الشريك |

## الأسئلة الشائعة

**س: متى يُعتبر الشريك high risk؟**
ج: عند وجود red flag واحد أو yellow flags متعددة.

**س: هل يمكن للشريك recovery بعد تعليق؟**
ج: نعم، إذا قدم خطة تصحيح مقنعة واجتاز فترة المراقبة.

**س: هل نحتفظ بالعمولات عند الإنهاء؟**
ج: حسب سبب الإنهاء وشروط العقد.

**س: كيف نتعامل مع red flag يتعلق بالسلوك؟**
ج: توثيق + تحقيق + إبلاغ + قرار.
