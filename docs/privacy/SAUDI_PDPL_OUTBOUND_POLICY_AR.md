# سياسة التواصل الصادر المتوافقة مع PDPL — Dealix (المرحلة 8)

سياسة التواصل الصادر المتوائمة مع نظام حماية البيانات الشخصية (PDPL) الصادر عن الهيئة السعودية للبيانات والذكاء الاصطناعي (سدايا): الأساس النظامي، تقليل البيانات، حقوق صاحب البيانات، والمعالجة داخل المملكة. الإرسال موقوف افتراضيًا (`send_enabled=false`).

هذه السياسة تبني على قائمة التحقق العامة ولا تكرّرها: `company_os/governance/pdpl_checklist.md`.

---

## 1. الأساس النظامي للمعالجة

| الجانب | القاعدة |
|--------|---------|
| طبيعة البيانات | بيانات شركات عامة وجهات اتصال عمل بدور وظيفي |
| الأساس | مصلحة عمل مشروعة ضمن سياق B2B، لا بيانات شخصية حساسة |
| لا فئات خاصة | لا تُجمَع بيانات الفئة الخاصة إطلاقًا (PDPL) |
| القوائم المشتراة | ممنوعة منعًا باتًا |

تفصيل الأساس النظامي العام: `company_os/governance/pdpl_checklist.md` §1.

---

## 2. تقليل البيانات

| المبدأ | التطبيق |
|--------|---------|
| الحد الأدنى | جمع وتخزين أقل الحقول اللازمة للتأهيل والتواصل |
| التصنيف | وفق `docs/privacy/PROSPECT_DATA_CLASSIFICATION.md` |
| الاحتفاظ | وفق `docs/privacy/OUTBOUND_DATA_RETENTION_MATRIX.md` (حذف خمول 90 يومًا) |
| لا بيانات شخصية في السجلات | نطاقات وأدوار فقط (مثل `optout-example.sa`) |

---

## 3. حقوق صاحب البيانات

| الحق | التطبيق |
|------|---------|
| حق العلم | الإفصاح عن المعالجة والغرض |
| حق الوصول | إتاحة بياناته عند الطلب |
| حق التصحيح | تصحيح البيانات عند الطلب |
| حق الإتلاف/الحذف | `docs/privacy/DELETION_REQUEST_RUNBOOK_AR.md` + حذف خمول |
| حق الاعتراض/إلغاء الاشتراك | `docs/outreach/UNSUBSCRIBE_POLICY_AR.md` ← الاستبعاد |

تفصيل الحقوق العام: `company_os/governance/pdpl_checklist.md` §3.

---

## 4. المعالجة داخل المملكة

| الجانب | الحالة |
|--------|--------|
| موقع المعالجة | داخل المملكة |
| النقل عبر الحدود | لا يوجد |
| التشفير | أثناء الحفظ والنقل |
| الوصول | مقيّد للمؤسس |

يتوافق مع `company_os/governance/pdpl_checklist.md` §4 و§5.

---

## 5. الربط مع طبقة القابلية للتسليم

- سياسة القابلية: `docs/outreach/EMAIL_DELIVERABILITY_POLICY_AR.md`.
- قائمة الاستبعاد القانونية: `data/outreach/suppression_list.jsonl`.
- معالجة الارتداد/الإلغاء: `docs/outreach/BOUNCE_UNSUBSCRIBE_HANDLING_AR.md`.

---

## 6. القواعد غير القابلة للتفاوض

1. لا بيانات فئة خاصة ولا قوائم مشتراة.
2. تقليل البيانات وحذفها عند المُحفّز.
3. احترام كل حقوق صاحب البيانات ضمن مهلة معلنة.
4. تبقى `send_enabled=false` حتى الحكم ≥ `LIMITED_SEND_READY` وموافقة المؤسس.
