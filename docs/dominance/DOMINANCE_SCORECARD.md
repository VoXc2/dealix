# Dominance Scorecard — قياس قوة الفئة

| البعد | إشارة |
|--------|--------|
| Category | السوق يستخدم **لغتك** |
| Offer | Revenue Intelligence **يبيع** |
| Proof | Proof Packs **قوية** |
| Retainer | Monthly OS **يتحول** |
| Product | modules **تقلّل** التسليم |
| Governance | **0** حوادث حرجة (حسب التعريف الداخلي) |
| Market | inbound + شركاء |
| Standard | Method **قابل للتعليم** |
| Venture | وحدة **ناضجة** |
| Holding | Core OS **يخدم** عدة وحدات |

## قرار تشغيلي

- category ضعيفة → زد **education**  
- offer ضعيف → ركز **wedge**  
- proof ضعيف → أصلح **delivery**  
- retainer ضعيف → أصلح **value cadence**  
- product ضعيف → **productize** التكرار  
- governance ضعيفة → **أوقف التوسع**
