# Dealix Business Unit Charter — قالب الميثاق

كل وحدة أعمال تحت المجموعة تحتاج **ميثاقًا واحدًا** يُحدّث دوريًا.

## الحقول

| الحقل | الغرض |
|--------|--------|
| **Name** | اسم الوحدة |
| **Problem** | المشكلة التي تحلها للمشتري |
| **Buyer** | المشتري الاقتصادي |
| **Primary Offer** | العرض الرئيسي (Sprint/برنامج) |
| **Recurring Offer** | retainer / اشتراك مُدار |
| **Product Module** | وحدات Core OS المرتبطة |
| **Proof Type** | نوع Proof Pack المتوقع |
| **KPIs** | مؤشرات شهرية |
| **Playbook** | مرجع التسليم |
| **Risk Profile** | حوكمة/جودة/سمعة |
| **Owner** | المالك التشغيلي |
| **Venture Readiness Score** | جاهزية spinout (0–100) |
