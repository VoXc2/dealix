# Dealix AI Operations Benchmark

Aggregated intelligence from real delivery (update quarterly or every N projects).

## Metrics tracked

- Average data quality score (where measured)
- Common missing fields in client datasets
- Most requested AI / RevOps workflows
- Most common risks (governance, PII, unrealistic expectations)
- Average time saved (when estimated honestly)
- Proof pack outcomes (did client act on recommendations?)
- Sprint → retainer conversion rate

## Purpose

Build **market authority** from evidence, not slogans. After ~10 comparable engagements in a segment, consider publishing an anonymized aggregate: **Dealix AI Operations Report — Saudi SMEs** (see `docs/company/DEALIX_AI_OPS_STANDARD.md` for operating principles).

## How to update

1. Pull themes from `POST_PROJECT_REVIEW.md` files (manual or scripted export).
2. Roll up into this file with **ranges and counts**, not client identifiers.
