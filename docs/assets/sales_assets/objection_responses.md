# Objection Responses (internal cheat sheet)

Align with `docs/sales/OBJECTION_HANDLING.md` for public-facing polish.

| Objection | Honest response angle |
|-----------|------------------------|
| "We already have salespeople" | Dealix removes **ops drag** — scoring, hygiene, drafts for approval — not replacing humans. |
| "Our data is messy" | We start with a **sprint** that cleans enough to act; full perfection is not the gate. |
| "We live on WhatsApp" | We support **draft + approval** workflows; we do not auto-send cold WhatsApp. |
| "Can you guarantee leads?" | No. We guarantee **process, governance, and measurable outputs** tied to your inputs. |
| "Can you scrape everything?" | No. We work from **provided** data and permitted exports. |
