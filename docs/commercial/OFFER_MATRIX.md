# مصفوفة العروض / Offer Matrix
<!-- COMMERCIAL EMPIRE | Owner: Founder | Date: 2026-05-17 -->

> القانون / Canonical: see [COMMERCIAL_WIRING_MAP.md](../COMMERCIAL_WIRING_MAP.md)

هذه الوثيقة مصنع الخدمات المنتَجة وسلّم العروض. السلّم نظام توجيه داخلي — لا يُعرض كاملًا للعميل. العميل يرى عرضًا واحدًا مناسبًا لحالته.
This document is the productized service factory and the offer ladder. The ladder is an internal routing system — it is not all shown to the customer. The customer sees one offer fitted to their situation.

## 1. قالب الخدمة المنتَجة / Productized Service Template

كل خدمة تُوصَف بعشرة حقول: المشكلة / الوعد / المدخلات / العملية / المخرجات / المدة / غير مشمول / السعر يبدأ من / مثال إثبات / مسار الترقية.
Each service uses ten fields: Problem / Promise / Inputs / Process / Outputs / Timeline / Not Included / Price Starts At / Proof Example / Upsell Path.

## 2. الخدمات المنتَجة / The Productized Services

### 2.1 تدقيق متابعة 10 عملاء / 10-Lead Follow-up Audit
- **Problem:** عملاء محتملون وصلوا، لكن لا أحد يعرف من تُوبع ومن أُهمل.
- **Promise:** تصنيف واضح لعشرة عملاء: حالة، مالك، فجوة، خطوة تالية.
- **Inputs:** 10 عملاء محتملين من العميل (تصدير CSV أو لقطة).
- **Process:** Detect → Diagnose → Decide.
- **Outputs:** جدول حالات + قائمة أولوية + ثلاث ملاحظات.
- **Timeline:** يوم عمل واحد / 1 working day.
- **Not Included:** إرسال أي رسالة، تكامل CRM.
- **Price Starts At:** 0 SAR (Free Mini Diagnostic).
- **Proof Example:** [SAMPLE_PROOF_PACK.md](SAMPLE_PROOF_PACK.md).
- **Upsell Path:** Agency Proof Pack.

### 2.2 حزمة إثبات الوكالة / Agency Proof Pack
- **Problem:** الوكالة تجلب اهتمامًا لكنها لا تستطيع إثبات ما حدث بعده لعملائها.
- **Promise:** حزمة قابلة للعرض تُري الوكالة وعملاءها أين تسرّبت القيمة.
- **Inputs:** 10 عملاء محتملين من حساب وكالة.
- **Process:** Detect → Diagnose → Decide → Draft → Prove.
- **Outputs:** Proof Pack بوسوم حقيقة + مسودات رسائل draft_only.
- **Timeline:** يومان عمل / 2 working days.
- **Not Included:** إرسال خارجي بلا موافقة.
- **Price Starts At:** 499 SAR (Revenue Proof Sprint).
- **Upsell Path:** Co-selling Pilot → Agency Partner OS.

### 2.3 تشخيص عمليات الإيراد المحوكَم / Governed Revenue Ops Diagnostic
- **Problem:** الشركة لا تعرف أين تتوقف عمليات الإيراد بعد وصول العميل.
- **Promise:** خريطة فجوات SOAEN عبر سير عمل واحد.
- **Process:** Detect → Diagnose → Decide.
- **Outputs:** تقرير تشخيص + قائمة فجوات مرتّبة.
- **Timeline:** 3–5 أيام عمل.
- **Price Starts At:** 1,500 SAR (Data-to-Revenue Pack).
- **Upsell Path:** Revenue Intelligence Sprint.

### 2.4 سبرنت ذكاء الإيراد / Revenue Intelligence Sprint
- **Problem:** الشركة تحتاج نتيجة مرئية على بيانات حقيقية قبل أي التزام شهري.
- **Promise:** نتيجة موثّقة على نطاق محدود في أسبوع.
- **Process:** الخطوات الثماني كاملة.
- **Outputs:** Proof Pack من 14 قسمًا.
- **Timeline:** أسبوع واحد.
- **Price Starts At:** 1,500 SAR (Data-to-Revenue Pack) أو 499 SAR كنسخة مصغّرة.
- **Upsell Path:** Governed Ops Retainer.

### 2.5 احتفاظ العمليات المحوكَمة / Governed Ops Retainer
- **Problem:** الفجوات تتكرر شهريًا بلا نظام مستمر.
- **Promise:** إيقاع شهري من التشخيص والإثبات.
- **Outputs:** تقرير قيمة شهري + Proof Pack.
- **Price Starts At:** 2,999 SAR/mo (Growth Ops Monthly).
- **Upsell Path:** Support OS Add-on / Executive AI Control Tower.

### 2.6 حزمة سياسة موافقة الذكاء الاصطناعي / AI Approval Policy Pack
- **Problem:** إجراءات الذكاء الاصطناعي تُنفَّذ بلا حد موافقة.
- **Promise:** سياسة موافقة موثّقة لكل إجراء خارجي.
- **Price Starts At:** 1,500 SAR (Data-to-Revenue Pack).
- **Upsell Path:** Executive AI Control Tower.

### 2.7 سبرنت دماغ الشركة / Company Brain Sprint
- **Problem:** المعرفة التشغيلية مبعثرة بلا مصدر واحد.
- **Promise:** قاعدة معرفة محوكَمة بمصدر ومالك.
- **Price Starts At:** 1,500 SAR (Data-to-Revenue Pack).
- **Upsell Path:** Growth Ops Monthly.

### 2.8 مكتب دعم العملاء بالذكاء الاصطناعي / Customer Support AI Desk
- **Problem:** أسئلة الدعم المتكررة تستهلك وقت الفريق.
- **Promise:** مسودات ردود محوكَمة بموافقة بشرية.
- **Price Starts At:** 1,500 SAR/mo (Support OS Add-on).
- **Upsell Path:** Executive AI Control Tower.

### 2.9 برج قيادة الذكاء التنفيذي / Executive AI Control Tower
- **Problem:** الإدارة لا ترى مصدر ومالك وموافقة كل إجراء ذكاء اصطناعي.
- **Promise:** رؤية تنفيذية واحدة + حزمة مجلس شهرية.
- **Price Starts At:** 7,500 SAR/mo (Executive Command Center).
- **Upsell Path:** Agency Partner OS (custom) — أي طبقة أكبر تُوسَم: خارطة طريق — غير مربوط بالدفع / Roadmap — not wired to checkout.

## 3. سلّم العروض كجدول توجيه / The Ladder as a Routing Table

```text
Free 0 SAR → 499 SAR → 1,500 SAR → 2,999 SAR/mo → 7,500 SAR/mo
مجاني → سبرنت إثبات → حزمة بيانات → نمو شهري → غرفة قيادة تنفيذية
```

| الدرجة / Rung | العرض المرجعي / Canonical offer | متى نوجّه إليه / When to route here |
|---|---|---|
| 0 SAR | Free Mini Diagnostic | أول تواصل، بناء ثقة |
| 499 SAR | Revenue Proof Sprint | يحتاج نتيجة مرئية صغيرة |
| 1,500 SAR | Data-to-Revenue Pack | بيانات جاهزة، نطاق أوسع |
| 2,999 SAR/mo | Growth Ops Monthly | فجوات متكررة تحتاج إيقاعًا |
| 7,500 SAR/mo | Executive Command Center | حوكمة على مستوى الإدارة |
| custom | Agency Partner OS | قناة وكالات |

## 4. قاعدة التسعير / The Pricing Rule

**بدل خفض السعر، خفّض النطاق / Don't discount the price — reduce the scope.**

إذا اعترض العميل على السعر، لا نخصم. ننزل درجة في السلّم أو نقلّص عدد العملاء في النطاق. السعر يبقى ثابتًا لأنه إشارة جودة وأمان.
If the customer objects to price, we do not discount. We move down one rung or shrink the lead count in scope. Price stays fixed because it signals quality and safety.
