# P3 — Executive Command Center

## الوعد

غرفة قيادة تنفيذية للمؤسس أو المالك تجمع الإيراد، الفرص، المتابعات، المخاطر، Proof Packs، والقرارات الأسبوعية في مكان واحد.

## متى نبيعه؟

بعد P2 أو بعد P1 قوي إذا كان صاحب القرار يحتاج رؤية تنفيذية وليس مجرد رسائل أو follow-up.

## السعر

- Setup: 20,000–60,000 SAR
- Monthly: 12,000–35,000 SAR/month
- Custom: حسب حجم التكامل

## المخرجات

1. Executive Cockpit
2. Weekly Decision Brief
3. Pipeline Risk Map
4. Client / Proof Pack Tracker
5. KPI Registry
6. Decision Log
7. Founder Weekly Digest
8. Governance Rules

## لا تبدأ به

لا تعرض P3 كأول منتج إلا إذا العميل كبير وصاحب القرار واضح ويطلب dashboard أو command center.
