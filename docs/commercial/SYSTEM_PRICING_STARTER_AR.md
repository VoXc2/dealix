# أسعار البداية للأنظمة الخمسة — System Starter Pricing

أسعار افتتاحية لـ Sprint تنفيذي صغير ومحدد. المشاريع الكاملة، أو الربط مع أنظمة
خارجية، أو التشغيل الشهري المستمر تُسعّر بعد التشخيص.

## جدول الأسعار

| النظام | السعر الافتتاحي | المدة | يشمل (أول نتيجة) |
| --- | ---: | --- | --- |
| نظام العروض والإثبات (Proposal & Proof OS) | 3,000 ريال | 5–7 أيام | نموذج عرض + Proof Pack |
| نظام استرجاع المتابعات (Follow-up Recovery OS) | 3,500 ريال | 7 أيام | Follow-up Queue + رسائل + تقرير |
| نظام تشغيل الإيرادات (Revenue Operating System) | 4,500 ريال | 7–10 أيام | خريطة تسرب + Workflow متابعة |
| نظام تشغيل العملاء على واتساب (WhatsApp Client OS) | 4,500 ريال | 7–10 أيام | Flow Map + Action Cards |
| لوحة القرار التنفيذي (Executive Command OS) | 5,500 ريال | 7–14 يوم | Daily Command + KPI Map |

السلّم متدرّج من 3,000 ريال (أسرع نتيجة وأضيق نطاق) إلى 5,500 ريال (أوسع ربط
وأعلى قيمة تنفيذية)، ليبدأ العميل من أصغر تعطل ويتوسّع حسب النتائج.

## ما يشمله السعر الافتتاحي

- تشخيص أولي لتحديد أكبر تعطل.
- Sprint افتتاحي واحد محدد المخرجات.
- حزمة تسليم واضحة خاصة بالنظام المختار.
- مسودات وتقارير جاهزة للمراجعة والاعتماد.

## ما لا يشمله السعر الافتتاحي

- تكاليف أدوات أو اشتراكات خارجية.
- ربط متقدم مع أنظمة طرف ثالث.
- تشغيل شهري مستمر بعد انتهاء الـ Sprint.
- حملات إعلانية أو إنتاج محتوى تسويقي.

## ملاحظة النطاق والتسعير النهائي

> النطاق النهائي يعتمد على حجم البيانات، عدد القنوات، عدد الـ workflows، الربط
> المطلوب، واحتياج التشغيل المستمر — ويُحدَّد بعد التشخيص. الأسعار أعلاه أسعار
> **بداية**، ولا تُعتمد أسعار نهائية لما بعد الـ Sprint إلا بموافقة المؤسس.

## كيف يُقدَّم السعر للعميل

1. يبدأ العميل بتشخيص سريع (الموقع: `/diagnostic`) أو نموذج البدء (`/start`).
2. نقترح النظام الأنسب لأكبر تعطل، ونعرض السعر الافتتاحي والمدة.
3. بعد التشخيص، نقدّم نطاق الـ Sprint بدقة قبل أي التزام.
4. أي توسعة أو تشغيل شهري يُسعّر بشكل منفصل ويُعتمد صراحةً.

## ضوابط

- لا ضمانات إيراد. القيمة تظهر في وضوح النظام وجودة التنفيذ.
- لا أسماء وحدات داخلية في تعاملات العميل.
- الأسعار بالريال السعودي وغير شاملة لأي رسوم أو ضرائب يفرضها طرف خارجي.
