# Dealix — التسعير الافتتاحي للأنظمة الخمسة

> الفكرة: ابدأ بـ **Sprint افتتاحي صغير**، ثم وسّع حسب النتائج. السعر واضح، والنطاق
> النهائي يُحدَّد بعد التشخيص.
>
> **مصدر الحقيقة:** حقل `startingPrice` و `priceIncludes` في `src/data/systems.ts`،
> ويتحقق منه `tests/systems.test.ts`.

---

## 1. جدول الأسعار الافتتاحية

| النظام | السعر الافتتاحي | مدة Sprint | يشمل |
| --- | ---: | ---: | --- |
| Proposal & Proof OS | **3,000 SAR** | 5–7 أيام | proposal + proof pack |
| Follow-up Recovery OS | **3,500 SAR** | 7 أيام | تحليل + queue + رسائل متابعة |
| Revenue Operating System | **4,500 SAR** | 7–10 أيام | leakage map + workflow متابعة |
| WhatsApp Client OS | **4,500 SAR** | 7–10 أيام | flow + action cards + handoff |
| Executive Command OS | **5,500 SAR** | 7–14 يوم | dashboard / report spec + تقرير قيادة يومي |

> الترتيب هنا تصاعدي بالسعر لتسهيل القراءة في صفحة `/pricing`.

---

## 2. لماذا هذه الأسعار؟

- ننزل عن السوق قليلًا في البداية لتسهيل الدخول، لكن **لا نقتل قيمة الخدمة**.
- لا نبدأ بـ 500 أو 1000 ريال؛ هذا يجعل العميل يراها خدمة بسيطة، وليست نظام تشغيل.
- هذه أسعار «أول Sprint»، وليست سعر المشروع الكامل أو التشغيل الشهري.

---

## 3. صيغة عرض السعر داخل الصفحة

```txt
يبدأ من 3,500 ريال
Sprint أولي لمدة 7 أيام
يشمل التشخيص + أول workflow + تقرير تنفيذي
```

تحت الجدول في صفحة Pricing:

```txt
هذه أسعار Sprint افتتاحي. المشاريع الكاملة أو الربط مع أنظمة خارجية أو التشغيل الشهري
تُسعَّر بعد التشخيص وباعتماد المؤسس.
```

---

## 4. قواعد التسعير (حوكمة)

مرتبطة بـ `company_os/governance/agent_permissions.md`:

- **الذكاء الاصطناعي لا يتخذ قرارات تسعير.** يقترح فقط؛ المؤسس يعتمد.
- لا تثبيت لأي سعر خارج الأسعار الافتتاحية بدون اعتماد المؤسس.
- أي خصم أو عرض خاص يدخل `approval_queue` كـ `pricing_offer` بمخاطرة `high`.
- التسعير فوق 5,000 SAR: اعتماد المؤسس + فترة تهدئة 24 ساعة.

---

## 5. مسار التوسعة بعد الـ Sprint

```txt
Starter Sprint  →  مشروع كامل (نطاق موسّع)  →  تشغيل شهري / Retainer
   (سعر معلن)        (تسعير بعد التشخيص)          (تسعير بعد التشخيص)
```

لا نعد العميل برقم إيراد محدد. نعِده بنظام واضح، أول مخرج ملموس، وتقرير يوضح أين القيمة.
