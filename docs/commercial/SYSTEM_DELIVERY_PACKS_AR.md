# حزم التسليم للأنظمة الخمسة — System Delivery Packs

ما الذي يستلمه العميل فعليًا في أول Sprint لكل نظام. الهدف أن تكون كل حزمة ملموسة
وقابلة للاستخدام والتكرار بعد التسليم.

---

## 1. نظام تشغيل الإيرادات — Revenue Operating System

- **المدة:** 7–10 أيام · **السعر الافتتاحي:** 4,500 ريال
- **أول نتيجة:** خريطة تسرب الفرص + أول workflow متابعة.

**حزمة التسليم:**
- خريطة تسرب الإيراد (Revenue Leakage Map)
- نموذج مراحل الفرصة (Opportunity Stage Model)
- Workflow متابعة قابل للتنفيذ
- قوالب مسودات متابعة
- تقرير إيرادات يومي/أسبوعي
- قائمة الإجراءات التالية للمؤسس

---

## 2. لوحة القرار التنفيذي — Executive Command OS

- **المدة:** 7–14 يوم · **السعر الافتتاحي:** 5,500 ريال
- **أول نتيجة:** تقرير القرار اليومي (Daily Command Report).

**حزمة التسليم:**
- خريطة المؤشرات (KPI Map)
- تقرير القرار اليومي (Daily Command Report)
- مصفوفة المخاطر والأولويات
- سجل القرارات (Decision Log)
- لوحة إجراءات المؤسس/الرئيس التنفيذي
- قالب المراجعة التنفيذية الأسبوعية

---

## 3. نظام استرجاع المتابعات — Follow-up Recovery OS

- **المدة:** 7 أيام · **السعر الافتتاحي:** 3,500 ريال
- **أول نتيجة:** Follow-up Queue + أول حزمة رسائل متابعة.

**حزمة التسليم:**
- قائمة المتابعة (Follow-up Queue)
- نموذج حالات العميل (Lead Status Model)
- حزمة رسائل المتابعة
- إيقاع التذكير (Reminder Rhythm)
- تقرير الاسترجاع (Recovery Report)
- قواعد التصعيد (Escalation Rules)

---

## 4. نظام تشغيل العملاء على واتساب — WhatsApp Client OS

- **المدة:** 7–10 أيام · **السعر الافتتاحي:** 4,500 ريال
- **أول نتيجة:** Flow Map + Action Cards للعميل.

**حزمة التسليم:**
- خريطة مسار واتساب (WhatsApp Flow Map)
- فحص الجاهزية (Readiness Scan)
- بطاقات الإجراء (Action Cards)
- سياسة التسليم للإنسان (Human Handoff Policy)
- دليل التحويل الآمن للبوابة (Secure Portal Handoff)
- المراجعة الأسبوعية لواتساب

> ملاحظة: لا تُطلب مفاتيح أو بيانات حساسة داخل واتساب؛ أي بيانات أو مدفوعات
> حساسة تُحوّل عبر بوابة آمنة. النظام مساعد سير عمل بعد اهتمام العميل، وليس بوتًا
> عامًا مفتوحًا.

---

## 5. نظام العروض والإثبات — Proposal & Proof OS

- **المدة:** 5–7 أيام · **السعر الافتتاحي:** 3,000 ريال
- **أول نتيجة:** نموذج عرض + Proof Pack لأول خدمة.

**حزمة التسليم:**
- قالب العرض (Proposal Template)
- قالب الإثبات (Proof Pack Template)
- النطاق وما هو خارج النطاق (Scope / Out-of-scope)
- كتلة المخاطر والافتراضات (Risk & Assumption Block)
- بطاقة الخطوة التالية (Next-step Card)
- قائمة مراجعة العرض (Proposal Review Checklist)

---

## مبادئ مشتركة لكل حزمة

- كل مخرج قابل للتسليم والاستخدام والتكرار، وليس مجرد توصية.
- المسودات تُجهَّز آليًا، لكن لا تُرسل أو تُنفّذ إلا بموافقة بشرية.
- كل إجراء مسجّل ضمن الحوكمة (الذكاء يجهّز، الإنسان يعتمد، النظام يسجّل).
- لا أسماء وحدات داخلية في المخرجات التي تصل للعميل.
