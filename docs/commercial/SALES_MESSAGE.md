# Sales Message & Service One-Liners

> The approved bilingual outreach message and the seven service one-liners.
> Commitment language only — no guarantees, no fabricated metrics, no cold
> automation. Every send is founder-approved and recorded at `CEL4`
> ([CEL spec](COMMERCIAL_EVIDENCE_STATE_MACHINE.md)).

**EN:** Dealix is a Governed Revenue & AI Operations Company. We build and run
the governed operating layer between AI experiments, revenue workflows,
approvals, evidence, and proof of value.

والأدلة، وإثبات القيمة.

## Rules for every message

- Send only after founder approval is recorded (`founder_confirmed=true`).
- Never send in bulk; never use cold WhatsApp or LinkedIn automation.
- No guaranteed outcomes, no ROI promises, no customer name without written permission.
- Classify every reply — including silence — so the evidence trail is complete.
