## 1. العقيدة / Doctrine

**العربية:** لا تُباع Dealix كمنصة ذكاء اصطناعي. تُباع كطريقة مُثبتة لتحويل النشاط الذي يحدث بعد وصول العميل المحتمل وبعد عمل الذكاء الاصطناعي إلى قرار إيرادي موثّق له: مصدر، ومالك، وموافقة، ودليل، وخطوة تالية.

**English:** Dealix is not sold as an AI platform; it is sold as a proven method to turn post-lead and post-AI activity into a documented revenue decision with source, owner, approval, evidence, and a next action.

كل رقم في هذه الطبقة يحمل وسم حقيقة: تقدير / مُلاحَظ / مؤكَّد من العميل / مؤكَّد بالدفع.
Every metric in this layer carries a truth label: Estimate / Observed / Client-confirmed / Payment-confirmed.

## 2. منهج Dealix — ثماني خطوات / The Dealix Method — 8 Steps

```text
1. Detect    كشف     — رصد leads أو سير عمل بعد الحملة أو بعد عمل الـAI
2. Diagnose  تشخيص   — تصنيف الحالة، المالك، الفجوة، المخاطرة
3. Decide    قرار    — ترتيب الأولوية: من يحتاج متابعة الآن
4. Draft     تجهيز   — صياغة الرسائل والقرارات بوضع draft_only
5. Approve   موافقة  — موافقة بشرية إلزامية قبل أي إجراء خارجي
6. Deliver   تسليم   — تنفيذ الإجراء المعتمد فقط
7. Prove     إثبات   — Proof Pack: ماذا حدث، بأي دليل
8. Expand    توسيع   — من العيّنة إلى السبرنت إلى الاحتفاظ الشهري
```

| Step | الوظيفة التجارية / Commercial function | المخرَج / Output |
|---|---|---|
| Detect / كشف | نقطة الدخول للعميل / customer entry point | قائمة leads أو workflow |
| Diagnose / تشخيص | تحويل الفوضى إلى تصنيف / chaos to classification | جدول حالات |
| Decide / قرار | إزالة الشلل التشغيلي / remove paralysis | قائمة أولويات |
| Draft / تجهيز | تقليل وقت الكتابة / cut writing time | مسودات draft_only |
| Approve / موافقة | حدّ الأمان والثقة / safety and trust line | سجل موافقات |
| Deliver / تسليم | الإجراء الفعلي / the real action | فعل واحد معتمَد |
| Prove / إثبات | سبب الشراء المتكرر / reason to buy again | Proof Pack |
| Expand / توسيع | نمو الإيراد / revenue growth | ترقية النطاق |

**أهداف 90 يومًا / 90-day targets (Estimate):** 5–8 تجارب مدفوعة، تشخيصان، سبرنت واحد، مرشّح احتفاظ واحد، 10 شركاء، تقرير معياري مصغّر واحد / 5–8 paid pilots, 2 diagnostics, 1 sprint, 1 retainer candidate, 10 partners, 1 mini benchmark report.

| القناة / Channel | الحصة / Share (Estimate) |
|---|---|
| تواصل المؤسس المباشر / Founder outbound | 40% |
| إحالات الشركاء / Partner referrals | 30% |
| محتوى وارد / Content inbound | 15% |
| ورش العمل / Workshops | 10% |
| إحالات العملاء / Customer referrals | 5% |

العقيدة الثابتة: لا ادّعاءات عائد مضمون، لا إثبات مزيّف، لا واتساب بارد، لا كشط بيانات، لا بريد إحالات عشوائي، وموافقة بشرية لكل إجراء خارجي. كل مسودة تبقى draft_only حتى الاعتماد.

Standing doctrine: no guaranteed ROI claims, no fake proof, no cold WhatsApp, no scraping, no affiliate/outreach spam, human approval for every external action. Every draft stays draft_only until approved.
