# Dealix CEO Daily Operating System

## كل صباح — 45 دقيقة

1. افتح production routes.
2. راجع 5 targets.
3. اختر 3 رسائل مخصصة.
4. راجع pipeline.
5. اختر أهم قرار يومي.

## كل ظهر — 60 دقيقة

1. إرسال 3–5 رسائل warm.
2. متابعة ردود الأمس.
3. حجز مكالمات.
4. تحديث tracker.

## كل مساء — 30 دقيقة

1. عدد الرسائل.
2. عدد الردود.
3. عدد المكالمات.
4. عدد العروض.
5. اعتراض اليوم.
6. قرار الغد.

## القاعدة

لا تبني feature جديدة قبل:
- 5 رسائل warm
- 1 discovery call
- 1 proposal مفتوح
