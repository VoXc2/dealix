# Decision Engine — من الإشارة إلى القرار

## أنواع قرارات Dealix (تشغيليًا)

Scale، Build، Pilot، Hold، Kill، Raise Price، Offer Retainer، Create Playbook، Create Benchmark، Create Business Unit، Create Venture Candidate.

## المبدأ

القرار يُبنى على **إشارات متكررة + إثبات + حوكمة** — ليس على انطباع فقط.

## الكود

`intelligence_compounding_os/decision_engine.py` — دالة `suggest_compounding_decision` للبوابات الحتمية والاختبارات.
