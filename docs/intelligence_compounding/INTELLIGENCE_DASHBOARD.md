# Intelligence Dashboard — لوحة الإشارات المركبة

## ما تعرضه (عربي تشغيلي)

- أعلى آلام السوق  
- أقوى إشارات توسع للعملاء  
- أنماط جاهزية البيانات  
- احتكاك سير العمل  
- مخاطر حوكمة متكررة  
- مرشحو التحويل إلى منتج  
- مرشحو benchmarks  
- أداء العروض  
- قضايا جودة عربية  
- إشارات retainer  
- إشارات وحدات أعمال  

## التغطية في الكود

`intelligence_dashboard.py` — `INTELLIGENCE_DASHBOARD_SIGNALS` و `intelligence_dashboard_coverage_score`.

## أمثلة قرارات (JSON مفاهيمي)

قرار مثل: «احتكاك موافقات متكرر → بناء Approval Center MVP» يربط **إشارة** بعدد تكرار بـ **قرار منتج** ورابط تجاري (Workspace / Retainers).
