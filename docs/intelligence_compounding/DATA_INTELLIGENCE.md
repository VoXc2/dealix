# Data Intelligence — ذكاء البيانات (بدون كشف بيانات العميل)

## ماذا نجمع؟

أنماط مجمّعة أو مجهّلة: معدلات التكرار، حقول ناقصة متكررة، وضوح المصدر، إشارات PII، جودة taxonomy للقطاع، جودة المدينة/المنطقة، فجوات allowed-use.

## مخرجات

- قوالب intake أفضل  
- رفع Data Readiness Score  
- تحسين lead scoring  
- مدخلات لـ benchmarks **عند استيفاء بوابات الأمان** — [INTELLIGENCE_TO_BENCHMARK_SYSTEM.md](INTELLIGENCE_TO_BENCHMARK_SYSTEM.md)

## بيانات عامة (سياق سوقي)

منصة البيانات المفتوحة السعودية (`Open Data`) مصدر **سياق عام** للمعايير والمقارنات — **لا يُخلط** مع بيانات عميل أو PII. مرجع عام: [Wikipedia — Open.data.gov.sa](https://en.wikipedia.org/wiki/Open.data.gov.sa)
