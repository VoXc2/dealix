# ضبط تكاليف الـ GTM (GTM Cost Control)

## بنود التكلفة
| البند | تحكّم | حد تنبيه |
|------|-------|----------|
| API/نماذج | حد شهري + تتبّع/مسودة | > 30% من التكلفة المتغيرة |
| أدوات (بريد/استضافة) | اشتراكات ضرورية فقط | مراجعة شهرية |
| وقت المؤسس | تتبّع ساعات (`FOUNDER_TIME_COST_MODEL_AR.md`) | > سعة البيع |
| إعلانات | افتراضياً 0 (عضوي) | أي إنفاق يحتاج موافقة |

## قواعد
- لا إنفاق متغيّر بدون مردود متتبَّع (ROI/قناة).
- الهامش الإجمالي > 60% خط أحمر.
- إيقاف القنوات سالبة الـ ROI (`pause`).
- مراجعة يومية مختصرة + أسبوعية للقنوات.
