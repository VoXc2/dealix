# ضبط تكلفة API والأدوات (API & Tool Cost Control)

## تتبّع
- تكلفة لكل: مسودة، رد مُصنَّف، تقرير.
- حد شهري لاستهلاك النماذج؛ تنبيه عند 70% و90%.

## تقليل التكلفة
- إعادة استخدام القوالب؛ تجنّب إعادة التوليد غير الضرورية.
- تنقيح المدخلات (PII/أسرار) يقلل الحجم والمخاطر.
- استخدام نماذج مناسبة للحجم؛ تجميع الطلبات حيث يصح.

## قواعد أمان مرتبطة
- لا أسرار في الـ prompts أو السجلات.
- لا بيانات شخصية للنماذج دون ضرورة وتنقيح.
