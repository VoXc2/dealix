# نموذج إيراد الـ Retainer (Retainer Revenue Model)

```
MRR = Σ (اشتراكات شهرية نشطة)
NRR = (MRR بداية + توسّع − انكماش − churn) ÷ MRR بداية
LTV ≈ ARPU × هامش × (1 / معدل churn الشهري)   [تقديري]
```

## الطبقات
| الطبقة | MRR (ر.س) | ساعات/شهر | هامش |
|--------|-----------|-----------|------|
| صغير | 3,000 | ~8 | ~75% |
| متوسط | 8,000 | ~16 | ~80% |
| مؤسسي | 20,000 | ~32 | ~85% |
