# نموذج هامش العروض (Offer Margin Model)

```
هامش العرض = (السعر − تكلفة التسليم) ÷ السعر
تكلفة التسليم = ساعات × سعر الساعة + تكلفة أدوات مباشرة
```

## العروض الحالية (من unit_economics)
| العرض | السعر (ر.س) | ساعات | الهامش التقديري |
|------|-------------|------|------------------|
| P1 Sprint | 2,500–7,500 | ~20 | ~85% |
| P2 صغير | 3,000/شهر | ~8 | ~75% |
| P2 متوسط | 8,000/شهر | ~16 | ~80% |
| P2 مؤسسي | 20,000/شهر | ~32 | ~85% |

## قواعد
- خط أحمر: هامش < 60% → ارفع السعر أو قلّل ساعات التسليم.
- السعر النهائي يحتاج موافقة بشرية (لا يحدده وكيل).
- راقب انزلاق النطاق (scope creep) فهو أكبر مهدد للهامش.
