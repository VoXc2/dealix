# نموذج ROI لقنوات GTM (مرجع موسّع)

يكمّل `CHANNEL_ROI_MODEL_AR.md` بربط القمع بالقناة.

## النسب المرجعية
```
تكلفة/صفقة بالقناة = تكلفة القناة ÷ صفقات القناة
مساهمة القناة = إيراد القناة ÷ إجمالي الإيراد
```

## مصفوفة القرار
| ROI | الحجم | الإجراء |
|-----|------|---------|
| مرتفع | منخفض | `scale` بحذر |
| مرتفع | مرتفع | حافظ + حسّن الجودة |
| منخفض | أي | `improve` ثم `pause` إن استمر |
| سالب | أي | `pause` فوراً |
