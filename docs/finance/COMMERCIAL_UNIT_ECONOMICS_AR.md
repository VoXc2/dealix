# اقتصاديات الوحدة التجارية (Commercial Unit Economics)

تجمع هامش العرض + CAC + الاسترداد + LTV في صورة واحدة لكل عرض.

| المؤشر | P1 Sprint | P2 Retainer (متوسط) |
|--------|-----------|----------------------|
| السعر | 2,500–7,500 ر.س | 8,000 ر.س/شهر |
| ساعات التسليم | ~20 | ~16/شهر |
| الهامش | ~85% | ~80% |
| التحصيل | 50%+ مقدماً | شهري مقدم |
| CAC | TBD | TBD |
| الاسترداد | فوري (هامش موجب) | TBD أشهر |
| LTV | السعر × هامش | MRR × هامش × بقاء(TBD) |

## قواعد
- العرض يجب أن يرتبط بالكتالوج (`data/productized_services/services.yaml`).
- لا سعر نهائي بدون موافقة؛ لا التزام يتجاوز النطاق.
- الهامش > 60% خط أحمر.
