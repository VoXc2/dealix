# نموذج CAC واسترداده (CAC & Payback Model)

> نموذج بافتراضات قابلة للتعديل — املأ الأرقام الفعلية. لا ضمانات.

## CAC
```
CAC = (تكلفة الأدوات/API + قيمة وقت المؤسس + تكلفة القناة) ÷ عدد العملاء المكتسبين
```
- وقت المؤسس: انظر `FOUNDER_TIME_COST_MODEL_AR.md` (تكلفة فرصة/ساعة TBD).
- تكلفة القناة: التواصل البارد ≈ تكلفة الأدوات + الوقت (لا إنفاق إعلاني افتراضاً).

## استرداد CAC (Payback)
```
Payback (أشهر) = CAC ÷ (متوسط الإيراد الشهري × الهامش)
```
- P1 دفعة واحدة: الاسترداد فوري إن الهامش موجب (تحصيل مسبق).
- P2 Retainer: 
```
Payback ≈ CAC ÷ (MRR × هامش)
```

## حدود القرار
| الحالة | الإجراء |
|--------|---------|
| CAC > 20% من LTV | `improve conversion` أو `reduce cost` |
| Payback > 6 أشهر (Retainer) | `increase price` أو `switch sector` |
| Payback فوري + هامش > 60% | `scale` (بحذر، مع جودة) |

## LTV (تقديري)
```
LTV ≈ متوسط الإيراد × الهامش × متوسط مدة البقاء (TBD)
```
