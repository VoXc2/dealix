# نموذج سعة البيع (Sales Capacity Model)

السعة محكومة بوقت المؤسس للتسليم (مرحلة مبكرة، مؤسس واحد).

```
أقصى صفقات/شهر = (ساعات متاحة للتسليم/شهر) ÷ (ساعات تسليم/عرض)
```
