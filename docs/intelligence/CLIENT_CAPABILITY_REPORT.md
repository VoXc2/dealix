# Dealix Capability Report — ما يراه العميل

العميل لا يرى Strategy Office كاملًا — يرى **تقرير قدرات** استراتيجي.

## محتوى مقترح

1. **Current capability score** (DCI أو تفكيك القدرات السبع)  
2. **Transformation gap** (DTG) والهدف بعد Sprint / Retainer  
3. **أعلى فرصة قيمة** قابلة للتنفيذ  
4. **Sprint أو Diagnostic موصى به**  
5. **ملاحظات مخاطر وحوكمة**  
6. **مقاييس proof متوقعة**  
7. **مسار retainer** (Continue / Expand / Pause)

**الرسالة:** «نقيس قدراتكم، نحدد فجوة التحول، نبدأ بأعلى فرصة **قابلة للتنفيذ**، ونثبت القيمة قبل التوسع» — لا «نبيع AI» فحسب.
