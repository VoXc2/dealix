# Anti-Empty-SaaS Rules — خمس قواعد ضد SaaS مبكر وفارغ

1. **No SaaS module before internal usage** — الاستخدام الداخلي أولًا.  
2. **No self-serve before assisted value is proven** — إثبات القيمة بمساعدة قبل self-serve.  
3. **No billing engine before repeatable retainers** — تكرار الدفع الشهري أولًا.  
4. **No marketplace before assets are proven** — أصول وثقة قبل السوق.  
5. **No enterprise controls before enterprise demand** — لا تبني RBAC/تعقيد بلا سحب حقيقي.
