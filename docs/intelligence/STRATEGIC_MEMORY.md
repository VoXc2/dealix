# Strategic Memory — ذاكرة الشركة

**المبدأ:** الشركة التي **لا تحفظ التعلم** تعيد نفس الأخطاء.

## مخرجات كل مشروع (أسئلة إلزامية منطقيًا)

- ما الذي **نجح**؟ ما الذي **فشل**؟  
- ما الذي **تكرر**؟ ما الذي **فاجأنا**؟  
- ما **الأكثر قيمة** للعميل؟ ما **المخاطر** التي ظهرت؟  
- ما الذي يصبح **template**؟ **منتج**؟ **محتوى**؟
