# Anti-Agency Rules — خمس قواعد ضد الانزلاق لوكالة

إذا التزمت بها، Dealix **لا تتحول لوكالة تقليدية**.

1. **No custom work without productization review** — أي عمل مخصص يمر مراجعة: هل يتكرر؟ هل رأس مال؟  
2. **No project closes without capital asset** — إغلاق بلا أصل قابل لإعادة الاستخدام = فشل مؤسسي.  
3. **No service scales without repeatable checklist** — لا توسع بلا تسليم قابل للتكرار.  
4. **No client work without proof pack** — لا إثبات = لا claim للبيع اللاحق.  
5. **No feature without repeated demand** — المنتج من **السوق** لا من «فكرة ليلاً».
