# حقيبة التوزيع — Dealix Distribution Kit

> فهرس رئيسي يربط استراتيجية الذهاب إلى السوق المكوّنة من 20 نقطة بملفات المستودع الفعلية.
> A master index mapping the 20-point go-to-market strategy to real repo files.

الاستراتيجية بلا ملفات قابلة للتنفيذ تبقى نيّة. هذه الحقيبة تحوّل كل نقطة إلى أصل في المستودع.
A strategy without executable files stays an intention. This kit turns every point into a repo artifact.

---

## كيف تستخدم هذه الحقيبة — How to Use This Kit

**العربية:**
- ابدأ من جدول الأقسام العشرين أدناه. كل صف يشير إلى ملف حقيقي يمكنك فتحه الآن.
- "الحالة" تخبرك إن كان الأصل موجوداً، أو جديداً (أُنشئ مع هذه الحقيبة)، أو موسَّعاً.
- لا تعيد كتابة المحتوى — افتح الملف المرتبط واعمل منه.
- عند تعارض الأسعار، خريطة الربط التجاري هي الحَكَم.

**English:**
- Start from the 20-section table below. Each row points to a real file you can open now.
- "Status" tells you whether the artifact already existed, is new (created with this kit), or was extended.
- Do not re-author content — open the linked file and work from it.
- When prices conflict, the Commercial Wiring Map is the arbiter.

## المصادر المرجعية — Canonical Sources

**العربية:**
- **الأسعار:** [`COMMERCIAL_WIRING_MAP.md`](COMMERCIAL_WIRING_MAP.md) هو المصدر الوحيد للأسعار. لا تذكر سعراً إلا كما يظهر فيه.
- **القواعد غير القابلة للتفاوض:** [`00_constitution/NON_NEGOTIABLES.md`](00_constitution/NON_NEGOTIABLES.md) هو الصياغة الرسمية.
- **تنبيه تعارض:** [`OFFER_LADDER_AND_PRICING.md`](OFFER_LADDER_AND_PRICING.md) يعرض نطاق تسعير مختلف/أقدم. هذا التعارض مرفوع لمراجعة المؤسس — **خريطة الربط التجاري هي الحَكَم**.

**English:**
- **Pricing:** [`COMMERCIAL_WIRING_MAP.md`](COMMERCIAL_WIRING_MAP.md) is the single source for prices. Quote a price only as it appears there.
- **Non-negotiables:** [`00_constitution/NON_NEGOTIABLES.md`](00_constitution/NON_NEGOTIABLES.md) holds the canonical wording.
- **Discrepancy flag:** [`OFFER_LADDER_AND_PRICING.md`](OFFER_LADDER_AND_PRICING.md) shows a different/older pricing band. This discrepancy is flagged for founder review — **the Commercial Wiring Map wins**.

## الصيانة — Maintenance

**العربية:**
عند تغيّر أي سعر، حدّث **فقط** [`COMMERCIAL_WIRING_MAP.md`](COMMERCIAL_WIRING_MAP.md). كل وثائق هذه الحقيبة تشير إليه ولا تكرّر الأرقام — هكذا لا يحدث انحراف. لا تنسخ سعراً إلى وثيقة أخرى.

**English:**
When any price changes, update **only** [`COMMERCIAL_WIRING_MAP.md`](COMMERCIAL_WIRING_MAP.md). Every kit document references it and never restates the numbers — that is how drift is prevented. Do not copy a price into another document.
