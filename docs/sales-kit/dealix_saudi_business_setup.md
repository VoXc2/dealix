# دليل تأسيس Dealix كشركة سعودية

> خطوات تأسيس شركة Dealix رسمياً في السعودية: السجل التجاري، ضريبة القيمة المضافة، فوترة ZATCA، الحساب البنكي.

## 1. السجل التجاري (CR)

### المنصة
- **وزارة التجارة**: [mc.gov.sa](https://mc.gov.sa) → بوابة "مراس" ([business.sa](https://business.sa))
- **منصة مركزية موحّدة** لإنشاء الشركات (منذ 2023)

### المتطلبات
- هوية وطنية سارية (للمؤسس)
- عنوان وطني مسجّل
- اسم تجاري متاح (يفضّل "Dealix" أو "ديلكس للذكاء الاصطناعي")
- رأس مال: مؤسسة فردية (بدون حد أدنى) أو ش.ذ.م.م (5,000 ر.س كحد أدنى)

### نوع الكيان الموصى به
**الخيار 1: مؤسسة فردية (Sole Proprietorship)**
- الأسرع (يوم واحد)
- لا يتطلب رأس مال
- المالك مسؤول شخصياً عن الديون
- مناسب للمرحلة الأولى (قبل الـ seed)

**الخيار 2: شركة ذات مسؤولية محدودة (LLC/ش.ذ.م.م)** — ⭐ موصى به
- فصل المسؤولية القانونية (حماية الأصول الشخصية)
- مطلوب لـ seed round من VC
- رأس مال مرن (5,000 ر.س يكفي للبدء)
- يأخذ 3-7 أيام عمل

### الخطوات (LLC)
1. **حجز الاسم التجاري** عبر مراس → 200 ر.س
2. **عقد التأسيس** (Articles of Association) — قالب جاهز من المنصة
3. **توثيق العقد** إلكترونياً (كاتب عدل) — 500 ر.س
4. **إصدار السجل التجاري** — 1,200 ر.س (سنة واحدة)
5. **الاشتراك في الغرفة التجارية** — 800 ر.س (إجباري)
6. **فتح ملف GOSI** (التأمينات) — مجاني، لكن إجباري إذا في موظفين

### الأنشطة الموصى بها في CR
- **نشاط رئيسي**: تطوير البرمجيات وتصميم وإدارة الكيانات السحابية
- **أنشطة فرعية**:
  - استشارات تقنية وإدارية
  - تقنية المعلومات (أنشطة خدمات)
  - التسويق الرقمي
- **رمز ISIC**: 6201 (أنشطة برمجة الحاسب)

### التكلفة الإجمالية المتوقعة
- مؤسسة فردية: **~1,500 ر.س**
- ش.ذ.م.م: **~3,500 ر.س + رأس المال**

### المدة المتوقعة
- مؤسسة فردية: **1-2 يوم**
- ش.ذ.م.م: **5-10 أيام**

## 2. ضريبة القيمة المضافة (VAT)

### متى تُسجّل؟
- **إجباري** إذا تجاوز الإيراد السنوي **375,000 ر.س**
- **اختياري** إذا تجاوز **187,500 ر.س**
- **للـ SaaS الموجّه للسعوديين**: سجّل من اليوم الأول لتجنّب مشاكل الفوترة لاحقاً

### المنصة
- **هيئة الزكاة والضريبة والجمارك (ZATCA)**: [zatca.gov.sa](https://zatca.gov.sa)
- تسجيل إلكتروني عبر "فاتورة" ([fatoora.zatca.gov.sa](https://fatoora.zatca.gov.sa))

### المتطلبات
- السجل التجاري (CR)
- الرقم المميز (TIN) — يُصدر تلقائياً عند التسجيل
- حساب بنكي (لتحصيل الضريبة)
- بيانات التواصل (عنوان، جوال، إيميل)

### الخطوات
1. **تسجيل الدخول** على zatca.gov.sa بهوية النفاذ الوطني
2. **اختيار "التسجيل في ضريبة القيمة المضافة"**
3. **تعبئة النموذج**:
   - بيانات المنشأة (CR)
   - النشاط الاقتصادي
   - الإيراد المتوقع السنوي
   - تاريخ بدء النشاط
4. **إرسال الطلب** → يُراجع في 2-5 أيام عمل
5. **استلام الشهادة** (VAT Certificate) بالبريد الإلكتروني

### النسبة الحالية
- **15%** على جميع المبيعات داخل السعودية
- **0%** على الصادرات (خارج السعودية + دول GCC)

### الإقرار الضريبي
- **شهري** إذا الإيراد > 40 مليون ر.س/سنة
- **ربع سنوي** إذا الإيراد أقل
- **الموعد**: آخر يوم من الشهر التالي للفترة الضريبية

## 3. فوترة ZATCA الإلكترونية (E-Invoicing)

### الخلفية
- **المرحلة 1 (ديسمبر 2021)**: فواتير XML/PDF A-3 إلزامية لكل المنشآت المسجّلة في VAT
- **المرحلة 2 (2023-2025)**: ربط مباشر مع ZATCA عبر API + QR code + CSID + UUID

### متطلبات Dealix (مرحلة 2)
- نظام فوترة متوافق مع ZATCA (Dealix يصدر فواتير SaaS للعملاء)
- **QR Code** على كل فاتورة (Base64 TLV)
- **UUID** فريد لكل فاتورة
- **Hash** للفاتورة السابقة (chain of invoices)
- **CSID** (Cryptographic Stamp Identifier) — يُصدر من ZATCA بعد onboarding
- **XML** بصيغة UBL 2.1

### الحل التقني لـ Dealix
#### الخيار 1: استخدام مزود معتمد (سريع)
- **Moyasar** — يصدر فواتير متوافقة تلقائياً مع ZATCA
- **Foodics / Odoo Saudi** — حلول ERP كاملة
- **التكلفة**: مدمجة في رسوم Moyasar (2.75%)

#### الخيار 2: بناء داخلي (Dealix API)
```python
# api/invoicing/zatca.py
import uuid, hashlib, base64
from lxml import etree

def generate_invoice(invoice_data):
    invoice_uuid = str(uuid.uuid4())
    # بناء XML UBL 2.1
    xml = build_ubl_xml(invoice_data, invoice_uuid)
    # حساب hash
    invoice_hash = hashlib.sha256(xml).hexdigest()
    # إضافة QR code (TLV)
    qr_tlv = build_tlv(seller, vat, timestamp, total, vat_total)
    qr_b64 = base64.b64encode(qr_tlv).decode()
    return {"xml": xml, "hash": invoice_hash, "qr": qr_b64, "uuid": invoice_uuid}
```

### الموعد النهائي لـ Dealix
- **فور تسجيل VAT** + تجاوز العتبة → إلزامي
- **أول العملاء B2B** سيطلبون فواتير متوافقة → ابدأ من اليوم الأول

### الموارد
- [دليل ZATCA للمطوّرين](https://zatca.gov.sa/ar/E-Invoicing/Pages/default.aspx)
- [GitHub - ZATCA-compatible libraries](https://github.com/topics/zatca)

## 4. الحساب البنكي التجاري

### البنوك الموصى بها (للشركات الناشئة)
| البنك | المميّزات | الرسوم الشهرية |
|-------|-----------|----------------|
| **الراجحي (Al Rajhi Business)** | الأكثر انتشاراً، دعم SAR + USD | 0 ر.س (حد أدنى رصيد 5,000) |
| **الأهلي (SNB)** | API قوي، تكامل مع Moyasar ممتاز | 50 ر.س |
| **STC Pay Business** | للـ startups (حتى 100k ر.س معاملة/شهر) | 0 ر.س |
| **Lean Technologies** | Open banking للشركات التقنية | حسب الاستخدام |

### الموصى بـ Dealix: **الراجحي Business + STC Pay**
- الراجحي للحساب الرئيسي + استقبال تحويلات VC
- STC Pay للمعاملات اليومية الصغيرة (< 100k ر.س)

### المتطلبات
- السجل التجاري (CR)
- شهادة VAT
- هوية المؤسس + صورة شخصية
- عقد الشركة (للـ LLC)
- عنوان وطني
- نموذج "اعرف عميلك" (KYC)

### الخطوات
1. **حجز موعد** عبر تطبيق البنك أو الموقع
2. **زيارة الفرع** (بعض البنوك تقبل عن بُعد عبر مقابلة فيديو)
3. **تقديم المستندات**
4. **التفعيل** — يأخذ 3-7 أيام عمل
5. **استلام الشيكات + البطاقة + الـ API keys** (لـ Dealix)

### ربط الحساب مع Moyasar
- **IBAN** للحساب → يُضاف في Moyasar dashboard
- **التسوية** (settlement): T+2 (يومين عمل)
- **الرسوم**: 2.75% من Moyasar → يُسحب تلقائياً قبل التحويل

## 5. GOSI (التأمينات الاجتماعية)

### متى تُسجّل؟
- **فور توظيف أول موظف** (حتى لو سعودي أو غير سعودي)
- المؤسس: اختياري (إذا ما يأخذ راتب)

### الاشتراكات (2026)
- **السعوديون**: 21.5% إجمالي
  - صاحب العمل: 11.75%
  - الموظف: 9.75%
- **غير السعوديين**: 2% (إصابات عمل فقط)

### المنصة
- **GOSI**: [gosi.gov.sa](https://gosi.gov.sa)
- ربط مباشر مع CR (تلقائي عند تسجيل موظف)
