# Moat — عدم التسليع (No Commodity Strategy)

أي عرض سطحي يصبح سلعة. الخندق في **قدرة + حوكمة + proof** لا في مخرج وحيد.

---

## إطار التحويل

| سلعة محتملة | موضع Dealix |
|-------------|-------------|
| Lead generation | **Revenue Intelligence:** جاهزية بيانات، source passport، scoring، draft-only، حوكمة، Proof Pack، انضباط pipeline |
| Chatbot | **Company Brain:** سجل مصادر، اقتباسات، صلاحيات، insufficient evidence، فجوات معرفية، QA، Proof Pack |
| Automation | **Governed Workflow:** مالك، موافقة، audit، SOP، مراجعة بشرية، إثبات توفير وقت |
| Dashboard | **Executive Operating Report:** مؤشرات مبنية على مصدر، ملاحظات حوكمة، توصيات قرار، proof events |

---

## قاعدة Dealix

لا تبع **output** فقط. بع **capability + governance + proof**.
