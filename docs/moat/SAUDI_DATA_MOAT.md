# Saudi Data Moat — بيانات عامة بلا خلط مع عميل

السعودية تملك **منظومة بيانات مفتوحة** تدعم ذكاء السوق و benchmarks **العامة** عند الاستخدام الصحيح.

---

## فرصة

- توسيع السوق وتجزئة القطاعات  
- رؤى مدن/مناطق  
- سياق benchmark  
- إثراء طبقة **عامة** فقط  

**مرجع:** منصة البيانات المفتوحة — وصف شائع لآلاف مجموعات البيانات من مئات الجهات بصيغ قابلة للقراءة آلياً ومبادئ خصوصية — [البيانات المفتوحة السعودية — data.gov.sa](https://data.gov.sa).

---

## ما لا تفعله Dealix

- لا تخلط البيانات العامة مع **PII** عميل  
- لا تستخدمها ذريعة **لـoutreach غير مسموح**  
- لا تدّعي دقة غير مثبتة  
- لا تتجاهل **الإسناد (attribution)** والترخيص
