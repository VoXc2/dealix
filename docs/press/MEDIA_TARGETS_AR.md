# الجهات الإعلامية المستهدَفة — Media Targets (AR)

هذه **فئات وأنواع** من الجهات الإعلامية المناسبة لرسالة Dealix — لا قائمة بجهات
بعينها، ولا أي ادّعاء لعلاقة قائمة أو تغطية سابقة. الاستهداف بالفئة لا بالاسم.

## قاعدة حاكمة
- ممنوع تماماً ادّعاء علاقة إعلامية، أو تغطية حصلت، أو اهتمام جهة بعينها.
- لا تُذكر أسماء صحفيين أو منافذ كأنهم على تواصل أو موافقون على التغطية.
- لكل لحظة صحفية: تُختار فئة/فئات مناسبة ثم ≤ 3 طروحات (مرجع `PRESS_OUTREACH_OS_AR.md`).

## فئات الجهات (أنواع لا أسماء)
| الفئة (نوع المنفذ) | لماذا قد تناسب رسالتنا |
|--------------------|-----------------------|
| منصّات ريادة الأعمال السعودية | تغطّي قصص بناء شركات محلية B2B |
| نشرات قطاع SaaS الإقليمية | مهتمّة بأدوات الإيرادات والتشغيل |
| مدوّنات/بودكاست تقنية الأعمال | تناقش أنظمة المبيعات والعمليات |
| نشرات بريدية متخصّصة في النموّ B2B | جمهور صنّاع قرار في القطاعات المستهدَفة |
| منصّات أخبار التقنية السعودية | تغطية إطلاقات وإنجازات موثّقة |
| مجتمعات/منتديات قطاعية مهنية | وصول مباشر لجمهور القطاع ذي الصلة |

(القائمة فئات قابلة للتوسعة بفئات أخرى من نفس الطابع — لا أسماء جهات.)

## معايير اختيار الفئة لكل لحظة
- ملاءمة الجمهور لقطاعاتنا المستهدَفة (مرجع `sector` enum).
- ملاءمة نوع المحتوى لنوع الإنجاز الموثّق.
- واقعية الوصول: فئة يُعقَل أن يهمّها إنجاز بهذا الحجم.

## ممنوعات صارمة
- لا «كما نُشر في ...»، لا «بالتعاون مع ...»، لا اقتباس صحفي لم يصدر فعلاً.
- لا تغطية مزعومة، لا جوائز مُختلَقة، لا أسماء عملاء، لا أرقام بلا مصدر.
- لا عبارات محظورة (نضمن/نضاعف الإيرادات/نتائج مضمونة/بدون مخاطرة/10x/
  guaranteed revenue/no risk). الأفعال المسموحة فقط: نساعد/نقلّل/نرتّب/نحلّل/نوضّح/نلاحظ.
