# First 30 Days — أول 30 يوم
**Dealix — Agent #3**

> **الغرض:** كل ما يحدث في أول 30 يوم من علاقة العميل مع Dealix. مكمل لـ `os/16_CLIENT_ONBOARDING_TEMPLATE.md`.

---

## 1. The 30-Day Mission

**Week 1:** Setup + First Workflow
**Week 2:** Reporting + Adoption
**Week 3:** Value Proof
**Week 4:** Optimization + Renewal Path

---

## 2. Week 1: Setup + First Workflow

### 2.1 Day 1 (Kickoff)
- Welcome email (founder approved)
- Kickoff call (60 min)
- Stakeholder introductions
- Success metric confirmation
- Communication plan
- Schedule setup

### 2.2 Day 2-3 (Setup)
- Access provisioning
- CRM integration
- Tool setup
- Initial training (1-2 hours)

### 2.3 Day 4-5 (First Workflow)
- First workflow designed
- Templates created
- Approval workflow setup
- First test run

### 2.4 Day 5 (Check-in)
- End of week 1 check-in
- Issues identified
- Next week plan

### 2.5 Deliverables (End of Week 1)
- ✅ Access complete
- ✅ First workflow designed
- ✅ Team trained
- ✅ Communication plan
- ✅ Next week plan

---

## 3. Week 2: Reporting + Adoption

### 3.1 Day 8-9 (Reporting Setup)
- Dashboard configuration
- Weekly report template
- Data flow setup
- Alerts setup

### 3.2 Day 10-12 (Adoption Push)
- Team training (additional)
- Usage monitoring
- Issue triage
- Quick wins

### 3.3 Day 12 (Mid-week Check-in)
- Adoption status
- Issues
- Quick wins
- Adjustments

### 3.4 Day 14 (Week 2 Check-in + First Weekly Report)
- First weekly report delivered
- Usage review
- Issues review
- Week 3 plan

### 3.5 Deliverables (End of Week 2)
- ✅ Reporting live
- ✅ Adoption > 50%
- ✅ First weekly report
- ✅ Issues triaged

---

## 4. Week 3: Value Proof

### 4.1 Day 15-16 (Value Documentation)
- Initial value identification
- Metrics tracking
- Quick wins documentation

### 4.2 Day 17-19 (Optimization)
- Adjust workflows
- Add templates
- Improve reports
- Address feedback

### 4.3 Day 21 (Week 3 Check-in + 30-Day Prep)
- 30-day report drafting
- Client feedback collection
- Plan for 30-day review

### 4.4 Deliverables (End of Week 3)
- ✅ Value documented
- ✅ Workflows optimized
- ✅ Client feedback
- ✅ 30-day report draft

---

## 5. Week 4: Optimization + Renewal Path

### 5.1 Day 22-23 (Final Optimization)
- Final adjustments
- Documentation
- Training refresh

### 5.2 Day 24-26 (30-Day Report + Review)
- 30-day report delivered
- 30-day review call
- Success metric review
- Issues closed
- Next 30 days plan

### 5.3 Day 27-28 (Renewal Path Setup)
- If retainer: confirm terms
- If implementation: discuss next phase
- If one-time: case study ask
- Reference request (with permission)

### 5.4 Day 30 (Wrap-up)
- 30-day summary
- Stakeholder update
- Renewal/expansion discussion
- Testimonial/case study

### 5.5 Deliverables (End of Week 4)
- ✅ 30-day report delivered
- ✅ Success metric reviewed
- ✅ Renewal/expansion discussed
- ✅ Case study request
- ✅ Next 30 days plan

---

## 6. The 30-Day Report Structure

### 6.1 Header
- Client name
- Date
- Version
- Author (CS Agent)
- Founder-approved

### 6.2 Executive Summary (1 page)
- Success metric status
- Key wins
- Key issues
- Next 30 days

### 6.3 Detailed Sections
- What was done
- What improved
- What's blocked
- What needs approval
- Next best action
- Renewal/expansion signal

### 6.4 Data Section
- Usage metrics
- Value metrics
- Risk indicators
- ROI estimate (with is_estimate)

### 6.5 Recommendations
- Continue / adjust / pivot
- Add / remove scope
- Renewal / expansion

### 6.6 Approval
- CS Agent signature
- Founder approval
- Client sign-off

---

## 7. The 30-Day Decision Points

### 7.1 At 30-Day Review
- ✅ Continue as planned
- ⚠️ Adjust scope or approach
- 🚨 Major issue (escalate)
- 🛑 Pause / refund (rare)

### 7.2 Based On
- Success metric achievement
- Client engagement
- Usage data
- Feedback

---

## 8. The 30-Day Success Criteria

### 8.1 For Each Offer
| Offer | First 30-Day Success |
|-------|----------------------|
| Diagnostic | 1 report delivered + recommendation |
| Workflow | 1 workflow live + 50% adoption |
| Starter | 2-3 workflows live + 1 dashboard |
| Full OS | 4-5 workflows live + 2 dashboards |
| Retainer | 2 experiments + 1 report |
| Custom | per SOW |

### 8.2 Universal
- ✅ First weekly report delivered
- ✅ Client trained
- ✅ Adoption > 50%
- ✅ First value documented
- ✅ Renewal/expansion discussed

---

## 9. Anti-Patterns

### 9.1 ❌ Don't Do
- ❌ Skip kickoff
- ❌ Miss weekly check-ins
- ❌ No 30-day report
- ❌ No success metric tracking
- ❌ No renewal discussion

### 9.2 ✅ Do
- ✅ Always kickoff
- ✅ Always weekly check-in
- ✅ Always 30-day report
- ✅ Always track metric
- ✅ Always discuss renewal
