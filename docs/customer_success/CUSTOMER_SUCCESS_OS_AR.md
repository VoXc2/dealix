## 1. The CS Mission

**Customer Success = العميل يحقق النتيجة المرجوة + يبقى + يجدد + يوسّع.**

ليس:
- ❌ Support (رد على تذاكر)
- ❌ Account management (علاقة فقط)
- ❌ Sales (محاولة بيع)

**بل:**
- ✅ Value delivery
- ✅ Outcome tracking
- ✅ Proactive support
- ✅ Renewal + expansion

## 6. The CS Lifecycle

### 6.1 Onboarding (Week 1-2)
**Goals:**
- Access complete
- First workflow live
- Client trained

**Activities:**
- Kickoff call
- Setup walkthrough
- First deliverable
- Training session

**Owner:** Delivery Lead + CS Agent

### 6.2 Adoption (Week 3-8)
**Goals:**
- Usage > 50%
- Issues resolved
- First report delivered

**Activities:**
- Weekly check-in
- Issue triage
- Usage tracking
- First 30-day report

**Owner:** CS Agent

### 6.3 Value Realization (Month 2-3)
**Goals:**
- Success metric met
- Client happy
- Proof documented

**Activities:**
- 60-day report
- 90-day report
- Success metric analysis
- Case study ask (with permission)

**Owner:** CS Agent + Founder

### 6.4 Retention + Expansion (Month 3+)
**Goals:**
- Renewed (or in progress)
- Expansion identified
- Reference available

**Activities:**
- Quarterly review
- Renewal discussion
- Expansion pitch
- Reference request

**Owner:** CS Agent + Founder

## 7. CS Metrics

### 7.1 Adoption
- Onboarding completion: > 95%
- Usage rate: > 70%
- First value: < 14 days

### 7.2 Value
- Success metric met: > 70%
- Client satisfaction: > 4/5
- NPS: > 50

### 7.3 Retention
- Renewal rate: > 70%
- Churn: < 10% (monthly)
- Expansion rate: > 20%

### 7.4 Operational
- Response time: < 24h
- Resolution time: < 7 days
- Health score avg: > 70

---

## 12. CS Success Factors

### 12.1 ✅ What Works
- Proactive (not reactive)
- Documented (visible)
- Value-focused (not just task)
- Personal (relationship)
- Consistent (cadence)

### 12.2 ❌ What Fails
- Reactive (only when client asks)
- Hidden (no reports)
- Task-focused (no value)
- Transactional (no relationship)
- Inconsistent (no cadence)
