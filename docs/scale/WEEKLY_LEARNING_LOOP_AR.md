# Weekly Learning Loop — حلقة التعلّم الأسبوعية

> النظام لا يكتفي بالتشغيل؛ يتعلّم كل أسبوع ويعدّل الاستهداف والزوايا والعروض.
> هذه الحلقة شرط لفتح Scale Mode (يجب وجود تقرير تعلّم أسبوعي).

---

## الحلقة

```txt
1. Measure   — اجمع نتائج الأسبوع (ردود، تحويل، تسليم، هامش).
2. Learn     — أي قطاع/زاوية/عرض أعطى أفضل نتيجة؟
3. Decide    — ماذا نثبّت؟ ماذا نوقف؟ ماذا نجرّب التالي؟
4. Adjust    — حدّث الاستهداف والزوايا والعروض و scale state.
5. Repeat    — أسبوع جديد، متغيّر واحد جديد.
```

---

## مدخلات الحلقة

| المصدر | ماذا يعطي |
|--------|-----------|
| `reports/experiments/WEEKLY_REVENUE_EXPERIMENTS.md` | نتائج التجارب |
| `reports/experiments/EMAIL_ANGLE_RESULTS.md` | أفضل زاوية إيميل |
| `reports/experiments/OFFER_CONVERSION_REVIEW.md` | أفضل عرض تحويلًا |
| `reports/deliverability/DAILY_DELIVERABILITY_REVIEW.md` | صحة الإرسال |
| `reports/scale/DELIVERY_CAPACITY_REVIEW.md` | استغلال القدرة |

---

## مخرجات الحلقة

- `reports/scale/WEEKLY_SCALE_REVIEW.md` — ملخّص أسبوعي وقرار الوضع التالي.
- تحديث `company_os/scale/scale_state.json` و `ultimate_scorecard.json`.
