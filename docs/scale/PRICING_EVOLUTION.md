# Dealix Pricing Evolution

السعر يتبع **نضج التسليم والأثر**، لا العاطفة.

---

## المراحل

| Stage | الوصف |
|-------|--------|
| **1. Founder Proof** | سعر أنسب نسبيًا مقابل **proof قوي** |
| **2. Repeatable Sprint** | سعر ثابت **أعلى** (تكرار مثبت) |
| **3. Managed Retainer** | **MRR** |
| **4. Platform + Service** | اشتراك + خدمة مُدارة |
| **5. Enterprise** | setup + عقد سنوي + دعم |
