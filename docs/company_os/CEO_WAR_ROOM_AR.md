# Dealix CEO War Room

## Daily Command

- Open demo.
- Review CEO Brain.
- Pick 5 warm targets.
- Approve 3 messages.
- Send manually.
- Record replies.
- Create/advance 1 P1 proposal.

## Weekly Command

- Review scorecard.
- Choose best niche.
- Kill weak niche.
- Improve offer.
- Turn proof into content.
- Ask for referrals.

## Monthly Command

- Review MRR.
- Review CAC/time cost.
- Review conversion from P1 to P2.
- Decide one strategic build only.
- Freeze low-value features.

## CEO Rule

If no outreach happened today, no new build work is allowed.
