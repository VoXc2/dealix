# Dealix SLA and Support Model

## Internal SLA

| Area | Target |
|---|---:|
| API health | 99%+ |
| Web health | 99%+ |
| First response to production issue | same day |
| Critical production issue | immediate founder escalation |
| Client P1 delivery | 5–7 days |
| Weekly P2 scorecard | once/week |

## Support Tiers

### P1 Diagnostic

- 5–7 day delivery.
- One review call.
- Proof Pack.

### P2 Monthly Ops

- Weekly scorecard.
- Weekly follow-up queue.
- Monthly executive summary.

### P3 Command Center

- Executive dashboard.
- Decision log.
- KPI registry.
- Governance and risk review.

## Support Rule

No custom support commitment without paid scope.
