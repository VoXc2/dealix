# Dealix Security Checklist

## Daily

- Production health works.
- No secrets in git status.
- No auto-send flags enabled.
- No suspicious workflow change.

## Weekly

- Review GitHub Actions.
- Review Railway variables.
- Review public domains.
- Review dependency warnings.
- Review access permissions.

## Monthly

- Rotate sensitive tokens.
- Review branch protection.
- Review client data handling.
- Review incident log.
- Review AI governance.
