# Dealix Quality Gate Model

## Gate 1 — Production

- API /healthz returns 200.
- Web /ar/demo returns 200.
- Revenue OS returns 200.

## Gate 2 — Build

- Python compile passes.
- Web build passes.
- No broken routes.

## Gate 3 — Commercial

- P1 offer exists.
- Outreach script exists.
- Proposal template exists.
- Scorecard exists.

## Gate 4 — Company Brain

- CEO brief generated.
- Content drafts generated.
- Growth scorecard generated.
- Approval queue exists.

## Gate 5 — Governance

- No auto-send.
- No scraping.
- No secrets committed.
- Human approval required.
