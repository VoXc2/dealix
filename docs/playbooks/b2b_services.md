# Vertical Playbook: B2B Services

## Sector overview

Agencies, consultancies, implementation partners, and professional services firms selling recurring projects or retainers.

## Common problems

- Weak lead tracking across sheets + CRM + WhatsApp context
- Manual follow-up; no shared definition of a qualified account
- Unclear ICP; everyone chases everything
- Proposal delays; scopes drift

## Best Dealix services

1. Lead Intelligence Sprint
2. Proposal / scope assistant workflows (governed)
3. Monthly RevOps OS (operating rhythm + proof pack cadence)

## Data usually available

- Lead spreadsheets and exports
- CRM dumps (HubSpot / others)
- Proposal docs and past SOWs
- WhatsApp/email text exports (with consent / policy boundaries)

## Best KPIs

- Qualified accounts per week
- Meetings booked from top-N list
- Proposal cycle time
- Pipeline value and stage hygiene
- Win rate from scored cohort vs. baseline

## Common objections

- "We already have salespeople"
- "Our data is messy"
- "We use WhatsApp for everything"

## Recommended scripts

- **Discovery:** one success metric + where truth lives today + approval path.
- **Outreach angles:** "prioritized actions + drafts for approval" not "magic leads."
- **Follow-up themes:** proof pack recap; next sprint boundary.

## Governance notes

- No cold WhatsApp automation
- Source attribution required for recommendations
- Approval before external communication
- No scraping; work from client-provided inputs

## Proof examples (anonymized patterns)

- Dedupe + source hygiene summary
- Top 50 accounts with rationale
- Flagged governance risks before any send
