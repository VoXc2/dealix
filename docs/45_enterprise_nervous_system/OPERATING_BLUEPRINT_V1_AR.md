# Operating Blueprint v1 — Dealix Enterprise Nervous System

## الهدف

تحويل المؤسسة من أقسام منفصلة وعمليات يدوية إلى منظمة Agentic ذات حوكمة عالية:

- AI ينفّذ ضمن السياسة
- البشر يشرفون على الاستثناءات
- الذاكرة المؤسسية موحدة
- الذكاء التنفيذي مستمر

## المبدأ التنفيذي

لا نقيس نجاح Dealix بعدد الميزات، بل بقدرة المؤسسة على:

1. إعادة تصميم سير العمل
2. تنفيذ سير العمل
3. حوكمة سير العمل
4. تقييم جودة سير العمل
5. توسيع النطاق مع الثقة

## Guardrails (Non-negotiable)

- لا أي إجراء خارجي بدون approval واضح.
- لا أي تنفيذ غير قابل للتتبع أو التدقيق.
- لا أتمتة رسائل خارجية حساسة بدون موافقة صريحة.
- كل قرار تنفيذي يجب أن يملك decision trace.

## Definition of Done (Organizational)

يُعتبر النظام "جاهز مؤسسة" عندما:

- جميع بوابات capability الأساسية = True.
- maturity band يصل `agentic_enterprise_ready` أو أعلى.
- معدل الالتزام بالسياسات >= 98%.
- قياس الأثر المالي والتشغيلي فعّال شهريًا.
