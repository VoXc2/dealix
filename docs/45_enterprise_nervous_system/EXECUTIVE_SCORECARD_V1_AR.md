# Executive Scorecard v1 — قياس القدرة المؤسسية

## لماذا هذه البطاقة؟

لأن قياس "عدد الميزات" لا يثبت التحول المؤسسي.
هذه البطاقة تقيس الجاهزية التشغيلية الحقيقية.

## Core KPIs

1. **workflow_cycle_time_reduction**
   - الهدف: >= 30%
   - الدورية: أسبوعي

2. **governed_autonomy_rate**
   - الهدف: >= 70%
   - الدورية: أسبوعي

3. **exception_escalation_precision**
   - الهدف: >= 90%
   - الدورية: أسبوعي

4. **policy_compliance_pass_rate**
   - الهدف: >= 98%
   - الدورية: يومي

5. **memory_grounding_score**
   - الهدف: >= 85/100
   - الدورية: أسبوعي

6. **workflow_success_rate**
   - الهدف: >= 92%
   - الدورية: يومي

7. **executive_signal_latency_hours**
   - الهدف: <= 24h
   - الدورية: أسبوعي

8. **revenue_leakage_prevented_sar**
   - الهدف: اتجاه تصاعدي
   - الدورية: شهري

9. **digital_workforce_uptime**
   - الهدف: >= 99%
   - الدورية: يومي

10. **continuous_improvement_velocity**
    - الهدف: >= 4 تحسينات معتمدة/شهر
    - الدورية: شهري

## Executive Review Rhythm

- Daily: compliance, uptime, workflow success
- Weekly: autonomy, cycle time, escalation quality, signal latency
- Monthly: value realization, improvement velocity, maturity band progression
