# Capability Roadmap v1 — Dealix

## Phase A — Control Plane First

الهدف: بناء طبقة التحكم والحوكمة قبل أي توسّع.

يشمل:

- Agent OS
- Workflow Orchestration
- Governance OS
- Evaluation + Observability
- Policy/Approval/Audit/Risk fabric

**Outcome:** أتمتة محكومة يمكن الوثوق بها.

## Phase B — Intelligence Plane

الهدف: تحويل البيانات التشغيلية إلى ذكاء تنفيذي.

يشمل:

- Organizational Memory
- Executive Intelligence
- Organizational Graph
- Knowledge Quality

**Outcome:** قرارات أسرع وأدق مع ذاكرة مؤسسية متراكمة.

## Phase C — Execution & Scale Plane

الهدف: توسيع التنفيذ بفرق رقمية واستدامة تشغيلية.

يشمل:

- Execution System
- Digital Workforce
- Continuous Evolution
- Transformation + Adoption
- Value Realization + Platform Reliability

**Outcome:** Dealix كطبقة تشغيل مؤسسية قابلة للتوسع.

## Prioritization Logic

الأولوية لا تُبنى على الرغبة، بل على:

`priority_score = (target_score - current_score) * system_weight`

أعلى `priority_score` = أعلى أولوية إصلاح.

## Maturity Bands

- `foundational` (<40)
- `emerging` (40–59)
- `scaling` (60–74)
- `agentic_enterprise_ready` (75–89)
- `enterprise_nervous_system` (>=90)
