# Company Growth Beast — سياسة أمان

## محظورات صلبة

- إرسال حي بدون موافقة مسجلّة.
- شحن حي / `moyasar_live_charge`.
- واتساب بارد.
- سكرابينغ أو قوائم مشتراة.
- أتمتة لينكدإن.
- إثبات أو شهادة بدون موافقة عميل / أحداث proof.

## أوضاع الإجراء

| وضع | معنى |
|-----|------|
| `draft_only` | مسودة للمراجعة الداخلية |
| `approval_required` | يحتاج موافقة مؤسس/عميل قبل التنفيذ الخارجي |
| `suggest_only` | رؤية داخلية |
| `blocked` | لا يُنفَّذ |

## PDPL وبيانات الدعم

- يمر نص الدعم عبر تنقيح بسيط للبريد/الجوال في `support_to_growth`.
- لا يُخزَّن PII في `session_store` للإنتاج طويل الأمد؛ استبدل بتخزين متعدد المستأجرين عند الجاهزية.

## قطاعات حساسة

عند الكلمات المفتاحية الطبية/المالية المرخصة: رفع تلقائي للتصعيد البشري قبل أي توصية خارجية.
