# غرفة تحكّم المؤسّس — Founder GTM Command Center

غرفة التحكّم هي الواجهة اليومية الوحيدة التي يحتاجها المؤسّس لتشغيل الإيرادات.
هدفها: **قرار واضح في أقل من 15 دقيقة صباحاً**، دون فتح عشرة ملفات.

## ماذا أرى كل صباح؟

1. **لوحة الأمس** — كم مسودّة أُنتجت، كم اجتازت البوّابات، كم تنتظر موافقتي.
   - المصدر: `reports/gtm/DAILY_GTM_REPORT.md`
2. **طابور الموافقة** — المسودّات الجاهزة، مرتّبة حسب الأولوية والمخاطر.
   - المصدر: `reports/outreach/APPROVAL_QUEUE.md`
3. **خطة دفعة الإرسال** (خطة فقط، لا إرسال) — كم وإلى أي قطاع، ومدى الجاهزية.
   - المصدر: `reports/outreach/SENDING_BATCH_PLAN.md` + `reports/outreach/SENDING_RAMP_READINESS.md`
4. **صحة قابلية التسليم** — هل الدومين سليم؟ ما الحُكم الحالي؟
   - المصدر: `reports/outreach/DELIVERABILITY_REVIEW.md`
5. **خط الأنابيب** — أين الفرص، وما الخطوة التالية لكل منها.
   - المصدر: `reports/commercial/PIPELINE_REVIEW.md`
6. **قرارات تحتاج موافقتي** — تسعير، إرسال، نشر، بيانات عميل.

## القرارات الأربعة اليومية للمؤسّس

| القرار | المدخل | الناتج |
|--------|--------|--------|
| الموافقة على المسودّات | طابور الموافقة | `approved` / `needs_revision` / `rejected` |
| الموافقة على خطة دفعة الإرسال | خطة الدفعة + حُكم القابلية | اعتماد أو تأجيل |
| الموافقة على التسعير/العروض | مراجعة العروض | اعتماد سعر نهائي |
| اختيار تركيز اليوم | تقرير الإشارات | قطاع/زاوية اليوم |

## مبادئ غرفة التحكّم

- **افتراضياً لا شيء يُرسَل.** كل ما تراه مسودّات وخطط. أنت من يفتح البوّابة.
- **لا مفاجآت.** أي إجراء خارجي يظهر في الطابور قبل حدوثه.
- **لا أسرار ولا PII** في أي تقرير تراه هنا.
- **قرار واحد لكل سطر.** إن لم يكن قابلاً للقرار، فهو لا ينتمي لغرفة التحكّم.

## أوامر التشغيل (Command Room CLI)

```bash
npm run commercial:check    # بوّابة الضوابط: يتحقق من الحدود والأمان
npm run commercial:plan     # خطة اليوم: مزيج المسودّات وخطة الدفعة (خطة فقط)
npm run commercial:quality  # بوّابة جودة المسودّات (claims/unsubscribe/تخصيص)
npm run commercial:brief    # ملخّص المؤسّس اليومي
npm run commercial:all      # تشغيل المتسلسلة كاملة
```

> هذه الأوامر تنتج تقارير وخططاً فقط. لا تُفعّل أي إرسال خارجي.
