# Production Environment Template

> **Purpose:** authoritative list of every environment variable Dealix reads in production code paths. Use this when:
> 1. Bootstrapping a new Railway / Fly / AWS deployment
> 2. Onboarding a contractor — share this file, not your `.env`
> 3. Filling out `preflight_check.py` P5 (env vars sanity)
>
> **Source-of-truth:** This file was generated by grepping `os.environ.get/getenv` across `api/`, `auto_client_acquisition/`, `dealix/`, `integrations/`, `scripts/`. Re-run the grep before each major release to catch drift.

---

## P1 — Required for Production Boot (API refuses to start without these)

| Variable | Example | Where used | Notes |
|----------|---------|-----------|-------|
| `APP_ENV` | `production` | `api/main.py` | `test` / `development` / `production` |
| `APP_DEBUG` | `false` | `api/main.py` | Never `true` in prod |
| `DATABASE_URL` | `postgresql+asyncpg://user:pass@host:5432/dealix` | `db/session.py` | Use `+asyncpg` driver |
| `BASE_URL` | `https://api.dealix.me` | `api/routers/health.py` | Used by `/healthz` ping + webhook URLs |
| `ADMIN_API_KEYS` | `key1,key2` | `api/middleware/tenant_isolation.py` | Comma-separated; rotated quarterly |
| `MOYASAR_SECRET_KEY` | `sk_live_...` | `dealix/payments.py` | Saudi PCI — store in secrets manager only |
| `MOYASAR_WEBHOOK_SECRET` | `whsec_...` | `api/routers/pricing.py` | Validated on every webhook callback |
| `REDIS_URL` | `redis://default:pass@host:6379/0` | DLQ, idempotency, rate-limit | Tier-1 required for queues |

## P2 — Required for AI to Function

| Variable | Used by |
|----------|---------|
| `ANTHROPIC_API_KEY` | Primary LLM router (Claude 4) |
| `OPENAI_API_KEY` | Secondary LLM router (fallback) |

## Pre-Deploy Checklist

Before flipping production traffic:

- [ ] All P1 variables set in Railway / hosting provider
- [ ] All P2 variables set (at least one LLM provider)
- [ ] All P3 variables set (at minimum: GOOGLE_MAPS, HUNTER, META_WHATSAPP_*)
- [ ] `MOYASAR_SECRET_KEY` is **sk_live_** (NOT sk_test_) in production
- [ ] `MOYASAR_WEBHOOK_SECRET` matches the one configured in Moyasar dashboard
- [ ] `WHATSAPP_MOCK_MODE=false` in production (default — just confirm it's not overridden)
- [ ] `APP_DEBUG=false`
- [ ] Run `python scripts/preflight_check.py` and resolve all P-severity failures
- [ ] Run `python scripts/daily_sanity.sh` and resolve all failures
- [ ] Trigger `/healthz` post-deploy — must return 200 with all dependencies green

## How to Generate This File Fresh

```bash
grep -rEho "os\.environ\.get\(['\"][A-Z_][A-Z0-9_]+['\"]" api/ auto_client_acquisition/ dealix/ integrations/ scripts/ \
  | sort -u
grep -rEho "getenv\(['\"][A-Z_][A-Z0-9_]+['\"]" api/ auto_client_acquisition/ dealix/ integrations/ scripts/ \
  | sort -u
```

If new variables appear that aren't in this file, update this file in the same PR.
