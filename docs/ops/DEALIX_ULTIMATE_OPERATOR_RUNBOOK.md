# Dealix Ultimate Operator Runbook

## المبدأ
كل يوم = نفس الإيقاع. لا عشوائية.

## Daily (15 min)
1. `python3 scripts/dealix_daily_operator.py --mode demo`
2. اقرأ CEO Brief
3. اقرأ pipeline report
4. اعتمد أو ارفض المسوّدات المعلّقة

## Weekly (60 min)
1. مراجعة المسوّدات المعلّقة
2. تحديث stages في CRM
3. إرسال العروض الجاهزة
4. تسجيل proof item واحد على الأقل

## Monthly (90 min)
1. مراجعة KPIs
2. تخطيط أول 100 ليدز
3. مراجعة الـ partners
4. تقرير شهري للـ founders
