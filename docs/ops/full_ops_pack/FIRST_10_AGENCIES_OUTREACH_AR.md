# أول 10 وكالات — رسائل وتتبع (عربي)

الهدف: لمسات قصيرة، قابلة للقياس، **بدون واتساب بارد**. التواصل عبر قنوات مسموحة (LinkedIn، إيميل بعد تعريف، إحالة، رد على تفاعل).

---

## قواعد

1. سجّل في `02_Operating_Board` أو سجل جانبي: `source` = `linkedin_manual` أو `referral` أو `x_post` حسب المصدر.
2. لا ترسل واتساب لرقم من قائمة شرائية بدون opt-in.
3. كل رسالة: هوية واضحة + سبب تواصل + طلب صغير (رد أو رابط فورم).

---

## قالب Agency Outreach (قصير)

```
السلام عليكم [الاسم]،

أطلقت Dealix Beta للوكالات.

الفكرة بسيطة:
نجهز لكم Diagnostic مجاني لعميل واحد:
3 فرص مناسبة + رسالة عربية + توصية قناة + Proof مختصر.

إذا أعجبتك العينة نكمل Pilot كامل 7 أيام.

أرسل لك رابط التجربة؟
```

---

## قالب Follow-up 24h

```
متابعة سريعة فقط.

أقدر أرسل لك عينة إضافية:
فرصة واحدة + رسالة متابعة + توصية قناة.

إذا مناسب نكمل Pilot كامل.
```
