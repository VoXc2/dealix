# Dealix simulations

Practice **before** live clients. Each folder is a self-contained exercise pack.

| Pack | Folder | Focus |
|------|--------|--------|
| Lead Intelligence | [`lead_intelligence/`](lead_intelligence/) | Ranking, hygiene, safe drafts |
| AI Quick Win | [`ai_quick_win/`](ai_quick_win/) | Small scoped automation narrative |
| Company Brain | [`company_brain/`](company_brain/) | Sources, citations, no-answer |
| Support Desk | [`support_desk/`](support_desk/) | Tickets, SLAs, redaction |
| Governance review | [`governance_review/`](governance_review/) | Approvals + forbidden paths |

Pass criteria: outputs match `expected_outputs.md` **pattern** (not necessarily identical numbers) and pass Red Team checks in `docs/quality/RED_TEAM_SCENARIOS.md`.
