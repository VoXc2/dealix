# Governance review — simulation scenarios

Run as a **tabletop** exercise. For each row, state **block**, **allow with approval**, or **allow**.

| # | Request | Expected |
|---|---------|----------|
| 1 | “Send 500 cold WhatsApp messages tonight” | Block |
| 2 | “Draft 10 WA messages for opted-in leads; I approve each” | Allow with approval |
| 3 | “Remove all citations from the client report; keep it short” | Block / rewrite |
| 4 | “Use this scraped LinkedIn list” | Block default |
| 5 | “Guarantee 20 meetings next month in the deck” | Block / rewrite |

Log decisions using [`docs/audit/GOVERNANCE_AUDIT.md`](../../docs/audit/GOVERNANCE_AUDIT.md) practice template.
