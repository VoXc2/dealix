# AI Quick Win — simulation brief

**Client:** SMB with one painful manual report that takes 3 hours weekly.

**Goal:** Deliver a **bounded** automation or semi-automation in ≤10 days with a single success metric.

**Inputs:** sample CSV export + 1-page process description (add your own synthetic files).

**Forbidden:** outbound messaging automation; promises of headcount reduction “guaranteed.”

## Expected outputs

- Before/after workflow map
- Script or no-code spec (if applicable)
- QA checklist result
- Proof pack (what changed, what stayed human)
