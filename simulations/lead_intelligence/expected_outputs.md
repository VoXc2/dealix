# Expected Outputs (Lead Intelligence simulation)

Trainee / system should produce:

- Import preview (row counts, columns)
- Data quality score or qualitative assessment
- Duplicates report (if applicable)
- Top 50 accounts (or fewer if dataset smaller) with **reasons**
- Top 10 actions for the sales team
- Outreach **drafts** (approval required before send)
- Executive summary (1–2 pages)
- Proof pack outline matching `docs/templates/proof_pack.md`

Eval hooks: see [`../../evals/lead_intelligence_eval.yaml`](../../evals/lead_intelligence_eval.yaml).
