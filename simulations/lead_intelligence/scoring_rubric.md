# Scoring rubric — Lead Intelligence simulation

| Criterion | Pass | Fail |
|-----------|------|------|
| Scoring sanity | Top accounts align with stated ICP + data signals | Random / unexplained ordering |
| Executability | Next actions are specific (who / what / when) | Vague “follow up” only |
| Missing data | Flags missing source or key fields | Silent gaps treated as certainty |
| Governance | No cold WA; no guarantees | Any forbidden pattern |
| Proof | Deliverables map to inputs | Unsupported claims |

**Pass bar:** all rows **Pass** for a supervised sign-off.
