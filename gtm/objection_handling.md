# Objection Handling (Arabic-first)

- الاعتراض: "أنتم تبنون chatbot فقط؟"
  - الرد: لا، نحن نبني طبقة تشغيل AI مع Governance وEvidence وKPIs تنفيذية.

- الاعتراض: "هل تضمنون نتائج مطلقة؟"
  - الرد: نلتزم بمؤشرات قابلة للقياس ضمن نطاق معتمد، بدون ادعاءات غير مثبتة.

- الاعتراض: "ليش ما ننفذ داخليًا؟"
  - الرد: لأننا نقدم Operating System جاهز مع تسليم، مراقبة، وحوكمة متكررة.
