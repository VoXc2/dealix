# Maturity Model

## Purpose
تحديد نضج العميل عبر مسار التحول من AI experiments إلى Agentic Operating Model.

## Levels
1. Fragmented Automation
2. Managed Workflows
3. Governed Autonomy
4. Operational Intelligence
5. Enterprise Civilization Layer

## Exit Rule
لا انتقال لمستوى أعلى بدون Evidence على impact + governance + resilience.
