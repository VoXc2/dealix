# Workflow Redesign Framework

## Redesign sequence

1. Map current workflow and failure modes.
2. Define policy boundaries and risk tiers.
3. Split into agent tasks + human approvals.
4. Add retry/idempotency/rollback contracts.
5. Attach observability + eval gates.

## Output artifacts

- redesigned workflow map
- approval map
- rollback map
- measurable KPI set
