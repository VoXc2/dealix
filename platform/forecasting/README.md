# Forecasting

**Layer**: L12 — Decision Intelligence Engine

## Mission
توقع المخاطر والعوائد والقدرة التشغيلية للأفق القادم.

## Inputs
time-series operational metrics, market signals, seasonality.

## Outputs
forecasts for demand, risk, revenue, capacity.

## Core KPIs
forecast MAPE, bias drift, forecast utilization by leadership.

## Required Controls
backtesting discipline, drift alerts, confidence interval reporting.

## Readiness Exit Criteria
- KPI baselines محددة ومراقبة بشكل دوري.
- سياسات الحوكمة مفروضة runtime وليست optional.
- يوجد مسار rollback / escalation موثق وقابل للتجربة.
- Evidence Pack متاح للمراجعة التنفيذية والتدقيق.
