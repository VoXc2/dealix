# Approval Center

**Layer**: L14 — Human-Agent Collaboration Engine

## Mission
تجميع الموافقات البشرية في مركز واحد قابل للتدقيق.

## Inputs
pending approvals, action previews, policy context.

## Outputs
approve/reject/defer actions with reason capture.

## Core KPIs
approval turnaround, approval quality, rework from approvals.

## Required Controls
mandatory rationale capture, batch safety limits, approval audit ledger.

## Readiness Exit Criteria
- KPI baselines محددة ومراقبة بشكل دوري.
- سياسات الحوكمة مفروضة runtime وليست optional.
- يوجد مسار rollback / escalation موثق وقابل للتجربة.
- Evidence Pack متاح للمراجعة التنفيذية والتدقيق.
