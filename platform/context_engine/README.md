# Context Engine

**Layer**: L11 — Organizational State Engine

## Mission
توحيد السياق التشغيلي متعدد المصادر في context graph موحد.

## Inputs
CRM, tickets, workflows, telemetry, policy metadata.

## Outputs
context bundles للقرارات والوكلاء.

## Core KPIs
context retrieval latency, relevance score, context miss rate.

## Required Controls
policy-aware filtering, deterministic context assembly, audit trails.

## Readiness Exit Criteria
- KPI baselines محددة ومراقبة بشكل دوري.
- سياسات الحوكمة مفروضة runtime وليست optional.
- يوجد مسار rollback / escalation موثق وقابل للتجربة.
- Evidence Pack متاح للمراجعة التنفيذية والتدقيق.
