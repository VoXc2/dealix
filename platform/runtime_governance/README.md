# Governed Autonomy Engine

Purpose: enforce runtime governance so autonomy is safe, auditable, and reversible.

## Core responsibilities

- Policy checks for every action and tool invocation.
- Approval checkpoints and escalation flows for sensitive actions.
- Write staging and reversibility contracts for critical operations.

## Required linked modules

- `platform/tool_fencing`
- `platform/escalation`
- `platform/write_staging`
- `platform/reversibility`
