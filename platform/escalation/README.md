# Escalation

**Layer**: L14 — Human-Agent Collaboration Engine

## Mission
ضمان تصعيد منضبط عند المخاطر أو عدم اليقين.

## Inputs
risk thresholds, confidence drops, policy violations.

## Outputs
escalation actions, owner routing, urgency tiers.

## Core KPIs
escalation response time, unresolved escalations, escalation accuracy.

## Required Controls
deterministic escalation paths, on-call mapping, acknowledgement SLAs.

## Readiness Exit Criteria
- KPI baselines محددة ومراقبة بشكل دوري.
- سياسات الحوكمة مفروضة runtime وليست optional.
- يوجد مسار rollback / escalation موثق وقابل للتجربة.
- Evidence Pack متاح للمراجعة التنفيذية والتدقيق.
