# Reversibility

**Layer**: L15 — Enterprise Trust Engine

## Mission
ضمان قابلية الرجوع الآمن للإجراءات الحرجة.

## Inputs
action manifests, state diffs, rollback playbooks.

## Outputs
rollback plans, rollback execution records.

## Core KPIs
rollback success rate, rollback time, irreversible action ratio.

## Required Controls
pre-action rollback contract, compensating actions, rollback drills.

## Readiness Exit Criteria
- KPI baselines محددة ومراقبة بشكل دوري.
- سياسات الحوكمة مفروضة runtime وليست optional.
- يوجد مسار rollback / escalation موثق وقابل للتجربة.
- Evidence Pack متاح للمراجعة التنفيذية والتدقيق.
