# Prioritization

**Layer**: L12 — Decision Intelligence Engine

## Mission
ترتيب المبادرات والمهام وفق الأثر والمخاطر والموارد.

## Inputs
initiative backlog, effort estimates, impact forecasts, constraints.

## Outputs
priority stacks with rationale and trade-offs.

## Core KPIs
priority stability, throughput of top-tier items, value per capacity unit.

## Required Controls
explicit trade-off policy, starvation protection, executive traceability.

## Readiness Exit Criteria
- KPI baselines محددة ومراقبة بشكل دوري.
- سياسات الحوكمة مفروضة runtime وليست optional.
- يوجد مسار rollback / escalation موثق وقابل للتجربة.
- Evidence Pack متاح للمراجعة التنفيذية والتدقيق.
