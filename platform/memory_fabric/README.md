# Operational Memory Engine

Purpose: provide organizational memory with lineage, retrieval quality, and decision citations.

## Core responsibilities

- Connect customer, workflow, governance, and incident memory planes.
- Preserve decision lineage from input to approval to action.
- Return citation-backed context to downstream workflows.

## Required linked modules

- `memory/`
- `platform/lineage`
- `platform/retrieval`
- `platform/reranking`
- `platform/citations`
