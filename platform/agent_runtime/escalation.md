# Escalation Path

## Escalation Required When

- risk level >= high
- policy conflict unresolved
- repeated retry exhaustion
- customer-impacting commitment

## Escalation Payload

- run summary
- failed checks
- policy decisions
- recommended options
- rollback suggestion

## Human-in-the-Loop Rule

الأفعال المالية/القانونية/الالتزامية لا تنفذ بدون موافقة بشرية صريحة.
