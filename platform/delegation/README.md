# Delegation

**Layer**: L14 — Human-Agent Collaboration Engine

## Mission
تنفيذ تفويض ذكي بالصلاحيات المناسبة والسياق الصحيح.

## Inputs
delegation intents, permission scopes, capability profiles.

## Outputs
delegation contracts and bounded execution plans.

## Core KPIs
delegation success rate, scope breach incidents, delegation reuse.

## Required Controls
least-privilege enforcement, scope expiry, emergency revoke.

## Readiness Exit Criteria
- KPI baselines محددة ومراقبة بشكل دوري.
- سياسات الحوكمة مفروضة runtime وليست optional.
- يوجد مسار rollback / escalation موثق وقابل للتجربة.
- Evidence Pack متاح للمراجعة التنفيذية والتدقيق.
