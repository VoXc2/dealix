# Dealix Platform Capability Manifests

This directory is a product-contract surface for the Agentic Enterprise phase.

It does not replace current production modules under `auto_client_acquisition/`.
Instead, it declares the forward capability topology for Systems 16–25 as stable
platform paths that can be mapped to implementation over time.

- Canonical registry: `platform/system_16_25_registry.json`
- Python contract helpers: `auto_client_acquisition/agentic_enterprise_os/`

The goal is to shift from feature-first planning to capability-first execution.
