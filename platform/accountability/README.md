# Accountability

**Layer**: L15 — Enterprise Trust Engine

## Mission
إسناد المسؤولية لكل قرار وتنفيذ بشكل غير ملتبس.

## Inputs
actor identity, role, delegation chain, approval records.

## Outputs
responsibility graph and accountability reports.

## Core KPIs
attribution completeness, orphan-action count, accountability SLA.

## Required Controls
actor provenance checks, delegated responsibility chain, signed action records.

## Readiness Exit Criteria
- KPI baselines محددة ومراقبة بشكل دوري.
- سياسات الحوكمة مفروضة runtime وليست optional.
- يوجد مسار rollback / escalation موثق وقابل للتجربة.
- Evidence Pack متاح للمراجعة التنفيذية والتدقيق.
