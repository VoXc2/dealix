# Explainability

**Layer**: L14 — Human-Agent Collaboration Engine

## Mission
شرح القرارات والإجراءات بلغة تشغيلية مفهومة.

## Inputs
decision rationale, evidence references, policy links.

## Outputs
human-readable explanations and evidence trails.

## Core KPIs
explanation usefulness score, contest rate, time-to-understand.

## Required Controls
trace-to-explanation parity, evidence citations, multilingual support roadmap.

## Readiness Exit Criteria
- KPI baselines محددة ومراقبة بشكل دوري.
- سياسات الحوكمة مفروضة runtime وليست optional.
- يوجد مسار rollback / escalation موثق وقابل للتجربة.
- Evidence Pack متاح للمراجعة التنفيذية والتدقيق.
